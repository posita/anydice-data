                                 Arbitrary Dice

   Besides creating a die based on a single number, you can also create a
   die based on a sequence, for example d{1,2,4,4}. This allows you to
   create dice of any shape. It is also possible to explicitly create dice
   based on another die or collection of dice.
output d{1..6} named "d6"

output d{-1..1} named "Fudge die"

output d{2, 3, 3, 4} named "2d2"

output 2d(2d2) named "4d2"

output d{} named "the empty die"
