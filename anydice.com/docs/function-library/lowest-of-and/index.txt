                          lowest of NUMBER and NUMBER

   This function yields the lowest of the two NUMBERs.

  Examples

output [lowest of 2 and 3]
output [lowest of d6 and 4]
output [lowest of d4 and d6]

  Do it yourself

function: lowest of A:n and B:n {
 if A < B { result: A }
 result: B
}
