                             lowest NUMBER of DICE

   This function yields the sum of the lowest NUMBER values rolled by
   DICE.

  Examples

output [lowest 3 of 4d6]
output [lowest d4 of 4d6]

  Do it yourself

function: lowest NUMBER:n of DICE:d {
 result: {(#DICE - NUMBER + 1)..#DICE}@DICE
}
