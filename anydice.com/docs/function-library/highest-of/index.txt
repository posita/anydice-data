                             highest NUMBER of DICE

   This function yields the sum of the highest NUMBER values rolled by
   DICE.

  Examples

output [highest 3 of 4d6]
output [highest d4 of 4d6]

  Do it yourself

function: highest NUMBER:n of DICE:d {
 result: {1..NUMBER}@DICE
}
