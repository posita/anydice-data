                                      Dice

   AnyDice has a small collection of predefined functions, ready for use.
     * absolute NUMBER
     * SEQUENCE contains NUMBER
     * count VALUES in SEQUENCE
     * explode DIE
     * highest NUMBER of DICE
     * lowest NUMBER of DICE
     * middle NUMBER of DICE
     * highest of NUMBER and NUMBER
     * lowest of NUMBER and NUMBER
     * maximum of DIE
     * reverse SEQUENCE
     * sort SEQUENCE
