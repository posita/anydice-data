\Rolling too many dice will cause an error, so adjust DICEMIN and DICEMAX to keep your dice reasonable!\

DICEMIM: 5 \Starting number of dice rolled\
DICEMAX: 10 \Ending number of dice rolled\
DIEBONUS: 26 \Bonus per die\
CURRENTPROGRESS: 997 \Your current project progress\
PROGRESSTARGET: 1535 \The total progress the project needs\
OMAKEBONUS: 15 \Omake bonus points\

function: evaluate ROLL:s {
 SUCCESSES: ROLL >= 0
 if SUCCESSES > 0 { result: 1 }
 result: 0
}

loop DICE over {DICEMIM..DICEMAX} {
 output [evaluate DICEd100+DIEBONUS*DICE+CURRENTPROGRESS-PROGRESSTARGET+OMAKEBONUS] named "[DICE]"
}
