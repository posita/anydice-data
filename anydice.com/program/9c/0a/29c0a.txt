HP: (3d6*5 + 3d6*5)/10

DEEP_ONE_DMG: 1d7+d4 \ d7 = average of d6 and d8 weapons \

function: combat N H R {
 DDMG:  N * DEEP_ONE_DMG / 2 \ divide by 2 is an approximation; deep one Brawl skill is 45%. \
 result: RdDDMG > H     \ likelihood of 1 = TPK \
}

loop PARTYSIZE over {1..5} {
 TOTAL_HP: HP * PARTYSIZE
 loop ENEMIES over {1..10} {
  loop ROUNDS over {1..5} {  
   output [combat ENEMIES TOTAL_HP ROUNDS] named "[ROUNDS] rounds of [PARTYSIZE] players fighting [ENEMIES] deep ones"
  }
 }
}
