function: roll TIMES:n die SIDES:n count SUCC:n explode on EXPL:n 
{EXPLODED: 0
SUCCESSES: 0
loop N over {1..TIMES} {DIEROLL: 1dSIDES
    SUCCESSES: SUCCESSES + [helper DIEROLL SUCC]
    EXPLODED: EXPLODED + [helper DIEROLL EXPL]}
result: SUCCESSES + [roll EXPLODED die SIDES count SUCC explode on EXPL]}
function: helper N:n GE:n {if N >= GE { result: 1 } result: 0}

output [roll X die 6 count 4 explode on 6]>=Y
