function: reroll N:n {
 if N < 6 { result: 1d20 }
 result: N
}

\ Create a custom die that represents "1d20 reroll < 6 once" \
CUSTOM_DIE: [reroll 1d20]

\ 1. The standard 2d20 drop lowest (Advantage) \
output [highest 1 of 2d20] named "2d20d1 (Standard Advantage)"

\ 2. The single die that rerolls 1-5 once \
output CUSTOM_DIE named "1d20 reroll < 6 once"

\ 3. The combo: Two of those custom dice, keeping the highest \
output [highest 1 of 2CUSTOM_DIE] named "The Combo"
