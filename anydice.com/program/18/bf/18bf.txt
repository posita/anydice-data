function: fight A:n and B:n until W{
H: 0
loop N over {1..10}{
if H > W { result: N }
T: [attack A at B]
H: H+T }
}


\This calculates the number of wounds done\
function: attack A:n at B:n {
if A > B { result: A-B }
result: 0
}

\"Hot Dice"\
function: hot A:n and B:n {
if A > B { result: -B}
if B > A { result: A}
if B = A { result: 0}
}

\Here's hot dice defined with 2d10. The first die is hot, second cold\
D: [hot d10 and d10]
