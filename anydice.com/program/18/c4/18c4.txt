

\This calculates the number of wounds done\
function: attack A:n at B:n {
if A > B { result: A-B }
result: 0
}

\"Hot Dice"\
function: hot A:n and B:n {
if A > B { result: -B}
if B > A { result: A}
if B = A { result: 0}
}

\Here's hot dice defined with 2d10. The first die is hot, second cold\
D: [hot d10 and d10]


\MODIFY THESE. This calculates the odds of dealing a number of wounds\
output [attack D at 2] named "+0 attack against 2 defense"
output [attack D at 1] named "+0 attack against 1 defense"
output [attack D at 0] named "+0 attack against 0 defense"
output [attack D+1 at 0] named "+1 attack against 0 defense"
output [attack D+2 at 0] named "+2 attack against 0 defense"
output [attack D+3 at 0] named "+3 attack against 0 defense"
output [attack D+4 at 0] named "+4 attack against 0 defense"
