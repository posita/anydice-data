
function: fr A of ROLL:s in B{
 EXPLODE : 0
 X : 0
 DIE : (ROLL=1)
 loop N over {1..#ROLL}{
   if N > DIE{
     if N <= DIE + A {
       if N@ROLL = B {
        EXPLODE: EXPLODE +1
       }
       if N@ROLL > 1 {
         X : X + N@ROLL
       }
     }
   }
 }
 if EXPLODE > 0 {
    X : X + EXPLODEd[explode dB]
 }
 result : X
}

output [fr 2 of 5d10 in 10]
output [highest 2 of 5d[explode d{d9+1}]]
