\ Frei Rollenspiel Dice Pool Mechanics \
\ (roll a pool of D6's and take the highest result; 1-2 is a Failure, 3-4 is a partial success, 5-6 is a success) \
\ In these charts, each of the following numbers 0-3 represent the % of each result \ 
\ 0: Failure \
\ 1: Partial Success \
\ 2: Success \
\ 3: 2+ Successes \

function: pool ROLL:s {
 if 1@ROLL >= 5 {
  result: 2 + (2@ROLL >= 5)
 }
 result: 1@ROLL >= 3
}

output [pool 2@2d6] named "0d"
loop P over {1..6} {
 output [pool Pd6] named "[P]d"
}
