function: attack D:n plus BONUS:n plus S:n vs DC:n  {
   if D = 2 { result: 0}
   else if D + BONUS + S>= DC { result: 1 } 
   else { result:0}
}

   output [attack 2d4 plus 1 plus 0 vs DC] named "D4 Attribute Die beats DC[DC] with No Training"
