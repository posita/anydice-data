function: lucky A:n if below B:n {
if A < B {result: d12}
result: A
}

function: unlucky A:n if above B:n {
if A >= B {result: d12}
result: A
}

output [count {X..12} in Nd12] + [count {X..12} in Ld[lucky d12 if below X]] + [count {X..12} in Ud[unlucky d12 if above X]]
