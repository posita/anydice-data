\ counts number of pairs, 3-of-a-kind, 4-of-a-kind, etc. in a sequence \
\ includes superior hands in results; e.g. 3-of-a-kind can count as a pair, 4-of-a-kind can count as TWO pairs, etc. \
\ AMOUNT: elements in a set (e.g. 2 to look for pairs, 3 to look for 3-of-a-kind, etc.) \
\ VALUES: the numbers that can legally be part of a set \
\ DICE: the sequence to search \
function: AMOUNT:n of a kind from VALUES:s in DICE:s {
    SETS: 0
    loop I over VALUES {
        SETS: SETS + ([count I in DICE] / AMOUNT)
    }
    result: SETS
}

output [3 of a kind from {1..6} in 5d6] named "3 of a kind"
output [4 of a kind from {1..6} in 5d6] named "4 of a kind"
output [5 of a kind from {1..6} in 5d6] named "5 of a kind"



\ counts number of "short straights" in a sequence; e.g. 2-in-a-row, 3-in-a-row, etc. \
\ includes superior straigths in results; e.g. 3-in-a-row counts as 2 instances of 2-in-a-row \
\ LENGTH: length of straight to search for (2 to find 2-in-a-rows, etc.) \
\ VALUES: the numbers that can legally START a straight; e.g. {2, 4} will count only 2-5 and 4-7 as "4-in-a-rows" \
\ DICE: the sequence to search \
function: LENGTH:n straight from VALUES:s in DICE:s {
    STRAIGHTS: 0
    loop I over VALUES {
        FOUND: 0
        loop J over {1..LENGTH} {
            if [count (I+J-1) in DICE] >= 1 {
                FOUND: FOUND + 1
            }
        }
        if FOUND >= LENGTH {
            STRAIGHTS: STRAIGHTS + 1
        }
    }
    result: STRAIGHTS
}

output [2 straight from {1..6} in 5d6] named "2 in a row"
output [3 straight from {1..6} in 5d6] named "3 in a row"
output [4 straight from {1..6} in 5d6] named "4 in a row"
output [5 straight from {1..6} in 5d6] named "5 in a row"
