CRITICALHIT:2
HIT:1
MISS:0

function: attack ROLL:n MOD:n vs DEFENSE:n {
 if ROLL = 1 { result: MISS }
 if ROLL = 20 {result: CRITICALHIT}
 if ROLL + MOD>= DEFENSE { result: HIT }
 result: MISS
}


function: damage ATTACKRESULT:n MOD:n with SOMANY:n DAMAGEDIE:n{
if ATTACKRESULT = 0 {result: SOMANYdDAMAGEDIE}
if ATTACKRESULT = 1 {result: SOMANYdDAMAGEDIE +MOD}
if ATTACKRESULT = 2 {result: SOMANY*DAMAGEDIE+MOD}
result: 0
}

LIGTHWEAPON: 6
DIE: 10

  output [damage [attack d20 DIVISION vs AC] DIVISION with 3 10] + [damage [attack d20 LIGTHWEAPON vs AC] LIGTHWEAPON with 2 6] +  [damage [attack d20 LIGTHWEAPON vs AC] LIGTHWEAPON with 2 6]named "[AC]"
}
