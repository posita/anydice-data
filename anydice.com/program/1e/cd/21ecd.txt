\
 Blades in the Dark

You roll a bunch of regular d6, then look at the highest number you got.

1–3: Not so good.
4–5: Success, but with undesired consequences.
6: Success without issues.
More than one 6: Critical success!

In AnyDice, it's defined as:
 0: Things go bad.
 1: Success, but with undesired consequences.
 2: Clean success.
 3: Critical success!
\

function: blades ROLL:s {
 if 1@ROLL = 6 {
  result: 2 + (2@ROLL = 6)
 }
 result: 1@ROLL >= 4
}

output [blades 2@2d6] named "0d"
loop P over {1..7} {
 output [blades Pd6] named "[P]d"
}
