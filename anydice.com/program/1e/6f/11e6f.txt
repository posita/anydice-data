\ The big problem with Savage Attacker is that improving your accuracy and damage by getting an ASI increases your damage way more than getting the feat does.        \
\ But if we're using a V. Human, then Savage Attacker has to compete with other feats.                                                                                \
\ In this case, UA Blade Mastery is better even though the average damage is about 1 point less, because it gives you more abilities.                                 \
\ (advantage on opportunity attacks, increased AC at the cost of a reaction)                                                                                          \
\ This isn't Savage Attacker's main problem, though. The problem is that it's only useful in combat, the damage increase worsens the more dice you reroll,            \
\ and that you can only do it once per set of damage dice per turn.                                                                                                   \
\ It's far better to take the Lucky feat or GWM.                                                                                                                      \
\ Another alternative is just taking a level of fighter and getting the GWF or Duelling fighting style, which additionally comes with the ever desirable Action Surge.\
\ This is a good way of getting your damage dice reroll kicks (especially since GWF works on all damage dice involved in an attack).                                  \
HIT: 1
CRITICALHIT: 2

function: attack ROLL:n vs DEFENSE:n {
 if ROLL = 1 { result: 0 }
 if ROLL = 20 { result: CRITICALHIT }
 if ROLL + ATTACK >= DEFENSE { result: HIT }
 result: 0
}

function: damage for ATTACKRESULT:n {
 if ATTACKRESULT = CRITICALHIT {
  result: CRITICALDAMAGE
 }
 if ATTACKRESULT = HIT {
  result: HITDAMAGE
 }
 result: 0
}


function: damagetwo for ATTACKRESULT:n {
 if ATTACKRESULT = CRITICALHIT {
  result: CRITICALDAMAGETWO
 }
 if ATTACKRESULT = HIT {
  result: HITDAMAGETWO
 }
 result: 0
}

ATTACK: 3 + 3
HITDAMAGE: [highest 2 of 4d6] + 3
CRITICALDAMAGE: [highest 4 of 8d6] + 3 
HITDAMAGETWO: 2d6 + 3
CRITICALDAMAGETWO: 4d6 + 3
loop AC over {12} {
 output [damage for [attack d20 vs AC]] + [damagetwo for [attack d20 vs AC]] named "[Lvl 6 Barb with 16 str and Savage Attacker]"
}
