\**** Damage calculator normal Vs advantage ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{19}

ATTACKMOD: 5
DMGBONUS: 6
PROFICIENCY: 6
ACCBONUS: 0
CRITROLL: 19
ATTACKTIMES: 12
DAMAGE: 1d10 + ATTACKMOD + DMGBONUS + 1d6
CRITDAMAGE: 2d10 + ATTACKMOD + DMGBONUS + 2d6
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "12 Eldritch Blasts w/ Hex & Hexblade's Curse"
