function: attack ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

ENEMYAC: d{10..20}

ATTACKMOD: 4
ACCBONUS: 2
DMGBONUS: 2
PROFICIENCY: 3

DAMAGEDICE: 1d8
HUNTERSMARK: 0

DAMAGE: DAMAGEDICE + ATTACKMOD + DMGBONUS + HUNTERSMARK
CRITDAMAGE: 2dDAMAGEDICE + ATTACKMOD + DMGBONUS + 2dHUNTERSMARK

EXTRAATTACK: 2
CRITROLL: 20
