\
Roll 1d10, if you roll 10, confirm the "perfect roll" by rolling another 1d10, and if that is over eight, you may add another 1d10 to the original roll. And if i roll 10 there aswell, i check if that roll is perfect too :) If its not perfect, it just counts as a ten. 
\

function: roll N:n {
 if N = 10 {
  result: N + [check perfect d10]
 }
 result: N
}

function: check perfect N:n {
 if N > 8 {
  result: [roll d10]
 }
 result: 0
}

output [roll d10] named "checking for perfect 10s"
