\ Define variables:\

\ N: number of dice:\
N: 3

\ T: type of dice (number of sides of the dice):\
T: 6

\ M: additional modifier:\
M: 2



\ O: the output:\
\    initialize 'O' to a single dT roll \
O: dT

\ loop one less then N times, each time adding \
\ another dT roll to 'O'\

\ L: lowest value: will save the lowest dT roll \
\   and subtract it from 'O'\
\ initialize 'L' to 'O', then in the loop compare\
\   'L' to each new dT, if a lower one comes along\
\   save that new low to 'L'\
L: O


loop I over {1:(N-1)}{
   \ R: new value gotten from this iteration's dT roll\
   R: dT
   O: O + R
   
   \ how does this R compare to L? \
   \ if R is less then L store this R as the new L \
   if R < L {
      L: R
   }
}

output O + M
