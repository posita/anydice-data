function: attack ROLL:n plus TOHIT vs AC:n do DMG:n m BONUS:n {
 if ROLL >= CRITROLL { result: 2*DMG + BONUS } 
 if ROLL = 1 { result: 0 }
 if ROLL + TOHIT >= AC { result: DMG + BONUS }
 result: 0
}

ENEMYAC: d{12..20}
CRITROLL: 20

output 2d[attack d20 plus 3 vs ENEMYAC do 3d6 m 1]
output 2d[attack d20 plus 7 vs ENEMYAC do 1d8 m 5]
output 2d[attack d20 plus 6 vs ENEMYAC do 1d12+1d8 m 4]
