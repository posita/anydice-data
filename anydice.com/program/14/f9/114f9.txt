\Getting dice to reroll is funky\
function: reroll R:n under N:n this D {
   if R < N { result: D } else {result: R}
}

\Great weapon fighter's axe & sword\
GWA: [reroll 1d12 under 3 this d12]
GWS: [reroll 1d6 under 3 this d6]

\Advantage and Disadvantage\
ADV: [highest 1 of 2d20]
DIS: [lowest 1 of 2d20]

\Attack die, attack mod, critical value, damage die, damage bonus,
 extra crit damage, target AC\
function: hit H:n mod M crit C dmg D bonus B extra E ac A {
   if H = 1 {result : 0}
   else if H >= C {result: D+D+B}
   else if H+M >= A {result: D+B} 
   else {result: 0}
}

\11th-level Fighter with Flametongue\
FIGHTER: [hit 1d20 mod 9 crit 19 dmg 1d8+2d6 bonus 5 extra 0 ac 18]
\11th-level Barbarian with Greataxe+2\
BARB: [hit ADV mod 11 crit 20 dmg d12 bonus 10 extra d12 ac 18]

output FIGHTER+FIGHTER+FIGHTER named "Fighter"
output BARB+BARB named "Barbarian"
