\Armor Values\
ARMLVL: 1
ARMTIER: 2
PREARM: d{0..(ARMTIER*(2+ARMLVL))}
POSTARM: 2d{ARMLVL..(ARMTIER*(2+ARMLVL))}/2

\Damage Values\
PREDMG: 2d{8..15}/2
POSTDMG: 2d{10..16}/2 

\Function to treat negative damage as 0\
function: A:n above zero {
  if A > 0 { result: A }
  result: 0
}

\Actual output commands\
output [PREDMG - PREARM above zero] named "before"
output [POSTDMG - POSTARM above zero] named "after"
