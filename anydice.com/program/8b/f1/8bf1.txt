\Weapon Values\
WEPLVL: 1
WEPTIER: 2
WEPDMG: 2d{WEPTIER+WEPLVL..(5+WEPLVL)*(WEPTIER+1)}/2

\DR Values\
PREDR: d{0..9}
POSTDR: 2d{0..8}/2 

\Function to treat negative damage as 0\
function: A:n above zero {
  if A > 0 { result: A }
  result: 0
}

\Actual output commands\
output [WEPDMG- PREDR above zero] named "before"
output [WEPDMG- POSTDR above zero] named "after"
