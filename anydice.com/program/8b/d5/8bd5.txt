\This is the dice type for attributes\
A: 1d20
B: 1d20
C: 1d20
\This indicates Number of skilldice\
SKILLNUMBER: 1
\This is the dice type of the skilldice\
SKILLDICE: d12
\This is the Attribute 1 TN you roll against\
TNAA: 3
\This is the Attribute 2 TN you roll against\
TNAB: 4
\This is the Attribute 3 TN you roll against\
TNAC: 7
\This is the Skill TN you roll against\
TNS: 5


\This function explodes the skilldice\
function: explode R:n on N:n {
 if N = R { result: 1 + [explode R on SKILLDICE] }
 result: N <= TNS
}
EXPLODED: [explode 1 on SKILLDICE]

\This function compares Attrib 1 to 3 against the Attribute TN\
function: compare ATA:n ATB:n and ATC:n {
if (ATA <= TNAA) & (ATB <= TNAB) & (ATC <= TNAC) {result: 3}
if (ATA <= TNAA) & (ATB <= TNAB) {result: 2}
if (ATA <= TNAA) & (ATC <= TNAC) {result: 2}
if (ATB <= TNAB) & (ATC <= TNAC) {result: 2}
if (ATA <= TNAA) {result: 1}
if (ATB <= TNAB) {result: 1}
if (ATC <= TNAC) {result: 1}
result: 0}

\Finally the output function which yields the result\
output [lowest of [compare A B and C] + SKILLNUMBERdEXPLODED and 10] named "Attribute TNs [TNAA], [TNAB], [TNAC]; and skill [SKILLNUMBER][SKILLDICE] under [TNS]"
