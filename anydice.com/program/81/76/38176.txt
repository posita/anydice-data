\These functions are used to calculate average damage for the Dice Battler game. It will calculate hit chance and damage using 3 dice.\


function: calc hit HIT DAMAGE {
 SUM: (HIT >= 1d6)*DAMAGE
 result: SUM
}

\BASE - Basic Attack + Basic Attack + Basic Attack\
BASE: [calc hit 1d6 4] + [calc hit 1d6 4] + [calc hit 1d6 4]
output BASE named "BASE"

\KNIGHT - Giant Slash + Sword Strike\
KNIGHT_HIT: [calc hit 2d6 6] + [calc hit 1d6 4]
output KNIGHT_HIT named "Knight Hit"

\MAGE - Magic Bolt [Empowered + Accuracy]\
MAGE_HIT: [calc hit 1d6+1 7]
output MAGE_HIT named "Mage Hit"

\TANK - Charge + Shield Bash\
TANK_HIT: [calc hit 2d6-2 5] + [calc hit 1d6+1 4]
output TANK_HIT named "Tank Hit"

\ROGUE - [SMOKE BOMB + HIDE] + Sneak Attack\
ROGUE_HIT: [calc hit (1d6) 14]
output ROGUE_HIT named "Rogue Hit"
