MISS: 0 \Resultados 0 sao erros\
HIT: 1  \Resultados 1 sao acertos\
CRITICALHIT: 2 \Resultados 2 sao acertos criticos\

function: attack ROLL:n vs DEFENSE:n {
 if ROLL = 1 { result: MISS }
 if ROLL = 20 {
  if ROLL + ATTACK >= DEFENSE { result: CRITICALHIT }
  result: HIT
 }
 if ROLL + ATTACK >= DEFENSE { result: HIT }
 result: MISS
}

ATTACK: 3 \Coloque o modificador de atributo\
loop AC over {5..30} {
 output [attack d20 vs AC] named "[AC]"
}
