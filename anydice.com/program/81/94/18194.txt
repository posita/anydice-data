function: evaluate ROLL:s {
 SUCCESSES: ROLL >= 8
\This is our TN, but it's backwards. 10=0, 9=1, 8=2, etc. Because our TN doesn't include ties, however, we have to shift it by 1. Good rule of thumb: 11-GAMETN = ANYDICETN (for a game TN of 3, use >= 8; hits on 8, 9, and 10 in anydice (3 opportunities), for 0, 1, and 2 in TitV.
\
 if SUCCESSES > 0 { result: SUCCESSES + (ROLL = 10) }
\IF the variable SUCCESSES is greater than 0, the RESULT will be SUCCESSES + an additional ROLL set to 10 (automatically qualifying as >= the TN), or in other words, RESULT will be SUCCESSES + 1
\
 result: 0
\The Untrue result. If the number of SUCCESSES, equal to the rolls with results >= our number, is NOT >0, THEN the result is 0.\
}

loop DICE over {1..5} {
\This just tells it to do this over and over for 1d, 2d, 3d, 4d, and 5d
\
 output [evaluate DICE d10] named "[DICE]d"
}
