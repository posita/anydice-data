function: X:n conditional {

if X - MOD = 1 {result: 0} \Critical Failure\

if X - MOD = 20 {result: 3} \Critical Hit\

if X >= AC {result: 2} \Hit\

else { result: 1} \Miss\
}

\
0 = Critical Miss
1 = Miss
2 = Hit
3 = Critical Hit
\

AC: 19
MOD: 3
ADV: [highest 1 of 2d20]
DIS: [lowest 1 of 2d20]

output [ADV + MOD conditional] named "Advantage against AC [AC], with +[MOD] to hit."
output [d20 + MOD conditional] named "Normal against AC [AC], with +[MOD] to hit."
output [DIS + MOD conditional] named "Disadvantage against AC [AC], with +[MOD] to hit."
