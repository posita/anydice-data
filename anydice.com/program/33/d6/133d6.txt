\**** Damage calculator, Source (slightly edited to fit my use case) https://dontstopthinking.com/dd-5e-calculations-using-anydice/ ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

\**** Hex, Radiant Soul, and Hexblade's Curse ****\
ATTACKMOD: 4
CHARACTERLEVEL: 6
PROFICIENCY: 3
ACCBONUS: 0
CRITROLL: 19
ATTACKTIMES: 1
BASEATTACKDAMAGE: 1d10
HEX: 1d6
RADIANTSOUL: CHARACTERLEVEL
HEXBLADESCURSE: PROFICIENCY
DAMAGE: BASEATTACKDAMAGE + HEX + ATTACKMOD + HEXBLADESCURSE + RADIANTSOUL
CRITDAMAGE: (BASEATTACKDAMAGE * 2) + (HEX * 2) + ATTACKMOD + HEXBLADESCURSE + RADIANTSOUL
ELDRICHBLASTONE: ATTACKTIMESd[attackroll d20 vs ENEMYAC]
output ELDRICHBLASTONE named "Eldrich blast 1"
