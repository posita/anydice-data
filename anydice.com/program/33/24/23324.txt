\ To change the number of AD simulated to X AD, change "XD" as appropriate at the bottom \
set "explode depth" to 10
EXPLODED: [explode d2-1]
function: explodea ROLLEDVALUE:n {
 if ROLLEDVALUE = 6 { result: 2 + [explodea d6] }
 if ROLLEDVALUE = 5 { result: 1 }
 if ROLLEDVALUE = 4 { result: 1 }
 if ROLLEDVALUE = 3 { result: 1 }
 if ROLLEDVALUE < 3 { result: 0 }
}
function: explodeb ROLLEDVALUE:n {
 if ROLLEDVALUE = 3 { result: 1 + [explodeb d3]}
 if ROLLEDVALUE = 2 { result: 1 }
 if ROLLEDVALUE = 1 { result: 0 }
}
EXPLODEDA: [explodea d6]
EXPLODEDB: [explodeb d3]
\ The number below is the number of dice simulated \
output 5d EXPLODED named "Original Rules"
output 5d EXPLODEDA named "Option A"
output 5d EXPLODEDB named "Option B"
output 5d{0,0,1,1,1,3} named "Option C"
