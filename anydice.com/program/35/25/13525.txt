\**** Damage calculator full basic damage dealing list ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}



ATTACKMOD: 4
DMGBONUS: 0
PROFICIENCY: 1
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 1
DAMAGE: 1d8 + 2d6 + 2*ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d8 + 4d6 + 2*ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll [highest 1 of 1d20] vs ENEMYAC] named "Bear damage if proned"
