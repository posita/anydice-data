\What are the effect of different modifiers?\
\A +1 modifier is equivalent to -1 decrease of target number\
\Modifiers and target number can thus be seen as the same thing mathematically... lets call it dif\
\Lets assume the average action is 8 target number\
\Dif = 8 - difficulty + modifiers\
\We're calculating dif here\

N: 2d6

function: semiexplode N:n {
 if  N = 12 {result:12 + d6}
 else {result: N}
}

NONE: [semiexplode N] >= 8
P_ONE: [semiexplode N] +1 >= 8
P_TWO: [semiexplode N] +2 >= 8
P_THREE: [semiexplode N] +3 >= 8
P_FOUR: [semiexplode N] +4 >= 8
P_FIVE: [semiexplode N] +5 >= 8
P_SIX: [semiexplode N] +6 >= 8
M_ONE: [semiexplode N] -1 >= 8
M_TWO: [semiexplode N] -2 >= 8
M_THREE: [semiexplode N] -3 >= 8
M_FOUR: [semiexplode N] -4 >= 8
M_FIVE: [semiexplode N] -5 >= 8
M_SIX: [semiexplode N] -6 >= 8

output NONE named "Dif 0"
output P_ONE named "+1"
output P_TWO named "+2"
output P_THREE named "+3"
output P_FOUR named "+4"
output P_FIVE named "+5"
output P_SIX named "+6"
output M_ONE named "-1"
output M_TWO named "-2"
output M_THREE named "-3"
output M_FOUR named "-4"
output M_FIVE named "-5"
output M_SIX named "-6"
