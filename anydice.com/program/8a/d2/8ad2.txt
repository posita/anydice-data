\Armor Values\
ARMLVL: 4
ARMTIER: 5
PREARM: d{0..(ARMTIER*(2+ARMLVL))}
POSTARM: 2d{(ARMTIER+ARMLVL)..(ARMTIER*(2+ARMLVL))}/2

\Damage Values\
PREDMG: 2d{12..16}/2
POSTDMG: 2d{12..20}/2 

\Function to treat negative damage as 0\
function: A:n above zero {
  if A > 0 { result: A }
  result: 0
}

\Actual output commands\
output [PREDMG - PREARM above zero]
output [POSTDMG - POSTARM above zero]
