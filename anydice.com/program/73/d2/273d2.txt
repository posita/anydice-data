PROFICIENCY: 3-5
ABILITY_BONUS: 5
MAGIC_BONUS: 1
TO_HIT: (PROFICIENCY+ABILITY_BONUS+MAGIC_BONUS)
CRIT_ON: 20
DAMAGE_DICE_MA: 1d8
DAMAGE_STATIC: (ABILITY_BONUS+MAGIC_BONUS+10)
function: attack ROLL:n vs AC:n with DAMAGE_DICE:n {
 if ROLL = 1 { result: 0}
 if ROLL >= CRIT_ON { result: DAMAGE_DICE + DAMAGE_DICE + DAMAGE_STATIC}
 if ROLL + TO_HIT >= AC { result: DAMAGE_DICE + DAMAGE_STATIC}
 result: 0
}
EDV_MA: [attack d20 vs 18 with DAMAGE_DICE_MA]
EDV_MA_ADV: [attack [highest 1 of 2d20] vs 18 with DAMAGE_DICE_MA]
EDV_A: EDV_MA + EDV_MA
EDV_B: EDV_MA_ADV + EDV_MA_ADV
output EDV_A named "EDV Two Attacks GWM"
output EDV_B named "EDV Two Attacks Reckless GWM"
