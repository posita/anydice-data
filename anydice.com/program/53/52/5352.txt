function: troll RR:s FF:n WW:n {
A: RR
if A >= WW {result: 1}
else {
if A <= FF {result: -1}
else {
if A >= FF & A <= WW {result:0}}}
}

function: bang RR:s FF:n WW:n {
B: [troll d20 FF WW] + [troll d20 FF WW] + [troll d20 FF WW]
result: B
}

loop N over {1..10} {
F: N
W: [lowest of N + 10 and 20]
output [bang 1d20 F W] named "low [F] high [W]"
}
