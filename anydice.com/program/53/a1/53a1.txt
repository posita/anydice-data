\ !! This looks best in the Transposed view !! \

DIE: d6 \ Die type \
COMP: 5 \ The pool we're comparing to \

function: wintielose A:n B:n {
 if A > B { result: 1 } \ A wins \
 if A < B { result: -1} \ B wins \
 result: 0 \ Neither wins: tie \
}

loop N over {1..10} {
 output [wintielose COMPdDIE NdDIE] named "[COMP]d[DIE] v [N]d[DIE]"
}
