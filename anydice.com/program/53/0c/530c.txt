\ I want to roll a x number of dice. in this example 5. then count the 1s,2s,3s and 4s rolled (and i want to roll at least 3 of these).
  Then roll again a x number of dice, 5 again for the example and check how many matches i got (if rolled 1,2,3 i need a 1,2,3)  and i want at least 3 matches.
  Also the result of 6 counts like a match for 1,2,3,4.
  So to recap, i search for the possibility to match at least 3 of the possible (1,2,3,4) results of the 1st roll with the second roll, (having also a 6 as a wild)
\



function: check ROLL:s {
 if (ROLL = 1 | ROLL = 2 | ROLL = 3 | ROLL = 4) > 2 {
  result: 5d(d6 = ROLL | d6 = 6) > 2 
 }
 result: 0
}
