\ Reroll function. \
function: reroll N:n once {
 if N < 3 { result: WD }
 result: N
}

\ WD is the weapon die. Change as you wish. \
WD: d12

\ GW becomes the weapon die with great-weapon reroll applied. \
GW: [reroll WD once]

\ Output a bunch of weapon dice. Change the loop range as desired. \
loop N over {1..5} {
 output NdGW named "[N] dice"
}
