\this is for MOS: close combat in which dice block each other, but surviving hits are not subtracted from the other side's total.\

function: ROLL:s blocked by POWER:s {
R: 0
PP: 1
loop P over {1..#ROLL} {
if P@ROLL < 4 { result: R }
if P@ROLL > PP@POWER { R: R + 1 }
\if P@ROLL = PP@POWER { R: R + 1 }\
else { PP: PP + 1 }
}
result: R
}

function: A:s p PA:s vs B:s p PB:s {
result: [A blocked by PB] + (PA > 3)
}
\output [0 p 2d6 vs 2d6 p 0]\

output [0 p 2d6 vs 0 p 0]
