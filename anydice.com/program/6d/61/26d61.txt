\** Attack Roll Function **\
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + ATTACK >= AC { result: DAMAGE }
 result: 0
}

\** Attack looped against enemy AC from 18-25 **\
ENEMYAC: d{15..20}

\** Edit Here **\
ATTACK: 19
DMGBONUS: 16
CRITROLL: 20
ATTACKTIMES: 2
DAMAGE: (1d8 + DMGBONUS)
CRITDAMAGE: (3d6 + (3*(DMGBONUS)))

\** Output **\
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "normal attack"
