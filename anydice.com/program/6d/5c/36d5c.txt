function: BONUS:n roll D:n atkroll RETURN:n crit CRIT:n   {
    AC: 7 \ base 65% chance to hit, using the BONUS input to modify the resulting value. \
      
    if D = 20 { result: RETURN+CRIT } \ natural 20s in DND are critical hits, they always hit and deal twice the dmg \  
    else if D = 1 { result: 0}
    else if D + BONUS > AC { result: RETURN } 
    else { result:0}
}
