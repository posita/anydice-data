\**** Damage calculator ****\

\Nerd stuff.\
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\Modify the AC here!\
ENEMYAC: d{17}

\Velociraptor Bite Attack.\
ATTACKMOD: 2
DMGBONUS: 0
PROFICIENCY: 2
ACCBONUS: 0
CRITROLL: 20
DAMAGE: 1d6 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d6 + ATTACKMOD + DMGBONUS
output [attackroll [highest 1 of 2d20] vs ENEMYAC] named "VeloBite"

\Velociraptor Claw Attack.\
ATTACKMOD: 2
DMGBONUS: 0
PROFICIENCY: 2
ACCBONUS: 0
CRITROLL: 20
DAMAGE: 1d4 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d4 + ATTACKMOD + DMGBONUS
output [attackroll [highest 1 of 2d20] vs ENEMYAC] named "VeloClaw"
