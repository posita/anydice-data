\ DW Red Dice. Example: [5red4]: roll 5d6, 4 and 5 is 1 hit, 6 is 2 hits and a reroll \
function: N:n red H:n {
  if H  < 1 {result: [Nred1]}
  if H  > 6 {result: [Nblue6]}
  result: Nd [explode d{0:H-1, 1:6-H,2}]
}


\ DW Blue Dice. Example: [5blue4]: roll 5d6, 4 and 5 is 1 hit, 6 is 2 hits\
function: N:n blue H:n {
  if H  < 1 {result: [Nblue]}
  if H  > 6 {result: [Nblue6]}
  result: Nd{0:H-1, 1:6-H,2}
}

\ High Payload Dice. (for covenant mines) Example: [5hp4]: roll 5d6, 4 and 5 is 2 hits, 6 is a reroll \
function: N:n hp H:n {
  result: Nd[hp [H between 1 and 6] exp d6]
}

\ auxilliary function for hp explosion thingie \
function: hp H:n exp E:n{
  INITIAL: d{0:H-1, 2:7-H}
  if E = 6 {result: INITIAL + [hp H exp d6]}
  result: INITIAL
}

\ covenant shields! Only red dice for now \
function: N:n cshield H:n {
  SHIELD: [NredH]
  result: SHIELD+[reroll[lowest 1 of SHIELD] red H ]
}

\ auxilliary function for inventive scientists \
function: reroll N:n red H:n  {
  if N = 0 { result: [1redH] }
  result: 0 
}

\ simple trim function \
function: D:d between L:n and H:n {
  result: [lowest of H and [highest of D and L]]
}

\calculates the amount of damage caused to a target. Does not include double crits\
function: N:n vs dr DR:n cr CR:n {
  if N >= CR  {result: 2}
  if N >= DR  {result: 1}
  result:0
}
