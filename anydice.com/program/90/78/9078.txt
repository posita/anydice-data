\ output [ATTACKS vs AC crit CRITRANGE MULT CRITEXTRA dmg DAMAGE EXTRA] named "My Weapon"
This function calculates how much damage you do on an attack
ATTACKS is your attack bonus (e.g. 10), this can include iterative attacks (e.g. {6,1})
AC is the opponent's armor class (e.g. 17)
CRITRANGE is the lower number of the critical threat range (e.g. 19 if your threat is 19-20)
MULT is your critical multiplier (e.g. 2)
CRITEXTRA is extra damage dice on a critical hit (e.g. 1d10 for flaming burst) otherwise leave blank
DAMAGE is the damage you deal with your weapon not including extra dice (e.g. 1d8+2)
EXTRA is extra damage dice (e.g. 1d6 for a sneak attack) otherwise leave blank \

function: ATTACKS:s vs AC:n crit CRITRANGE:n MULT:n CRITEXTRA:d dmg DAMAGE:d EXTRA:d {
  SUM: {0}
  loop ATTACK over ATTACKS {
    SUM: SUM + [1d20 1d20 plus ATTACK ac AC crit CRITRANGE MULT CRITEXTRA dmg DAMAGE EXTRA]
  }
  result: SUM
}
function: ATTACKS:s vs AC:n crit CRITRANGE:n MULT:n dmg DAMAGE:d {
  result: [ATTACKS vs AC crit CRITRANGE MULT 0 dmg DAMAGE 0]
}
function: ATTACKS:s vs AC:n crit CRITRANGE:n MULT:n CRITEXTRA:d dmg DAMAGE:d {
  result: [ATTACKS vs AC crit CRITRANGE MULT CRITEXTRA dmg DAMAGE 0]
}
function: ATTACKS:s vs AC:n crit CRITRANGE:n MULT:n dmg DAMAGE:d EXTRA:d{
  result: [ATTACKS vs AC crit CRITRANGE MULT 0 dmg DAMAGE EXTRA]
}
function: ROLL:n CRITROLL:n plus ATTACK:n ac AC:n crit CRITRANGE:n MULT:n CRITEXTRA:d dmg DAMAGE:d EXTRA:d {
  HIT: 0
  THREAT: 0 
  if (ROLL = 1) { \automatic miss\
  } else if (ROLL = 20) { \automatic hit\
    HIT: 1
    THREAT: 1
  } else if (ROLL + ATTACK < AC) { \missed\
  } else { \hit\
    HIT: 1
    if (ROLL >= CRITRANGE) { 
      THREAT: 1
    }
  }
  if (!HIT) { result: 0 }
  if (THREAT & (CRITROLL + ATTACK >= AC | CRITROLL = 20) & CRITROLL != 1) {
    result: [crit MULT DAMAGE] + EXTRA + CRITEXTRA 
  }
  result: DAMAGE + EXTRA
}
function: crit MULT:n DAMAGE:d {
  SUM: {0}
  loop N over {1..MULT} {
    SUM: SUM + DAMAGE
  }
  result: SUM
}
