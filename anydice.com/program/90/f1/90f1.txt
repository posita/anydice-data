\Function to check if two numbers are the same\
function: compare X:d and Y:d {
 result: X = Y
}

\Function to roll a number of dice; each pair of values adds that value (triples add it twice, quadruples thrice, etc.) \
function: look for matches in DICE:d {
 VALUE: 0
 loop N over {1..#DICE} {
  loop M over {N+1..#DICE} {
   MATCH: [compare N@DICE and M@DICE]
   if MATCH {
    VALUE: VALUE + N@DICE
   }
  }
 }
 if VALUE = 0 {
  VALUE: 1
 }
 result: VALUE
}

output [look for matches in 3d6]
