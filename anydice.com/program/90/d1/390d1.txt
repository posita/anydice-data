\This function checks and outputs the number of successes, using dice that explode on success. 0 for no success, then 1 for each. Contains a 1 die function for using the integrated roller and a multi die function for checking total probabilities.\

function: alttrans R:n again N:n {
 if N >= R { result: 1 + [alttrans R again d4] }
 result: N = 4
}

ALTERNATE: [alttrans 4 again d4]

\Swap all instances of "4"s to "10"s to check an exploding 10% probability instead. Altering the number needed for success can capture other probabilities that can be expressed as fractions.\
\For instance, a 2 in 7 chance could be expressed by changing all dice to d7, making result: N = 6, and ALTERNATE [alttrans 6 again d7]\

loop N over {1,6} { 

\{1,6} indicates results of 1 individual die and 6 total transformations, in order.  The roller allows repeated rolls of one die; using the latter will instead output total count of alt transformations.\

 output [lowest of 10 and NdALTERNATE] named "[N]alternate transformations"
}

\"lowest of 10 and NdALTERNATE" = function depth for checking probabilities. Change and check export tab to look at rarer outcomes\

\"Standalones" just counts selected standalone transformations and will need to be altered character to character. d38 max, including all mutually exclusive options\
