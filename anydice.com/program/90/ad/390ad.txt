\ Tiny-Systeme: 
  * W6-Pool: 2W6, 1W6 Nachteil, 3W6 Vorteil
  * Erfolg wenn mindestens eine 5 oder eine 6 gewürfelt wird
  * Nach Zielen (Focus) zählt auch eine 4 als Erfolg          \

loop N over {1..3} {
  output [count {5, 6} in Nd6] > 0 named "[N]d: Erfolg mit 5 und 6"
}

loop N over {1..3} {
  output [count {4, 5, 6} in Nd6] > 0 named "[N]d: Erfolg mit 4, 5 und 6"
}
