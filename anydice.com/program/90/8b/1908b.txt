\\\\\ Change these variables \\\\\
NUMBEROFDICE: 1    \ Number of dice you roll with a regular damage roll, e.g. "1" for a longsword, "2" for a greatsword. \
DIEFACES: 8        \ How many faces the damage die has, e.g. "8" for 1d8, "6" for 1d6 or 2d6. \
EXTRADICE: 0       \ Number of dice you roll *in addition* to the dice you get from a normal critical hit, e.g. Savage Attacker.\


\\\\\ Main program \\\\\
output (NUMBEROFDICE)dDIEFACES named "Regular attack"
output ((NUMBEROFDICE*2)+EXTRADICE)dDIEFACES named "RAW critical hit"
output ((NUMBEROFDICE*2)+EXTRADICE)d[highest of (DIEFACES/2) and 1dDIEFACES] named "At least half"
output NUMBEROFDICEdDIEFACES+(NUMBEROFDICE*DIEFACES+EXTRADICE*DIEFACES) named "Crit damage maximized"
