\Definition of main function, A are actions invested, B are free actions from grudges or other sources that do not proc bonuses\
function: calculate effective A actions with B free actions{ result: [bonus actions A sets of three]+ AdX + B }

\Constants for user modification\
\A: 4 Actions invested current defined in execution loop\
F: 0 \Free Actions\
Y: 1 \Bonus actions per set of two\
Z: 3 \Bonus Actions per set of three\

\function calculating probability for free action proc\
X:1d10 > 4


\Helper to prevent overflows, if you need to change this you're probably way out of scope\
set "maximum function depth" to 5



\Functions for calculating procs from action sets\
function: bonus actions I sets of two { 
  if I>1 {
    B: I -2
    result: [bonus actions B sets of two]+2+Y
  } else {
    result:  I 
  } 
}

function: bonus actions I sets of three{ 
  if I>2 {
    B: I -3
    result: [bonus actions B sets of three]+3+Z
  } else {
    result:  [bonus actions I sets of two] 
  } 
}

loop A over {1..6}{
 output [calculate effective A actions with F free actions] named "expected range for [A] actions"
}
