\
 Trophy Gold Hunt
 0: Lose all hunt tokens and encounter something terrible
 1: You encounter something terrible
 2: Gain one hunt token and encounter something terrible
 3: Gain one hunt token
\

function: hunt ROLL:s {
 if 1@ROLL = 1 {
  result: 0
 }
 if 1@ROLL = 2 | 1@ROLL = 3 {
  result: 1
 }
 if 1@ROLL = 4 | 1@ROLL = 5 {
  result: 2
 }
 result: 3
}

loop P over {1..2} {
 output [hunt Pd6] named "[P]d"
}
