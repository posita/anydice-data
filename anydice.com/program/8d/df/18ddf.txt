function: luck ROLLA:n and ROLLB:n{
 RES: 0

 if ([count {1..3} in ROLLA]) {
   RES: RES +1
 }
 if ([count {8..10} in ROLLA]) {
   RES: RES -1
 }

 result: RES
}
LUCK: [luck d10 and d10]

N: 4
DADO: 10
DIFS: {1..10} \* HABS: {0..10} *\

loop D over DIFS {
  output [count {1..11-D} in NdDADO] + LUCK named "DIF [D] ([N]D[DADO])"
}
