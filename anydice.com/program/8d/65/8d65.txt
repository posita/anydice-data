function: diceroll DIE:n DIFFICULTY:n {
 \if DIE = 10 {result: 2}\
\Uncomment the line above if using Specialty feats.\
 if DIE >= DIFFICULTY {result: 1}
 if DIE = 1 {result: -1}
 if ((DIE < DIFFICULTY) & (DIE !=1)) {result: 0}
}
\ This is the rule of rolling one single die.\
\1 is a Success, 0 is a Fail, -1 is a Botch.\
