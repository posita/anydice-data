WINB: 1d20 + 2@ROLLS > DC
WINC: 1d20 + 3@ROLLS > DC

function: highest of A:n and B:n and C:n {
 if A > B {
  if A > C { result: A }
  result: C
 }
 if B > C {
  result: B
 }
 result: C
}
