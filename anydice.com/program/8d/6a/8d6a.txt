function: diceroll DIE:n vs DIFFICULTY:n {
\if DIE = 10 {result: 2}\
\Uncomment this line if rolling for a trait in which the character is specialized.\
if DIE >= DIFFICULTY {result: 1}
if DIE = 1 {result: -1}
result: 0
}
\This sets the rule for rolling a die in oWoD.\
\1 means a success, -1 means in rolling a “1” and losing a success, 0 means neither.\

function: evaluate ROLL:s {
if (ROLL > 0) > 0 {
result: [highest of 0 and ROLL]
}
result: [highest of -1 and ROLL]
}
\This function checks if there are successes in a given sequence of roll results. If there are any, it returns either 0 (a fail) or a greater number (the amount of successes). It can’t be a botch, as at least one success was rolled. If no successes are spotted, it returns -1 if there is at least one roll result of losing a success, and returns 0 if there is no such a result.\

DICEROLL: [diceroll d10 vs 6]
\Sets the die type and difficulty.\
AUTOSUCCESSES: 0

if AUTOSUCCESSES > 0 {output [highest of 0 and [evaluate P d DICEROLL]]}
else {output [evaluate P d DICEROLL] + AUTOSUCCESSES named "Dicepool of [P]"}}
\Loop calcuates things for every dicepool in a given range.\
