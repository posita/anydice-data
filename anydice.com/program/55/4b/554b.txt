BASEDAMAGE: 0
DUALDAMAGE: 0

BASEKNIFE: 3d6 <= 16
DUALKNIFEA: 3d6 <= 12
DUALKNIFEB: 3d6 <= 12


BASEDAMAGE: BASEKNIFE * (BASEDAMAGE + (1d6 + 1)*4)


DUALDAMAGE: (DUALKNIFEA * (DUALDAMAGE + (1d6 + 1)*4)) + (DUALKNIFEB * (DUALDAMAGE + (1d6+1)*4))


output BASEDAMAGE named "Normal knife damage is [BASEDAMAGE]"
output DUALDAMAGE named "Dual knife damage is [DUALDAMAGE]"
