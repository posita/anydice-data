SUCCESS: {28}
TOP: 12*3
FRIENDMIN: 3

R: 3d(d{FRIENDMIN..15})
A: 3d(d{0..15})
L: d{0..19}
LR: 3d(d{12..15})

function: reroll A:n with B and L:n chance of LR {
 if (A >= SUCCESS) {
  result: A
 } else {
  if L > 0 {
   result: B
  } else {
   result: LR
  }
 }
}
\output A
output R\
FINAL: [reroll A with R and L chance of LR] 
output FINAL >= SUCCESS
output FINAL > TOP
output FINAL
