\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

\Attack\
ATTACKMOD: 5
WEAPONDIE: 1d4
DMGBONUS: 8
DICEDMGBONUS: 0
PROFICIENCY: 4
ACCBONUS: -2
CRITROLL: 19
DAMAGE: WEAPONDIE + DICEDMGBONUS + ATTACKMOD + DMGBONUS
CRITDAMAGE: ((WEAPONDIE + DICEDMGBONUS)*2) + ATTACKMOD + DMGBONUS
output [attackroll [highest 1 of 3d20] vs ENEMYAC] named "Elven Accuracy"
