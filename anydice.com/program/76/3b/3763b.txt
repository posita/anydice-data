function: successes on DIE:n between LOW:n and HIGH:n with EXTRA:n {
  if DIE = 1 { result: 0 }
  if DIE = 20 { result: 4 }
  if DIE > HIGH { result: 3 }
  if DIE + EXTRA > HIGH { result: 3 }
  if DIE + EXTRA >= LOW { result: 2 }
  result: 1
}

function: old dice roll F:n and S:n {
  if F = 20 { result: 4 }
  if F > 10 { result: 3 }
  else {
    if S = 20 { result: 4 }
    if S > 10 { result: 3 }
    if S > 5 { result: 2 }
    if S = 1 { result: 0 }
    result: 1
  } 
}

\
For these outputs:
0 - Critical Fail
1 - Fail
2 - Partial Success
3 - Success
4 - Critical Success

The function assumes you push the roll if you don't get a full success.
\
