Du får klistra in det här i programrutan.

\
 RESULTAT FÖRKLARING
 0: Det går ordentligt på röven.
 1: Lycka, men med oönskade konsekvenser med 1 eller fler 5or.
 2: Lyckat med 1 sexa.
 3: Lyckat med fler än 2 sexor!
\

function: blades ROLL:s {
 if 1@ROLL = 6 {
  result: 2 + (2@ROLL = 6)
 }
 result: 1@ROLL >= 5
}

loop P over {1..10} {
 output [blades Pd6] named "[P]d"
}
