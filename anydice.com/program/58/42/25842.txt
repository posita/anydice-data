\**** Damage calculator normal Vs advantage ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: 16

ATTACKMOD: 5
DMGBONUS: 3
PROFICIENCY: 3
ACCBONUS: 0
CRITROLL: 19
ATTACKTIMES: 3
DAMAGE: 1d6 + ATTACKMOD + DMGBONUS + 1d6
CRITDAMAGE: 2d6 + ATTACKMOD + DMGBONUS + 2d6 + 2 
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "Mordred full setup"
