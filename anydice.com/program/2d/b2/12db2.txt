\DMG CALC\
 function: attack ROLL:n vs AC:n
{
 if ROLL = 1 { result: 0 }
 if ROLL >= THREAT { result: CRIT } 
 if ROLL + PROF + ABIMOD + TOHIT >= AC { result: DMG }
 result: 0
}
THREAT: 20
TOHIT: 0
TODMG: 0
ABIMOD: 3
DMG: 2d6 + ABIMOD + TODMG
CRIT: DMG*2 + ABIMOD + TODMG
NUMATTACKS: 1
PROF: 2
AC: 16
output NUMATTACKSd[attack d20 vs AC] named "[NUMATTACKS] attacks, [PROF] prof, [ABIMOD] ability vs [AC] AC"
