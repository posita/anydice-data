\*gwf reroller*\ function: gwf DIE:n {GWFDIE: {{1..2}:2, {3..DIE}:DIE+2} result: dGWFDIE } \\
TOHIT: 0 \non-ABIMOD accuracy modifiers\
TODMG: 0 \non-crittable damage\
ABIMOD: 3
DMG: 2d[gwf 6] + ABIMOD + TODMG
CRIT: 4d[gwf 6] + ABIMOD + TODMG
NUMATTACKS: 1
PROF: 2
AC: 16
ATK: [highest 1 of 2d20] \advantage, disadvantage, normal\
THREAT: 20
\*damage calculator*\
function: attack ROLL:n {
 if ROLL = 1 { result: 0 }
 if ROLL >= THREAT { result: CRIT } 
 if ROLL + PROF + ABIMOD + TOHIT >= AC { result: DMG }
 result: 0
}
output NUMATTACKSd[attack ATK] named "[NUMATTACKS] attacks vs [AC] AC"
