\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{10..15}

ATTACKMOD: 5
DMGBONUS: 0
PROFICIENCY: 4
ACCBONUS: -3
CRITROLL: 20
ATTACKTIMES: 1
DAMAGE: 2d8 + 1d6 + 1d8 + 10 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 4d8 + 2d6 + 2d8 + 10 + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll [highest 1 of 2d20] vs ENEMYAC] + ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "67 Rifle Sharpshooter Multiattack on a AC 10-15"
