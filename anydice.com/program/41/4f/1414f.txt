\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

function: bonusattackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: BONUSCRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: BONUSDAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

ATTACKMOD: 5
DMGBONUS: 2
PROFICIENCY: 4
ACCBONUS: 2
CRITROLL: 20
ATTACKTIMES: 1
DAMAGE: 1d8 + ATTACKMOD + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d8 + ATTACKMOD + ATTACKMOD + DMGBONUS
BONUSDAMAGE: 1d4 + ATTACKMOD + ATTACKMOD + DMGBONUS
BONUSCRITDAMAGE: 2d4 + ATTACKMOD + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] + ATTACKTIMESd[attackroll d20 vs ENEMYAC] + ATTACKTIMESd[bonusattackroll d20 vs ENEMYAC] named "Level 12 Aki PAM, Life Drinker Blackadder"
