\
 Hypothetical mechanic Silhoutte by Dreampod 9, but with d12:

 Roll Xd12, take the highest die.
 If it is 1-11, it is your result, otherwise, if it is 12,
 your result is 12 + 1 per each other die that shows twelve.
\

function: silhouette ROLL:s{
 if 1@ROLL = 12 {
  result: 11 + [count 12 in ROLL]
 }
 result: 1@ROLL
}

loop D over {1..6}{
 output [silhouette Dd12] named "[D]d"
}
