\On peut mettre des variables\
\ la compétence\
COMP: 3
\ la caractéristique\
CARAC: 3
\ le bonus fixe \
BONUS: 0
\ la difficulté du jet (ND) \
ND: 25

\ le calcul des probas avec tous les résultats possibles \
CALCUL: [highest CARAC of (CARAC+COMP)d [explode d10]] + BONUS
\ le calcul de la proba de toucher uniquement 0 = raté, 1 = réussi \
output CALCUL > ND named "Proba de réussir"
\ la table complète \
output CALCUL named "Table des probas"
\ la table complète, même résultat que l'exemple de base \
output [highest 3 of 6d [explode d10]] + BONUS

\NOTE : si vous voulez un résultat plus précis, il faudra bosser un peu plus. La fonction "explode" de base relance 10 fois max un dé qui explose en série \
\NOTE : ne rentrez pas de score délirant, c'est très long à calculer avec ce bousin \
