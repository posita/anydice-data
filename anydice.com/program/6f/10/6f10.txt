\
 Hypothetical mechanic Silhoutte by Dreampod 9, but with d20:

 Roll Xd20, take the highest die.
 If it is 1-19, it is your result, otherwise, if it is 20,
 your result is 20 + 1 per each other die that shows twenty.
\

function: silhouette ROLL:s{
 if 1@ROLL = 20 {
  result: 19 + [count 20 in ROLL]
 }
 result: 1@ROLL
}

loop D over {1..6}{
 output [silhouette Dd20] named "[D]d"
}
