\
 Hypothetical mechanic Silhoutte by Dreampod 9, but with d8:

 Roll Xd8, take the highest die.
 If it is 1-7, it is your result, otherwise, if it is 8,
 your result is 8 + 1 per each other die that shows eight.
\

function: silhouette ROLL:s{
 if 1@ROLL = 8 {
  result: 7 + [count 8 in ROLL]
 }
 result: 1@ROLL
}

loop D over {1..6}{
 output [silhouette Dd8] named "[D]d"
}
