\ Degenesis Rolls \
MODI: 0 \ Situational Modifier \

\ Best used using At Least, and using the Table or Graph view. \
\ Value is the amount of Successes \
SHOWTRIGGERS: 0 \ Change to 1 to show triggers instead \

\ ------------------ \
\ ---- Function ---- \
loop N over {(1)..(6+6)}{
 ROLL: (N+MODI)d6
 SUCC: [count {4, 5, 6} in ROLL]
 TRIG: [count {6} in ROLL]
 if SHOWTRIGGERS = 1 {
   output TRIG named "[N]d6 Triggers"
 }
 else {
   output SUCC named "[N]d6"
 }
}
