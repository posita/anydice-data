function: mixedhighest N:n of A:s and B:s with PENALTY:n {
  R: {}
  ROLL: [sort {A, B}]
  loop P over {1..N} { R: {R, P@ROLL - PENALTY} }
  result: R >= 4  \ here, R is still a sequence \
}
