X: 3 \ Dicepool \
G: 2 \ 1s required for a Glitch. \

function: ROLL:s glitch
{
 result: ((ROLL >= 5)>0) & ((ROLL = 1)=G)
}

function: ROLL:s critical glitch
{
 result: ((ROLL >= 5)=0) & ((ROLL = 1)>=G)
}

function: ROLL:s glitch or critical glitch
{
 result: ((ROLL = 1)>=G)
}

output [X d6 glitch] named "Probability of glitch with [X] dice"
output [X d6 critical glitch] named "Probability of critical glitch with [X] dice"
output [X d6 glitch or critical glitch] named "Probability of glitch or critical glitch with [X] dice"
