function: roll D:s under T:n {
 H: 1
 R: 1@D + 2@D + 3@D
 if R <= T {H: 1@D}
 if 1@D = 2@D & 2@D = 3@D {H: H + 2} 
 else if 1@D = 2@D
       | 2@D = 3@D 
       | 3@D = 1@D {H: H + 1} 
 result: H
}
function: D:s from A:n to B:n {
 if D>=A & D<=B {result: 1}
 else {result: 0}
}
