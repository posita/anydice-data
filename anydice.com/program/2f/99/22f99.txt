PROF:3
DEX:5
function: if NUM:n add prof add dex less than AC:n then add BONUS:d {
   if NUM+PROF+DEX<AC {
      result:NUM+BONUS
   }
   else {result:NUM}
}
function: ROLL:d add prof add dex less than AC then add BONUS:d{
   result: [if ROLL add prof add dex less than AC then add BONUS]
}
