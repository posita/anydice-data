\D: # sides. N: # dice.\
\Inputs dice to sequence. Sequence contains 4 placeholders, each of which contain all possible results for 1 die. L iterates over all numbers on the die and counts how many dice rolled that in the sequence, picking the max width. Thus the function outputs L for every given combination and graphs their percentages.\

D: 10
N: 4

function: length ROLL:s {
 L: 1
 loop C over {1..D} {
  L: [highest of L and [count C in ROLL]]
 }
 result: L
}

output [length NdD] named "[N]d[D]"
