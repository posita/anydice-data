function: con save ROLL:n plus BONUS:n vs SAVE:n {
  if ROLL + BONUS < SAVE {
    result: 0
  } else {
    result: 1
  }
}

ADVANTAGE: 1@2d20
STRAIGHT: 1d20
PROF: 3
CON: 3

output [con save STRAIGHT plus CON vs 10] named "Base"
output [con save STRAIGHT plus PROF + CON vs 10] named "Proficiency"
output [con save ADVANTAGE plus CON vs 10] named "Warcaster"
output [con save ADVANTAGE plus PROF + CON vs 10] named "Warcaster Proficiency"
