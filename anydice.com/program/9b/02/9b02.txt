[From http://criticalrole.wikia.com/wiki/Longbow_of_the_Sky_Sentinel]

Once per long rest, the wielder can use a Storm Arrow, which splits 
the next arrow fired from the bow into three separate arrows. 
Each arrow that hits deals an extra 1d6 Lightning damage.
\
function: stormarrow AC:n {
  SUM: 0
  loop N over {1..3} {
    SUM: SUM + [attack 1 AC SENTINELATK SENTINELDMG+1d6]
  }
  result: SUM
}

function: attack NUM:n AC:n ATK:n DMG:d {
  SUM: 0
  loop N over {1..NUM} {
    if ATK >= AC {
      SUM: SUM + DMG
    }
  }
  result: SUM
}
