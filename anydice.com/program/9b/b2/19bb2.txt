\**** Damage calculator normal Vs advantage ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-22\
ENEMYAC: d{15..22}

ATTACKMOD: 5
DMGBONUS: 12
PROFICIENCY: 6
ACCBONUS: -5
CRITROLL: 18
ATTACKTIMES: 3
DAMAGE: 2d6 + 2d6 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 4d6 + 4d6 + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "Fighter GWM attack"
