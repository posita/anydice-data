\Function for weapon/spell with attack rolls (Do not edit)\
function: ROLL:n vs AC:n {
 if ROLL >= CRIT_MIN  {
  result: CRIT_TOTAL
 }
 if ROLL + ATTACK_BONUS >= AC {
  result: HIT_TOTAL
 }
 result: 0
}

\Function for spells/cantrips with saves (Do not edit)\ 
function: ROLL:n SAVE_BONUS:n save vs DC:n {
 if (ROLL + SAVE_BONUS) >= DC {
  result: SAVE_DAMAGE}
 result: FAIL_DAMAGE
}

\ Function for attacks that force saves on hits (Do not edit)\
function: on ATTACK_ROLL:n AC:n hit save SAVE_ROLL:n {
 if ATTACK_ROLL >= CRIT_MIN  {
  result: CRIT_TOTAL + d [SAVE_ROLL SAVE_BONUS save vs DC]
 }
 if (ATTACK_ROLL + ATTACK_BONUS) >= AC {
  result: HIT_TOTAL + d [SAVE_ROLL SAVE_BONUS save vs DC]
 }
 result: 0
}
