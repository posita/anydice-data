\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

function: buttattackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: BUTTCRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: BUTTDAMAGE }
 result: 0
}

function: sattackroll ROLL:n vs AC:n {
 if ROLL >= SCRITROLL { result: SCRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + SPROFICIENCY + SATTACKMOD + SACCBONUS >= AC { result: SDAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

ATTACKMOD: 5+1
DMGBONUS: 10+5+6
PROFICIENCY: 6
ACCBONUS: 0
CRITROLL: 19
ATTACKTIMES: 3
DAMAGE: 1d10 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d10 + ATTACKMOD + DMGBONUS

BUTTDAMAGE: 1d4 + ATTACKMOD + DMGBONUS
BUTTCRITDAMAGE: 2d4 + ATTACKMOD + DMGBONUS

SATTACKMOD: 0
SDMGBONUS: 0
SPROFICIENCY: 4
SACCBONUS: 5
SCRITROLL: 20
SATTACKTIMES: 1
SDAMAGE: 3d6 + SATTACKMOD + SDMGBONUS
SCRITDAMAGE: 6d6 + SATTACKMOD + SDMGBONUS



output ATTACKTIMESd[attackroll 1@3d20 vs ENEMYAC] + 1d[buttattackroll 1@3d20 vs ENEMYAC] + 1d[sattackroll d20 vs ENEMYAC] named "Hexblade lvl 20 turn"
