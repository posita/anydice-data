\ output [ATTACKS vs AC crit THREAT MULT CRITEXTRA dmg DAMAGE EXTRA] named "My Weapon"
This function calculates how much damage you do on an attack
ATTACKS is your attack bonus (e.g. 10), this can include iterative attacks (e.g. {6,1})
AC is the opponent's armor class (e.g. 17)
THREAT is the lower number of the critical threat range (e.g. 19 if your threat is 19-20)
MULT is your critical multiplier (e.g. 2)
CRITEXTRA is extra damage dice only on a critical hit (e.g. 1d10 for a flaming burst sword) otherwise leave blank
DAMAGE is the damage you deal with your weapon not including extra dice (e.g. 1d8+2)
EXTRA is extra damage dice (e.g. 1d6 for a sneak attack) otherwise leave blank \

function: ATTACKS:s CRITATT:n vs AC:n crit THREAT:n MULT:n CRITEXTRA:d dmg DAMAGE:d EXTRA:d {
  SUM: {0}
  loop ATTACK over ATTACKS {
    SUM: SUM + [1d20 plus ATTACK CRITATT ac AC crit THREAT MULT CRITEXTRA dmg DAMAGE EXTRA]
  }
  result: SUM
}
function: ATTACKS:s CRITATT:n vs AC:n crit THREAT:n MULT:n dmg DAMAGE:d {
  result: [ATTACKS CRITATT vs AC crit THREAT MULT 0 dmg DAMAGE 0]
}
function: ATTACKS:s CRITATT:n vs AC:n crit THREAT:n MULT:n CRITEXTRA:d dmg DAMAGE:d {
  result: [ATTACKS CRITATT vs AC crit THREAT MULT CRITEXTRA dmg DAMAGE 0]
}
function: ATTACKS:s CRITATT:n vs AC:n crit THREAT:n MULT:n dmg DAMAGE:d EXTRA:d {
  result: [ATTACKS CRITATT vs AC crit THREAT MULT 0 dmg DAMAGE EXTRA]
}
function: ATTACKS:s vs AC:n crit THREAT:n MULT:n CRITEXTRA:d dmg DAMAGE:d EXTRA:d {
  result: [ATTACKS 0 vs AC crit THREAT MULT CRITEXTRA dmg DAMAGE EXTRA]
}
function: ATTACKS:s vs AC:n crit THREAT:n MULT:n dmg DAMAGE:d {
  result: [ATTACKS 0 vs AC crit THREAT MULT 0 dmg DAMAGE 0]
}
function: ATTACKS:s vs AC:n crit THREAT:n MULT:n CRITEXTRA:d dmg DAMAGE:d {
  result: [ATTACKS 0 vs AC crit THREAT MULT CRITEXTRA dmg DAMAGE 0]
}
function: ATTACKS:s vs AC:n crit THREAT:n MULT:n dmg DAMAGE:d EXTRA:d {
  result: [ATTACKS 0 vs AC crit THREAT MULT 0 dmg DAMAGE EXTRA]
}
function: ROLL:n plus ATTACK:n CRITATT:n ac AC:n crit THREAT:n MULT:n CRITEXTRA:d dmg DAMAGE:d EXTRA:d {
  if (ROLL = 20) { result: [crit 1d20 ATTACK+CRITATT AC MULT DAMAGE] + EXTRA + CRITEXTRA }
  if ((ROLL >= THREAT) & (ROLL + ATTACK >= AC)) { result: ([crit 1d20 ATTACK+CRITATT AC MULT DAMAGE] + EXTRA + CRITEXTRA) }
  result: (ROLL + ATTACK >= AC) * (DAMAGE + EXTRA)
}
function: crit ROLL:n ATTACK:n AC:n MULT:n DAMAGE:d {
  if (ROLL + ATTACK > AC) {
    result: DAMAGE*MULT
  }
  result: DAMAGE
}
\ function: crit MULT:n DAMAGE:d {
  SUM: {0}
  loop N over {1..MULT} {
    SUM: SUM + DAMAGE
  }
  result: SUM
