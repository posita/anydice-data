\ 1. CONFIGURATION \
\ Fate Dice Pools \
FATE_EASY: 1d10
FATE_MEDIUM: 2d10
FATE_HARD: 3d10
FATE_FORMIDABLE: 4d10

\ Define Skill Levels \
SKILL_UNTRAINED: 1
SKILL_NOVICE: 2
SKILL_COMPETENT: 3
SKILL_PROFICIENT: 4
SKILL_SEASONED: 5
SKILL_EXPERT: 6
SKILL_MASTER: 7

\ Define Attribute Dice Tiers \
ATTR_LOW: 1d12       \ Clumsy / Weak \
ATTR_AVG: 1d10       \ Standard \
ATTR_HIGH: 1d8       \ Good \
ATTR_GIFTED: 2d12    \ Entry to Tier 2 \
ATTR_HEROIC: 2d10    \ Superhuman Standard \
ATTR_GODLY: 2d8      \ Maximum possible \

\ 2. THE FUNCTION \
function: check SKILL:n with ATTRIBUTE:d and FATE:d {
 result: [count {1..SKILL} in FATE] + [count {1..SKILL} in ATTRIBUTE]
}
