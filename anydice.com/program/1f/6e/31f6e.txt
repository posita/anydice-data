\ RUINA CALCULATOR \

\ clash A-B, where \ A: 4 B: 7 \, against C-D, where \ C: 2 D: 11

\ The first output shows the winning chances: -1 for A-B win, 0 for draw, and 1 for C-D win \
\ (so they're shown in the same order you input the clash sides in - with draw added in the middle) \
\ The second output shows the expected difference (with the same sign as the first output) \
\ To calculate mass-summations, use sequences as min and max \
\ (for example, C: {2, 3} D: {4, 5} is a summation of 2-4 and 3-5) \
\ The last two graphs are mostly just for some optional perspective on summations \
\ (you might want to switch the view to graph if the last point interests you) \

function: diff X:n vs Y:n {
  result: Y - X
}

function: clash X:n vs Y:n {
  if X > Y { result: -1 }
  if X = Y { result: 0 }
  result: 1
}

function: fix X:s to Y:s {
  L: [lowest of #X and #Y]
  S: 0
  loop N over {1..L} {
    if (N@X > N@Y) { 
      S: S + N@Y
    } else {
      S: S + d{N@X..N@Y}
    }
  }
  result: S
}

output [clash [fix A to B] vs [fix C to D]] named "winner"
output [diff [fix A to B] vs [fix C to D]] named "roll difference ((C-D) - (A-B))"
output [fix A to B] named "roll -1 (A-B)"
output [fix C to D] named "roll 1 (C-D)"
