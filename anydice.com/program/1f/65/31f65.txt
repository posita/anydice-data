\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + ATTACKMOD >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
AC: d{13..15}

ATTACKMOD: 6
DAMAGEMOD: 4
CRITROLL: 20
ATTACKTIMES: 1
DAMAGE: 1d10 + DAMAGEMOD
CRITDAMAGE: 2d10 + DAMAGEMOD
output ATTACKTIMESd[attackroll d20 vs AC] named "Glaive Attack"
