\ 1. CONFIGURATION \
\ Fate Dice are constant (Standard Roll) \
FATE_DICE: 2d10

\ Define Skill Levels \
SKILL_UNTRAINED: 1
SKILL_LOW: 2
SKILL_AVG: 3
SKILL_AVGPLUS: 4
SKILL_HIGH: 5
SKILL_VHIGH: 6
SKILL_MASTER: 7

\ Define Attribute Dice Tiers \
ATTR_LOW: 1d12       \ Clumsy / Weak \
ATTR_AVG: 1d10       \ Standard \
ATTR_HIGH: 1d8       \ Good \
ATTR_GIFTED: 2d12 \ Gifted \
ATTR_HEROIC: 2d10 \ Heroic \
ATTR_GODLY: 2d8      \ Maximum possible attribute \

\ 2. THE FUNCTION \
\ Counts successes in Fate Pool + successes in Attribute Pool \
function: check SKILL:n with ATTRIBUTE:d {
 result: [count {1..SKILL} in FATE_DICE] + [count {1..SKILL} in ATTRIBUTE]
}
