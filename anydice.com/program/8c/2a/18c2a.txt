DBSEVEN: 2d6+10
DBTEN: 3d8+10
ATK: 30
SNIPER: 3d10

CRITONE: 1d{0:13,1:7}
CRITTWO: 1d{0:13,1:7}

\ Each crit procs Sniper \
output DBTEN + CRITONE * (DBSEVEN + SNIPER) + CRITTWO * (DBSEVEN + SNIPER) + ATK
\ Either crit procs Sniper \
output DBTEN + CRITONE * DBSEVEN + CRITTWO * DBSEVEN + (CRITONE | CRITTWO) * SNIPER + ATK
\ Both must crit to proc Sniper \
output DBTEN + CRITONE * DBSEVEN + CRITTWO * DBSEVEN + (CRITONE * CRITTWO) * SNIPER + ATK
