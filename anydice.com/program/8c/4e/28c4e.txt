MISS: 0
HIT: 1
CRITICALHIT: 2
PROFICIENCYBONUS: 4
MISSDAMAGE: 0

function: attack ROLL:n vs DEFENSE:n {
 if ROLL = 1 { result: MISS }
 if ROLL = 20 { result: CRITICALHIT }
 if ROLL + ATTACK >= DEFENSE { result: HIT }
 result: MISS
}

function: damage for ATTACKRESULT:n {
 if ATTACKRESULT = CRITICALHIT {
  result: CRITICALDAMAGE
 }
 if ATTACKRESULT = HIT {
  result: HITDAMAGE
 }
 result: MISSDAMAGE
}


\Booming Blade + Crystal Scimitar Melee Attack\
ATTACKMODIFIER: 5
ATTACK: PROFICIENCYBONUS + ATTACKMODIFIER
HITDAMAGE: 1d6 + 3d8 + 5
CRITICALDAMAGE: 2d6 + 6d8 + 5
BBCS: [damage for [attack d20 vs 20]] 

\Booming Blade (Move) + Crystal Scimitar Melee Attack\
ATTACKMODIFIER: 5
ATTACK: PROFICIENCYBONUS + ATTACKMODIFIER
HITDAMAGE: 1d6 + 3d8 + 3d8 + 5
CRITICALDAMAGE: 2d6 + 6d8 + 3d8 + 5
BBCSMOVE: [damage for [attack d20 vs 20]] 

\Crystal Scimitar Melee Attack\
ATTACKMODIFIER: 5
ATTACK: PROFICIENCYBONUS + ATTACKMODIFIER
HITDAMAGE: 1d6 + 1d8 + 5
CRITICALDAMAGE: 2d6 + 2d8 + 5
CS: [damage for [attack d20 vs 20]] 

\2 Booming Blade Crystal Scimitar + 2 Crystal Scimitar\
output 2dBBCS + 2dCS named "Haste Strategy"
output 2dBBCSMOVE + 2dCS named "Haste Strategy (Move)"
