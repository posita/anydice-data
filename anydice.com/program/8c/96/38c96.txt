function: ROLL:n reroll orless MIN:n as REROLL:d 
{
  if ROLL = MIN { 
        result: REROLL   \If the first roll is below or equal to the set minimum value, it rolls again. \
         }
  else { result: ROLL }
}

X: [d6 reroll orless 1 as d6] \This rerolls 1s currently. Replace 1 with the value of your choice. \
output [count {3..6} in 1dX] \This hits on 3s currently. Replace 3 with the threshold of your choice. Replace 1 with the number of attacks of your choice.\
