DIE: d6
ATTDICEPOOL: 2         \ Attackers dice pool. Change this number to anything you want. \
DEFDICEPOOL: 1         \ Defenders dice pool. Change this number to anything you want. \

function: count ATTACKERSDICE:s successes against DEFENDERSDICE:s and take botches into account {
  SUCCESS: [count {1@DEFENDERSDICE..[maximum of DIE]} in ATTACKERSDICE]
  if SUCCESS = 0 {
    result: -(#ATTACKERSDICE <= [count 1 in ATTACKERSDICE])
  }
  result: SUCCESS
}

output [count ATTDICEPOOLdDIE successes against DEFDICEPOOLdDIE and take botches into account] named "[[ATTDICEPOOL][DIE] vs [DEFDICEPOOL][DIE]: -1 is a botch"
