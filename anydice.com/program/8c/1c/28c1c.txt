MISS: 0
HIT: 1
CRITICALHIT: 2
PROFICIENCYBONUS: 4

function: attack ROLL:n vs DEFENSE:n {
 if ROLL = 1 { result: MISS }
 if ROLL = 20 { result: CRITICALHIT }
 if ROLL + ATTACK >= DEFENSE { result: HIT }
 result: MISS
}

function: damage for ATTACKRESULT:n {
 if ATTACKRESULT = CRITICALHIT {
  result: CRITICALDAMAGE
 }
 if ATTACKRESULT = HIT {
  result: HITDAMAGE
 }
 result: MISSDAMAGE
}

MISSDAMAGE: 0

\Booming Blade + Shadow Blade Melee Attack\
ATTACKMODIFIER: 3
ATTACK: PROFICIENCYBONUS + ATTACKMODIFIER
HITDAMAGE: 6d8 + 3
CRITICALDAMAGE: 12d8 + 3
BBSB: [damage for [attack [highest 1 of 2d20] vs 20]]
