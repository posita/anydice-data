\ function to calculate the chance of fumble when you have to fail AND have two or more 1s to fumble\
function: roll ROLL:s target TARGET:n {
  if (ROLL >= TARGET) { result: 1 }           \ success \
  if (ROLL = 1) >= 2 { result: -1 }           \ fumble  \
  result: 0                                   \ failure \
}
loop T over {5} {                             \ change target number here \
  DIE: {1..6}
  loop N over {2..12} {                       \ change dice pool size here \
    output [roll NdDIE target T] named "[N]d10 vs. [T] (-1 = fumble, 0 = fail, 1 = success)"
  }
}
