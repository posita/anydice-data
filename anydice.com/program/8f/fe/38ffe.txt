

\ COMPARING the DPR of TWO ATTACKERS AGAINST THE SAME ENEMY IN WEIRD WIZARD \

\ DO NOT EDIT BELOW THIS LINE \

ABOONSBANES: ABOONS - ABANES
BBOONSBANES: BBOONS - BBANES

\ rerolls 1's if BRUTAL is 1 \
function: reroll DICE:n as REROLL:d if BRUTAL:n {
  if BRUTAL=1 {
    if DICE=1 {
      result: REROLL
    }
    result: DICE
  }
  result: DICE
}

\ REROLL whole dice set if HIGHEST is 1, thanks to Vilarinof of stackexchange! \
function: highest of two DICE:d if HIGHEST {
  if HIGHEST=1 {
    result: [highest of DICE and DICE]
  }
  result: DICE
}

AREROLLED: [reroll d6 as d6 if ABRUTAL]  \ now we got the new "rerolled dice sequence" for attacker A \
BREROLLED: [reroll d6 as d6 if BBRUTAL]  \ and for attacker B. If they don't have brutal, it's just d6 \

\ Hit Probabilities. Thanks to stealth_elephant on reddit \
