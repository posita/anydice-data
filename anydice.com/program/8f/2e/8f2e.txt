\This compares the damage expectations of a top-tier two handed weapon (Greatsword, 2d6) \
\  against a Duel-wielder using top-tier 1 handed weapons in both hands (Longsword, 1d8) \
\  Greatweapon Fighting Style and Two-weapon fighting style are assumed, as well as the  \
\  feat Duel Wielder to enable strength bonus on the offhanded attack and allowing non-  \
\  light weapon use.                                                                     \

\ ** SET THE STRENGTH MODIFIER TO USE FOR CALCULATIONS ** \
STR_MOD: 0


\This function accounts for GreatWeapon Fighting Style, \
\which re-rolls a 1 or 2, non-recursively               \
function: gwfs N:n {
 if N < 3  { result: 1d6 }
 result: N
}

\ One attack per action \
output 1d8 + STR_MOD + 1d8 + STR_MOD named "DW 1 apa"
output 2d[gwfs 1d6] + STR_MOD named "2H 1 apa"

\ Two attacks per action \
output 2d8 + 2*STR_MOD + 1d8 + STR_MOD named "DW 2 apa"
output 4d[gwfs 1d6] + 2*STR_MOD named "2H 2 apa"
