\These are the dice type for attributes\
ATA: 1d20
ATB: 1d20
ATC: 1d20
\This is the Attribute 1 TN you roll against\
TNA: 14
\This is the Attribute 2 TN you roll against\
TNB: 12
\This is the Attribute 3 TN you roll against\
TNC: 9
\This defines the explode threshold\
TNE: 0

\This function explodes the Attribute Dice\
function: explode R:n on N:n {
 if N = R { result: 1 + [explode R on ATA] }
 if N = R { result: 1 + [explode R on ATB] }
 if N = R { result: 1 + [explode R on ATC] }
 result: N <= TNE
}


\This function compares Attrib 1 to 3 against the Attribute TN for success count\
function: compare ATA:n ATB:n and ATC:n {
if (ATA <= TNA) & (ATB <= TNB) & (ATC <= TNC) {result: 3}
if (ATA <= TNA) & (ATB <= TNB) {result: 2}
if (ATA <= TNA) & (ATC <= TNC) {result: 2}
if (ATB <= TNB) & (ATC <= TNC) {result: 2}
if (ATA <= TNA) {result: 1}
if (ATB <= TNB) {result: 1}
if (ATC <= TNC) {result: 1}
result: 0}
