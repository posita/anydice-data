\This function calculates the amount of crits of a Proton Torpedo.\

function: ROLL:n crits {
  if ROLL > 9 { result: ROLL / 10 + [[ROLL modulo 10] crits]} \ counts all natural crits and starts to counts all Eyes\
  if ROLL > 0 { result: 1 }                          \ if theres at least one Eye add one crit \
  result: 0
}

function: A:n modulo B:n {
  result: A-(A/B)*B
}

output [4d{0,0,1,1,0,0,0,10} crits]
