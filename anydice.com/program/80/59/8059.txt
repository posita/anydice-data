\ This program calculates the probability of a certain amount of hits with Assault Missiles or Plasma Torpedoes and Guidance Chips without a focus. \

function: ROLL:n dice{
  if ROLL = 4 { result: 4 } \ 4 Hits \
  result: ROLL + 1          \ X Hits + Chips \
}

\ This is the actual dice roll. 4 dice are rolled, each of them has the faces: 0,0,0,0,1,1,1,1 \
\ 0 = Blank or Eye, 1 = Hit or Crit \

output [4d{0,0,0,0,1,1,1,1} dice]
