\ This program calculates the probability of a certain amount of hits with Homing Missiles with a focus. \

function: ROLL:n count hits in S:n dice{
  if ROLL >= S*10 - 10 { result: S }                                                  \ with chips all dice are hits \
  if ROLL >= 10 { result: ROLL/10 + [[ROLL modulo 10] count hits in S-ROLL/10 dice] } \ count all hits, then analyze the rest \
  if ROLL >= S - 1 { result: S }                                                      \ with chips all eyes are hits \
  if ROLL >= 1 { result: ROLL + 1 }                                                   \ count all eyes, use chips \
  result: 1                                                                           \ no eyes, use chips \
}

function: A:n modulo B:n {
  result: A-(A/B)*B
}

\ This is the actual dice roll. 4 dice are rolled, each of them has the faces: 0, 0, 1, 1, 10, 10, 10, 10 \
\ 0 = Blank 1 = Eye, 10 = Hit or Crit \

output [4d{0,0,1,1,10,10,10,10} count hits in 4 dice]
