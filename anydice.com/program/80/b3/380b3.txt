\These functions are used to calculate average damage for the Dice Battler game. It will calculate hit chance and damage using 3 dice.\


function: calc hit HIT DAMAGE PIERCE{
 SUM: (HIT >= 1d6)*DAMAGE
 if PIERCE = 1{
 result: SUM +((HIT+3) >= 1d6)*(DAMAGE-1)
} else {
 result: SUM
}
}
