 loop N over {HAND..DRAW-1}{
  HAND: HAND+(1d100<=LANDS)
 }

 result: HAND
}

function: dice one ONE:n greater than dice two TWO:n {
if (ONE > TWO) {result: 1}
if (ONE < TWO) {result: 2}
result: 0
}
