\ This program calculates the probability of a certain amount of hits with Proton Torpedoes and Guidance Chips without a focus. \
\ For Concussion Missile, it's exactly the same logic and probabilities. \

function: ROLL:n dice{
  if ROLL = 40 { result: 4 } \ 4 Hits \
  if ROLL > 30 { result: 4 } \ 3 Hits and at least one Eye \
  if ROLL > 20 { result: 4 } \ 2 Hits and at least one Eye \
  if ROLL > 10 { result: 3 } \ 1 Hit  and at least one Eye \
  if ROLL >  0 { result: 2 } \ 0 Hits and at least one Eye \
  result: 1                  \ only blanks \
}

\ This is the actual dice roll. 4 dice are rolled, each of them has the faces: 0, 0, 1, 1, 10, 10, 10, 10 \
\ 0 = Blank, 1 = Eye, 10 = Hit or Crit \

output [4d{0,0,1,1,10,10,10,10} dice]
