function: savethrow SAVE:n vs dc DC:n for DAMAGE:n half{
if SAVE >= DC {result: DAMAGE/2}
if SAVE < DC {result: DAMAGE}
}

function: savethrow SAVE:n vs dc DC:n for DAMAGE:n suck{
if SAVE >= DC {result: 0}
if SAVE < DC {result: DAMAGE}
}

\attack simulates the average damage of an attack in D&D 5e\
function: attack ROLL:n plus TOHIT:n vs ac AC:n for DICE:d plus FLOAT:n{
if ROLL = 20 {result: DICE + DICE + FLOAT}
if (TOHIT + ROLL) >= AC {result: DICE + FLOAT}
if (TOHIT + ROLL) < AC {result: 0}
}

\attackcrange accommodates for edge cases with the customization of hit damage effects, miss effects, critical hit damage effects, and expanded critical hit range\
function: attackcrange ROLL:n plus BONUS:n vs ac AC:n hit HIT:d miss MISS:d crit CRIT:d crange CRANGE:n{
    if ROLL = 1 {
        result: MISS
    }
    if ROLL >= CRANGE {
        result: CRIT
    }
    if ROLL + BONUS >= AC {
        result: HIT
    }
    result: MISS
}

\reroll replaces the result of a given DIE (any die) with a given minimum (MIN) if the result is lower than the given minimum\
function: reroll DIE:n as REROLL:n if less than MIN:n{
if DIE >= MIN {result: DIE}
if DIE < MIN {result: REROLL}
}

\lucky simulates using the Lucky feat on an attack roll. LUCKY always picks the highest die.\
function: lucky DIEA:n DIEB:n{
if DIEA >= DIEB {result: DIEA}
if DIEA < DIEB {result: DIEB}
}

\READ ME\
\DPR in D&D 5e is usually calculated with a 60% chance to hit. this is known as bounded accuracy. this is because enemy AC increases as player attack modifiers increase, making a 60% chance to hit common at almost all levels of play. however, with magic items or class features that increase your attack roll result, such as a +1 Rod of the Pact Keeper or Archery Fighting Style, one can reach 65% and 70% respectively in ideal environments where enemy AC has not been increased by the DM to counter your high accuracy\

\the attack function can be used to calculate DPR as shown below in EXAMPLE attack\
