function: reroll lowest of ROLL:s as REROLL:d if less than MIN:n {
  LOWEST: (#ROLL)@ROLL       \ the lowest die is sorted last \
  if LOWEST >= MIN {
    result: ROLL + 0
  } else {
    REST: {1..#ROLL-1}@ROLL  \ all but the lowest die \
    result: REST + REROLL
  }
}

output 1d8 + 5 named "Rapier"
output [reroll lowest of 1d8 as d8 if less than 5] + 5 named "Rapier with Piercer"
output 2d8 + 5 named "Rapier Critical"
output [reroll lowest of 3d8 as d8 if less than 5] + 5 named "Rapier Critical with Piercer"
