\ “exploding” d6s that are counted rather than summed \
function: splodeN:n {
 if N = 6 { result: 1 + [splode d6] }
 result: N = {1,6}
}

\ I don’t know why this bit is here? is it just to make the code in the next bit cleaner?\
POOL: [splode d6]

\ combines the exploding skill dice and the non-exploding black dice \
function: heresy S:n vs B:n {
result: SdPOOL - [count {1,6} in Bd6]
}

\ example single-roll output \
\ first number is the pool size (attribute + skill), second number is the difficulty (black dice) \
output [heresy 6 vs 3]

\ calculating all possible combos from 1 to 10 skill dice and 0 to 12 black dice \
loop S over {1..10} {
 loop B over {0..12} {
 output [heresy S vs B] named "[S]d vs [B]d"
}
}
