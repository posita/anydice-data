\ Maid RPG calculator - calculates stress taken in an attack \
\ Negative stress means the attacker takes stress \

\ Attacker's stat \
ATK: 3
\ Attacker's bonus or penalty to die rolls \
AD: 0
\ Attacker's bonus or penalty to the result \
AB: 0
\ Defender's stat \
DEF: 2
\ Defender's bonus or penalty to die rolls \
DD: 0
\ Defender's bonus or penalty to the result \
DB: 0

function: compare AR:n to DR:n {
  if AR = DR {
    result: 0
  } else if AR < DR {
    result: -(DR/ [highest of ATK and 1])
  } else {
    result: AR / [highest of DEF and 1]
  }
}

output [compare [highest of ATK * [highest of d6 + AD and 0] + AB and 0] to [highest of DEF * [highest of d6 + DD and 0] + DB and 0]] named "Stress taken"
