\ Rerolls every R again with an N dice \
function: reroll R:n again N:n {
 if N = R { result: [reroll R again d6] }
 result: N
}

\ Rolls N dices R times and sums them \
function: roll R:n dice N:n {

  SUM: 0
  loop N over {1..R}{
   SUM: SUM+ [reroll 1 again d6]
 }
 result: SUM
}
