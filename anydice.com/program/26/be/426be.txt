MYHIT: 2d6+2

LOWDODGE : 2d6-1
BASEDODGE: 2d6
STRONGDODGE: 2d6+2
GREATDODGE: 3d6+1

BASEDMG: 3
BASECRIT: 15
BASECRITD: 100

LONGDMG: 3
LONGCRIT: 0
LONGCRITD: 100

SPEARDMG: 2
SPEARCRIT: 5
SPEARCRITD: 0

SPEARTOTDMG: SPEARDMG + BASEDMG
LONGTOTDMG: LONGDMG + BASEDMG

SPEARTOTCRIT: BASECRIT + SPEARCRIT
LONGTOTCRIT: BASECRIT + LONGCRIT


LONGTOTCRITD: BASECRITD + LONGCRITD
SPEARTOTCRITD: BASECRITD + SPEARCRITD

HASSPEARCRIT: (1d100<=SPEARTOTCRIT)
HASLONGCRIT: (1d100<=LONGTOTCRIT)

SPEARHIT: SPEARTOTDMG + HASSPEARCRIT*SPEARTOTCRITD*SPEARTOTDMG/100
LONGHIT: LONGTOTDMG + HASLONGCRIT*LONGTOTCRITD*LONGTOTDMG/100

ENEMYDODGE: LOWDODGE

HASHIT: MYHIT>=ENEMYDODGE
HASHITPEN : MYHIT-4 >= ENEMYDODGE

LONGATTQ: HASHIT*LONGHIT
SPEARATTQ: HASHIT*SPEARHIT
TWOATTQ: HASHITPEN*LONGHIT + HASHITPEN*SPEARHIT



output SPEARATTQ
output LONGATTQ
output TWOATTQ
