function: ROLL:n vs AC:n {
if ROLL = 20 { result: CRITICAL}
if ROLL + ATTACK >= AC {
  result: DAMAGE
 }
 result: 0
}

AC: 15
STAT: 4+1
ATTACK: 3+STAT
DAMAGE: (1d12 + STAT)
CRITICAL: (2d12 + STAT)

output [1@2d{2..20} vs AC] + [1@2d{2..20} vs AC] named "+1 Lance with Headband (adv)"
