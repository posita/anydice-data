\ Assumptions \
\ +5 Primary stat \
\ +4 Proficiency \
\ No feats or fighting styles (to keep things simple) \
\ Output is the Average of enemies with AC range 12 to 22 (or average AC = 17 \


\ Generic Magic Weapons \
function: ROLL:n with ATTACKBONUS:n for DMGDIE plus DMGBONUS:n vs AC:n {
 if ROLL = 1 { result: 0 }
 if ROLL = 20 { result: DMGDIE + DMGDIE + DMGBONUS}
 result: (ROLL + ATTACKBONUS >= AC) * (DMGDIE + DMGBONUS) }
output [d20 with +10 for 1d6 plus 6 vs d{12..22}] named "Magic Shortsword +1"
output [d20 with +11 for 1d6 plus 7 vs d{12..22}] named "Magic Shortsword +2"
output [d20 with +12 for 1d6 plus 8 vs d{12..22}] named "Magic Shortsword +3"


\ Sword of Sharpness at various magic levels with the +14 bonus damage only on a nat 20 \
function: ROLL:n with ATTACKBONUS:n for DMGDIE plus DMGBONUS:n vs AC:n {
 if ROLL = 1 { result: 0 }
 if ROLL = 20 { result: 2d6 + DMGBONUS + 14 }
 result: (ROLL + ATTACKBONUS >= AC) * (DMGDIE + DMGBONUS) }
output [d20 with +9 for 1d6 plus 5 vs d{12..22}] named "Shortsword of Sharpness +0"
output [d20 with +10 for 1d6 plus 6 vs d{12..22}] named "Shortsword of Sharpness +1"
output [d20 with +11 for 1d6 plus 7 vs d{12..22}] named "Shortsword of Sharpness +2"
