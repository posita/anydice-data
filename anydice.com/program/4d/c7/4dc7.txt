\ Assumptions \
\ 20 Primary stat \
\ +4 Proficiency \
\ No feats or fighting styles (to keep things simple) \
\ Using Longsword and Greatsword only \
\ Average AC of enemies is 12 to 22 (probably a little low for level 9+ party) \


\ Generic Magic Weapons \
function: ROLL:n with ATTACKBONUS:n for DMGDIE plus DMGBONUS:n vs AC:n {
 if ROLL = 1 { result: 0 }
 if ROLL = 20 { result: DMGDIE + DMGDIE + DMGBONUS}
 result: (ROLL + ATTACKBONUS >= AC) * (DMGDIE + DMGBONUS) }
output [d20 with +11 for 1d8 plus 7 vs d{12..22}] named "Magic Longsword +2"
output [d20 with +11 for 2d6 plus 7 vs d{12..22}] named "Magic Greatsword +2"
