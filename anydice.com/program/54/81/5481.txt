function: N:n dice DIE:n sides reroll MIN choose CHOOSE
{
   \ initialise variables \
   if MIN < 0
   {
      MIN: 0
   }
   X: DIE-MIN 
   DISCARD: N - CHOOSE
   ANS: 0
   
   \ Calculation \
   
   result: {1..CHOOSE}@NdX + CHOOSE * MIN
}

output [3 dice 6 sides reroll 0 choose 3] named "3d6"
