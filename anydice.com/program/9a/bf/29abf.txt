
function: diceroll DIE:n vs DIFFICULTY:n {
if DIE >= DIFFICULTY {result: 1}
if DIE = 1 {result: -1}
result: 0
}

\This sets the rule for rolling a die in SciFantaPunk.\
\1 means a success, -1 means rolling a “1” and losing a success, 0 means neither.\

function: smartevaluate ROLL:s {
