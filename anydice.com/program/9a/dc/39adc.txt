\
CoC Opposed Roll Program
1 = fumble, 2 = failure, 3 = regluar success, 4 = hard success, 5 = Extreme success, 6 = Critcal
If you want to check opposed checks just add 2 and multiply one by 10
\
function : coc roll DICE:s with skill SKILL:n{
 if DICE = 1{
  result: 6
 }
 else if DICE >= 100 {
  result : 1
 }
 else if DICE >= 96 & SKILL < 50{
  result : 1
 }
 else if DICE <= SKILL/5{
  result : 5
 }
 else if DICE <= SKILL/2{
  result: 4
 }
 else if DICE <= SKILL{
  result: 3
 }
 if DICE > SKILL {
  result : 2
 }
}
output [coc roll 1d100 with skill 25] named "Normal Roll"
output [coc roll 1d100 with skill 25]*10 + [coc roll 1d100 with skill 25] "Opposed rolls"
