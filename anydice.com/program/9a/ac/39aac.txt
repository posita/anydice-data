CoC Opposed Roll Program
NO fumble/critical atm
The values of the attacker and defender have been codfied such that 
DEFENDER : 1 = failure, 2 = regluar success, 3 = hard success, 4 = Extreme success
Attacker : 10 = failure, 20 = regluar success, 30 = hard success, 40 = Extreme success
So if the roll is 21 then the attacker gained a regular success and the defender got a failure
\
SKILL : 25
FIFTH : SKILL/5 
HALF : SKILL/2 - FIFTH
REG : SKILL - FIFTH - HALF
MISS : 100-SKILL
ATTACKER : 1d{10:MISS,20:REG,30:HALF,40:FIFTH}

SKILL : 25
CRITICAL : 1
if SKILL >= 50{
 FUMBLE : 1
}
else{
 FUMBLE : 5
}

FIFTH : SKILL/5
HALF : SKILL/2 - FIFTH 
REG : SKILL - FIFTH - HALF
MISS : 100-SKILL - CRITICAL - FUMBLE
DEFENDER : 1d{0:FUMBLE,1:MISS,2:REG,3:HALF,4:FIFTH,5:CRITICAL}
output DEFENDER
output FUMBLE named "fumble"
output MISS named "miss"
output REG named "reg"
output HALF named "half"
output FIFTH named "FIFTH"
output CRITICAL named "critical"
output FUMBLE + MISS+ REG +HALF+FIFTH+CRITICAL named "sum"
