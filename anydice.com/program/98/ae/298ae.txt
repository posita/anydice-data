function: attack with BONUS:n vs AC:n {
  D: d20
  if D = 20 { result: 1 }
  else if D = 1 { result: 0 }
  else if D + BONUS >= AC { result: 1 }
  else { result: 0 }
}

function: advattack with BONUS:n vs AC:n {
  X: [attack with BONUS vs AC]
  Y: [attack with BONUS vs AC]
  if X | Y { result: 1}
  else { result: 0 }
}
