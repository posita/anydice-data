/ Adding the same numbers together in a pool of die, i.e. rolling a 4, 5, 3 & 3 means you can have the following results of 3, 4, 5 or 6 [3+3] /
/ The only thing you need to change are the varibles DIE and NUMBEROFDICE below /

DIE: d10
NUMBEROFDICE: 5

function: showhighestsumofequals POOL:s {
  HIGHESTNUMBER: 1@POOL
  TEMP: 0
  loop N over {1..[maximum of DIE]} {
    TEMP: [count N in POOL] * N
    if TEMP > HIGHESTNUMBER {
      HIGHESTNUMBER: TEMP
    }
  }
  result: HIGHESTNUMBER
}

output [showhighestsumofequals NUMBEROFDICEdDIE] named "Probability for [NUMBEROFDICE]d10 where equals are added"
