\**** Buknaar Attack & Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 1
}

\If the enemy your smiting (if your not smiting, put 0d8) is an undead or fiend creature type, put 1d8. If not, put 0d8.\
UNDEADORFIENDDICE: 0d8

ENEMYAC: 19

ATTACKMOD: 4
DMGBONUS: 0
PROFICIENCY: 4
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 2
DAMAGEDICE: 2d6
CRITDAMAGEDICE: 2d6 + 2d6
SMITEDICE: 4d8 + UNDEADORFIENDDICE
CRITSMITEDICE: 4d8 + 4d8 + UNDEADORFIENDDICE + UNDEADORFIENDDICE

DAMAGE: SMITEDICE + DAMAGEDICE + ATTACKMOD + DMGBONUS
CRITDAMAGE: CRITSMITEDICE + CRITDAMAGEDICE + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "Buknaar attacks."
