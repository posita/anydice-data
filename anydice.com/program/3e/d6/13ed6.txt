\ The God Bound Damage function \
\ This allows you to roll dice and calculate out the damage that would be done in godbound if that damage wasn't rolled straight  \
\ Use:  1d[d2+3 gbdamage]
the 1 is the number of dice you are rolling
the 2 is the size of the dice
the 3 is the bonus you have on the damage roll from strength or some such
Note: that I haven't found anywhere in godbound where there are both multiple dice being rolled, and a damage bonus on the roll
Note: you can use this like any other dice roll, such as 3d6 
\
function: X:n gbdamage{
if X > 9 {result: 4}
if X = 9 {result: 2}
if X = 8 {result: 2}
if X = 7 {result: 2}
if X = 6 {result: 2}
if X = 5 {result: 1}
if X = 4 {result: 1}
if X = 3 {result: 1}
if X = 2 {result: 1}
if X < 2 {result: 0}
}
\ ----------------------------------------------------------------------

 This is the On Hit function \
\ this allows you to have different effects on a hit or a miss \
\ Use: 1d[X<2 onhit]
The 1 number of attacks you are making
The 2 is the number you need to roll on a d20 to hit\

\ Function setup:
function: X:n onhit {
if X = 1 {result: 3}
if X = 0 {result: 4}
}
 The 3 is the amount of damage done on a miss (usually 0) 
 The 4 is the amount of damage done on a hit 
 Note: you can use any type of roll to calculate damage, even using the GodBound Damage function 
 Note: you can put a letter after onhit (eg: tohita) to differentiate different versions of the function 

--------------------------------------------------------------------------------------------------\



function: X:n onhita {
if X = 1 {result: 4}
if X = 0 {result: 1d10+5}
}

function: X:n onhitb {
if X = 1 {result: 0}
if X = 0 {result: 1d[d10+5 gbdamage]}
}
