\ function to find if a roll beats a specified DC with 1 auto-fail 20 auto-passing \
function: skill N:n DC:n
{
  if N=1 { result: 0 }
  if N=20 { result: 1 }
  result: N>=DC
}

\ this is the targets bonus to save \
SAVE: 7
\ this is for any debuffs affecting the target \
PEN: 0
\ this is the DC of the save \
DC: 16

\ if not affected by anything unusual \
output [skill d20+SAVE-PEN DC] named "save VS something"

\ if affected by missfortune they'll be rolling twice \
output [skill [lowest of d20 and d20]+SAVE-PEN DC] named "save VS something with Misfortune"
