function: attackroll highest NUMBER:n of ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 else if ROLL = 1 { result: 0 }
 else if ROLL + PROFICIENCY + ATTACKMOD >= AC { result: DAMAGE }
 result: 0
}

\change these values\

\Your primary attack modifier (eg, STR, DEX)\
ATTACKMOD: 3 

\Your proficiency\
PROFICIENCY: 3

\The number you land a crit on (default 20, change to 19 if you're nuts are huge)\
CRITROLL: 20

\How many main hand attacks do you have?\
ATTACKTIMES: 1

\if your weapon is a +1 or +2 weapon, put it here.\
WEAPONBONUS: 1

\damage die of your weapon\
WEAPONDAM: 1d6

\extras\
FLOWERPOWER: 1d4



loop ENEMYAC over {12..18}{
DAMAGE: WEAPONDAM + ATTACKMOD + WEAPONBONUS + FLOWERPOWER
CRITDAMAGE: WEAPONDAM + WEAPONDAM + ATTACKMOD + WEAPONBONUS + FLOWERPOWER + FLOWERPOWER
output ATTACKTIMESd[attackroll highest 1 of 1d20 vs ENEMYAC] named "Flower Short Sword vs AC [ENEMYAC]"
}
