function: mexp DIE:d {
 result: [mexp DIE DIE helper]
}

function: mexp DIE:d N:n helper{
 if N = 1 {result: DIE }
 if N = 10 { result: N + [mexp DIE DIE helper] }
 result: N
}

function: lfr A:n B:n C:n{
  result: [highest B of A d[explode d10]] +C
}

function: lfrreroll A:n B:n C:n{
  result: [highest B of A d[mexp d10]] +C
}

\to make a normal x k y +z (explosions, no rerolling of 1: lfr x y z (so 4k2+3 would be lfr 4 2 3)
 to make a x k y +z with rerolling 1 (emphasis): lrreroll x y z
 to make a roll without explosions or rerolls, simply use xdy+z \

set "maximum function depth" to 5
\If the calculations aborts due to taking to long, reduce depth\

output [lfr 5 3 2] named "5k3+2"
output 4d10+3 named "4k4+3 no explosions"
output [lfrreroll 6 3 0] named "6k3+0 reroll ones"
