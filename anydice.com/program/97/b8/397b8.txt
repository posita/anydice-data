set "explode depth" to 3
function: evaluate ROLL:s {
  if [count 6 in ROLL] >= 1 {
    result: ROLL + [explode d6] -1
  } else {
    result: ROLL
  }
}

\ In the example below the "10" is the number of hit points of the target and the 5 is the damage from Murdering Winter \
output ([evaluate 2d6]+10) - ([evaluate 2d6]+5)
