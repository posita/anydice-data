\**** RPG Test Roll ****\

function: attackroll ROLL:n vs DV:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= DV { result: DAMAGE }
 result: 0
}

\Rolled against DV of 6\
ENEMYDV: d{6}

ATTACKMOD: 3
DMGBONUS: 1d6
PROFICIENCY: 2
ACCBONUS: 0
CRITROLL: 7
ATTACKTIMES: 1
DAMAGE: 1d4 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 1d4 + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll d20 vs ENEMYDV] named "Attack"
