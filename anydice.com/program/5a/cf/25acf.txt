function: roll ROLL:n at plus ATTACKBONUS:n vs AC:n ac for CRITABLEDAM:d plus HITDAM:d {
if ROLL = 20 {
result: CRITABLEDAM + CRITABLEDAM + HITDAM
}
    if ROLL = 1 {
    result: 0
    }
        if ROLL >= AC - ATTACKBONUS {
        result: CRITABLEDAM + HITDAM
        }
          else{
          result: 0
          }
}

\"Enter your your attack information below. Advantage is written 1@2d20 and disadvantage is 2@d20. CRITDAMAGE is the damage that should double as a crit; your flat damage bonus and other non-critable damage sources is HITDAM. To calculate per round damage with multiple attacks, set your attack info to a capitalized variable and add the variables together. Multiplication will not reroll the attack dice, so you must add."\

W: [roll 1d20 at plus 9 vs 16 ac for 1d10+1d6 plus 5]
