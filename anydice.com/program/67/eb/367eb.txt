SOGLIA: 4
function: calcolo su A:s B:s C:s D:s E:s {
 SORTED: [sort {A, B, C, D, E}]
 R: 1@SORTED + 2@SORTED 
 loop N over {3..#SORTED}{
  if N@SORTED >= 4 { R : R + 3 }
 }
 result: R
}
