\ This program rolls 5 attack dice with rerolls and focus tokens

This program assumes that the focus token is used last. Thus, it uses a die that has a failure (0) and a success (1) result.
The dice is numbered with 0,0,1,1,1,1,1,1. Rerolls are represented by rolling 2 dice and using the highest result. This
is practically a different process, but leads to the same mathematical results. \

X: 0 

loop N over {1..5}{
 X: 1@2d{0,0,1,1,1,1,1,1} + X
}

output X
