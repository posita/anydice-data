\ A - Attribute \
A: 11
\ B - Skill \
B: 3
\ C - TAG/focus? (0=FALSE | 1=TRUE) \
C: 0

\ target number, sum of A & B \ 
X: A+B

\ run the calculation. One success if it rolls equal or less than target number
 a second success if it roll equal or less than Skill number\
if C = 1 {loop Y over {2..5} {
 output [count {1..X,1..B} in Yd20]  named "[Y]d"
}}
else {loop Y over {2..5} {
 output [count {1..X,1} in Yd20]  named "[Y]d"
}}
