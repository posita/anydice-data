function: explode A:n B:n {
if A = B { 
result: A + [explode dB B ]
}
result:A
}

set "maximum function depth" to 4

X: [explode 1d10 10]
Y: [explode 1d4 4]
output [explode d12 12] named "d12"
output [explode d14 14] named "d14"
output X + Y named "d10 + d4"
