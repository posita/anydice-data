\ HEY YOU! \
\ The value of this "M" variable below is equivalent to the modifier you would have on your roll in a D&D skill check. It's automatically converted into each of the systems below. It can be positive OR negative. Once you change it, hit "Calculate" at the bottom to update the results! \

\------------------------>>>\ M: 0 \<<<-----------------------\


\ THE SYSTEMS \
\ Below is a description of each system and the math associated with how it works. \


\ D&D d20 \
\ This system is simple, but has the "problem" that it has an even chance of you rolling any given number within the range (5%) \

A: 1d20+M
output A named "D&D (1d20+[M])"



\ Star Wars D6 \
\ This system is based on a conversion table from d20 scores/modifiers into d6. This system centers with the average amount of dice rolled centering at about 10, but scores higher than average are more rewarding than D&D, and scores lower than average are more punishing than D&D. \
\ It also has a greater change of rolling near the center of your range than the extremes, so you generally feel more balanced, rather than rolling a freak 3 equally as often as a 15. More fun for specializing \

C: (M+6)@{0,0,0,1,1,2,2,3,4,4,5,6,6,7,8,8} \ This is the modifier to die conversion table \
D: (M+6)@{0,1,2,0,1,0,2,1,0,2,1,0,2,1,0,2} \ This is the modifier to modifier conversion table \
output \ This part is the exploding/destiny dice --->\[explode [highest of [lowest of C and 1] and 0]d6]+\ The rest of the dice --->\([highest of C-1 and 0]d6)+D named "Star Wars D6 ([C]d+[D])"



\ D10 System \
\ This is a great mod for d20 games since you literally just swap the d20 for 2d10. Same everything, except standard deviation is lower and the min is slightly higher. \

B: 2d10+M
output B named "D10 System (2d10+[M])"



\ Custom D6 System \
\ This is based on a custom rough conversion from the modifier to a number of D6s. This maintains a similar mean to the d20 system but with a smaller deviation, and more dice to roll which is fun \

E: (M+10)/3
output [explode d6]+(E-1)d6 named "D6 System ([E]d)"



\ The D12 System \
\ This was an experiment. It's a d12 + or - a number of d4s equal to what would usually be your modifier. The default/average value is lower, and progress is better rewarded than the d20. Unfortunately, being bad at something is also VERY punishing. \

F: 1d12+Md4
output F named "D12 System (1d12+[M]d4)"



\ The D8 System \
\ The D10 system but with D8s. Idk why I put this here \

G: 2d8+M
output G named "D8 System (2d8+[M])"



\ The Centered D6 System \
\ This system is 10 + your modifier as your base score (e.g. 10 + 4 = 14), then you roll 2d6. One d6 is added to the score, and the other is subtracted. That's your result. (e.g. 14 + 1d6[4] - 1d6[2] == 16) \
\ The purpose of this system is to keep the mean exact while restricting the deviation. Matches closely with the mean of D&D but you can't really do significantly better or worse than your base score \

I: 6
H: 10+M+1dI-1dI
X: 10+M
output H named "Centered D[I] system ([X] +1d[I] -1d[I])"
