\**** BASIC DAMAGE CALCULATOR ****\

\** Attack Roll Function **\
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ABILITYMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}
\** Attack Roll Function sharpshooter**\
function: attackrolln ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGEN } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ABILITYMOD + ACCBONUSN >= AC { result: DAMAGEN }
 result: 0
}

\** Attack looped against enemy AC from 15-20 **\
ENEMYAC: d{20}

\** Edit Here **\
ABILITYMOD: 5
DMGBONUS: 2
DMGBONUSN: 12
PROFICIENCY: 4
ACCBONUS: 2
ACCBONUSN: -3

ATTACKTIMES: 3

DAMAGE: 1d8 + ABILITYMOD + DMGBONUS
CRITDAMAGE: 2d8 + ABILITYMOD + DMGBONUS

DAMAGEN: 1d8 + ABILITYMOD + DMGBONUSN
CRITDAMAGEN: 2d8 + ABILITYMOD + DMGBONUSN
