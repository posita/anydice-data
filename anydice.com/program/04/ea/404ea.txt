\Damage and attack function\
MISS: 0
HIT: 2d6+4
CRIT: 4d6+4
function: attack ROLL:n vs DEFENSE:n{
 if ROLL=1{result: MISS}
 if ROLL=20{
  if ROLL+ATTACK>=DEFENSE{result: CRIT}
  result:HIT
 }
 if ROLL+ATTACK>=DEFENSE{result: HIT}
 result: MISS
}

\Stats for attacking and outputs\
ATTACK: 4+2
AC: 12
DIE: d20
output [attack DIE vs AC] named "Single Attack"
output DIE+ATTACK named "ToHit"
output HIT named "Damage"
output CRIT named "Crit"

\Used to calculate multiattack\
TOTAL: 0
MULT: 4
loop N over {1..MULT}{
TOTAL: (TOTAL+[attack DIE vs AC])
}
output TOTAL named "[MULT] Attacks"
