function: SCORE {
  R: d6
  if R <= 3 { result: 0 }       \ 1-3 -> 0, stop
  if R <= 5 { result: 1 }       \ 4-5 -> 1, stop
  result: 1 + [SCORE]           \ 6 -> 1 and roll again (recursively)
}

\ Build a die that equals the result of one full SCORE process:
S: d@SCORE           \ create a distribution from the SCORE function

\ Sum D independent such dice:
TOTAL: DdS

output TOTAL named "Distribution of total score"
\ Probability of reaching target T or more:
function: GEQ:n on A:s {
  result: sum x in {0..} where x >= n: 1 * (A = x)
}
\ AnyDice also supports "output [probability TOTAL >= T]" directly:
output 1 - cumulative TOTAL T-1 named "P(total >= T)"
