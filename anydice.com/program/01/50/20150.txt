\**** Spirit Shroud vs Haste ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 14-18\
ENEMYAC: d{14..18}

ATTACKMOD: 5
DMGBONUS: 0
PROFICIENCY: 5
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 6
DAMAGE: 1d10 + 1d8 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d10 + 2d8 + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "Spirt Shroud with double EB"

CRITDAMAGE: 8d6 + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "Haste with GWM"
