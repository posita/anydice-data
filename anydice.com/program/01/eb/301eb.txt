\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= 20{ result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}
function: savingthrow ROLL:n vs DC:n DAMAGE:n{
 if ROLL < DC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{12..14}
SAVEMOD: d{-2..0}

PROFICIENCY: 3
ATTACKMOD: 3
DC: 15
DMGBONUS: 0
ACCBONUS: 0
DAMAGE: 1d8 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 3d8 + ATTACKMOD + DMGBONUS
output [attackroll d20 vs ENEMYAC] named "hammer"
output [savingthrow d20 + SAVEMOD vs DC 2d8] named "toll the dead (2d8)"
output [savingthrow d20 + SAVEMOD vs DC 2d12] named "toll the dead (2d12)"
