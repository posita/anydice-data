\
PF2e
 -1: Critical fail.
 0: Fail.
 1: Success.
 2: Critical success!

 To simulate fortune effects, use 1@2d20 instead of 1d20 for "keep higher," or 2@2d20 for "keep lower."
\

function: pathfinder ROLL:n mod MOD dc DC {
 SUCMOD: 0
 if ROLL = 20 {SUCMOD: 1}
 if ROLL = 1 {SUCMOD: -1}
 
 result: (ROLL + MOD >= DC) + SUCMOD
}

output [pathfinder 1@2d20 mod 8 dc 19]
