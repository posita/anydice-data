function: cargaorka A:n B:n under N:n {
   \Solo repite dado si es menor que N\
   YES: 0 
   if A + B < N {
      if A > 4 | B > 4 { YES: 1}
      if YES= 0 { result: 1d6 + 1d6 }
      else{
         if A < B { result: B + 1d6 }
         else { result: A + 1d6 }
      }
   }
   else{ result: A + B }
}




output [cargaorka 1d6 1d6 under 9] named "Carga Orka"
