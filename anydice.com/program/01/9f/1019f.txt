loop N over {1..15}
{
    X: (N+2)/3
    output 1@Xd6 + N + 3 named "[X]d6+[N] with proficiency"
}
\The goal here is to make a system where a small shift in attributes has a softer effect, while a broader difference has a stronger effect. This way, a character isn't immediately outclassed by the person immediately above them. By going with "keep highest", it trends toward the maximum as the character progresses, which fits for more "heroic" feeling gameplay.\
