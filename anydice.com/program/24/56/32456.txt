\*** Attack Action***\
function: attackrolla ROLL:n vs AC:n{
 if ROLL = 1 { result: 0 }
 if ROLL = 20 { result: CRITDMGA }
 if ROLL + ATKBONUS >= AC { result: DAMAGEA }
 result: 0
}

\*** Once Per Turn Damage ***\
function: onceperturn ROLL:n vs AC:n{
 if ROLL = 1 { result: 0 }
 if ROLL + ATKBONUS >= AC { result: DAMAGEO }
 result: 0
}

\** Great Weapon Fighting Function **\
function: gwfdice DMG:n size SIZE:n {
 if DMG <= 2 { result: 1dSIZE }
 result: DMG
}

\** Variables **\
ATKBONUS: 6
AC: d{11,13,15..18}
DAMAGEA: 1d[gwfdice 1d10 size 10] + 4
CRITDMGA: 2d[gwfdice 1d10 size 10] + 4
ATTACKS: 1
DAMAGEB: 1d[gwfdice 1d4 size 4] + 4
CRITDMGB: 2d[gwfdice 1d4 size 4] + 4
DAMAGEO: 1d6
OCHANCE: 2

\* Output *\
output ATTACKSd[attackrolla d20 vs AC] named "Attack Action"
output 1d[attackrollb d20 vs AC] named "Bonus Action"
output 1d[onceperturn [highest 1 of OCHANCEd20] vs AC] named "Once Per Turn Damage"
