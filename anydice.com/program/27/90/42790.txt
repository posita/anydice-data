\---T&T DARO Function---\
function: daro ROLL:s {
  if 1@ROLL = 2@ROLL {
    result: ROLL + 1d6 + [daro 1d6]
  }
  result: ROLL
}

\---Example Saving Roll---\
\Level 1 Save = 20. If stat is 10, target is 10+\
\Using 2d6 as base\
SAVE_ROLL: [daro 2d6]
output SAVE_ROLL+2
output SAVE_ROLL+2 >= 20 \Example 20+ save\
