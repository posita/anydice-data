\CONTESTED GRAPPLE CHECK\
\In the output section, a 1 is considered a successful grapple and a 0 is considered a failure.\
\Your Athletics bonus:\
X: 15
\Do you have advantage? 1 for YES, 0 for NO.\ \Do you have disadvantage? 1 for YES, 0 for NO.\
 Y: 1                                          Z: 0
\Their Athletics/Acrobatics bonus:\
P: 5
\Do they have advantage? 1 for YES, 0 for NO.\ \Do they have disadvantage? 1 for YES, 0 for NO.\
 Q: 0                                           R: 0

ADVANTAGE: [highest 1 of 2d20]
DISADVANTAGE: [lowest 1 of 2d20]

if (Y = Z) {
  YOUR_ROLL: d20
} else if (Z > Y) {
  YOUR_ROLL: DISADVANTAGE
} else {
  YOUR_ROLL: ADVANTAGE
}

if (Q = R) {
  THEIR_ROLL: d20
} else if (R > Q) {
  THEIR_ROLL: DISADVANTAGE
} else {
  THEIR_ROLL: ADVANTAGE
}

YOURS: YOUR_ROLL + X
THEIRS: THEIR_ROLL + P

output YOURS > THEIRS named "successful grapple"
