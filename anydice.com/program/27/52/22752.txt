CRITS: 10 \number of 40%/2.5x crit damage tiles\
SHOTS: 1 \set to 6 if you're testing out the combiner skills, otherwise leave at 1\

\don't touch anything below this comment please <3\

function: critnum SHOTS:n CRITS:n {
  if (SHOTS > 1) {
    result: [critdam (1@SHOTSd(CRITSd(1d5>3)))]
  }
  else {
    result: [critdam (CRITSd(1d5>3))]
  }
}

function: critdam SUCCESS:n {
  POWER: 1000
  if (SUCCESS > 0) {
    loop X over {1..SUCCESS} {
      POWER: POWER * 5
    }
    loop X over {1..SUCCESS} {
      POWER: POWER / 2
    }
  }
  result: POWER
}

output [critnum SHOTS CRITS] named "multiplier from 1 projectile among [SHOTS] rolling n number of crits from [CRITS] tiles (divide by 1000)"
