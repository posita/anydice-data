\Generic reroll if X function.\
function: reroll REROLL:s on ROLL:n to REPLACEMENT:d
{
 if [REROLL contains ROLL]
 {
  result: REPLACEMENT
 }
 result: ROLL
}

\Specific reroll function to format GWM\					 
function: C:n gwm D:n
{
 T: 0
 loop N over {1..C}
 {
  T: T + [reroll 1 on 1dD to 1dD]
 }
 result: T
}

\Calculate the outcome of an attack roll.\
function: R:n attack A:n with B:n bonus and P:n and D:d dicepool crit on C:n
{
 if      R     >= C {result: 2dD + B} \Crit\
 else if R+B+P >= A {result: 1dD + B} \Hit\
 else                {result: 0  } \Miss\
}

\Calculate the outcome of an attack roll.\
function: R:n spell A:n with B:n bonus and P:n and D:d dicepool
{
 if      R     >= 20 {result: 2dD} \Crit\
 else if R+B+P >=  A {result: 1dD} \Hit\
 else                {result: 0  } \Miss\
}
