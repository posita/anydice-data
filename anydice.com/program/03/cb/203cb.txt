\ Attack roll calculator \
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}
\looped against enemy AC from 8-25\
ENEMYAC: d{8..25}

\***modifications to attack roll and damage***\
ATTACKMOD: 8  \* Attribute modifier *\
WEAPONMOD: 0  \* Weapon modifier (e.g. +1 sword)*\
DMGBONUS: 0  \* Other modifiers (e.g. Agonizing blast, sharpshooter, etc.) *\
PROFICIENCY: 3  \* Total Character level:bonus >> 1:2 , 5:3 , 9:4 , 13:5 , 17:6 *\
ACCBONUS: 0 \* Other modifiers (e.g. -5 for sharpshooter/GWM *\
CRITROLL: 20
ATTACKTIMES: 1
DAMAGE: 1d4 + 1d8 + 1d6 + 4d6 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d4 + 2d8 + 2d6 + 8d6 + ATTACKMOD + DMGBONUS
