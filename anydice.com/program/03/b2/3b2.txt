\
 Each d6 either explodes or implodes.
 
 Explosion: rolling a 6 ads another d6.
 6s after that are added again.
 Once a 6 is rolled, 1s aren't replaced on subsequent rolls.
 
 Implosion: rolling a 1 replaces it with -1d6.
 6s that are adding another -d6.
 Once a 1 is rolled, 1s aren't replaced on subsequent rolls.
\

function: evaluate N:n{
 if N = 6 { result: 6+1d[explode d6] }
 if N = 1 { result: -[implode d6] }
 result: N
}

function: explode N:n{
 if N = 6 { result: 6+1d[explode d6] }
 result: N
}

function: implode N:n{
 if N = 6 { result: 6+1d[implode d6] }
 result: N
}

set "maximum function depth" to 3
\
 A maximum function depth of 4 means that explosions/implosions
 cascade at most 3 levels deep. Rolls after that are dropped.
\

EVALUATED: d[evaluate d6]

output EVALUATED named "one die"
output 3d EVALUATED named "three dice"
