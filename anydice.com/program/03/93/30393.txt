function: reroll R:n under N:n {
   if R < N { result: 1d10 } else {result: R}
}
function: boost R:n under N:n {
   if R < N { result: R+1 } else {result: R}
}

L: 11
X: 0
Y: 0
loop N over {1..L}{
 X: X + [boost 1d10 under 6]
 Y: Y + [reroll 1d10 under 2]
}
output Y+10+36 named "Roll (Reroll 1st 1)"
output X+10+36 named "Roll (+1 if below avg)"
output (L*6)+10+36 named "Take Average"
