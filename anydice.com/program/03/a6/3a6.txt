\***Probability for one attack roll***\
ATTACK:5 \*enter the # of dice in attack pool*\
ARMOR:2  \*enter # of dice in defense pool*\

function: evaluate attack ROLL:s SOAK:s{
 SUCCESSES: ROLL >= 5
 DEFENSE: SOAK >= 5
 if SUCCESSES > DEFENSE { result: SUCCESSES - DEFENSE }
 result: 0
}

loop DICE over {ATTACK} {
 output [evaluate attack DICE d6 ARMOR d6] named "[DICE]d"
}
