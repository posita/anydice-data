\***Probability for a range of attack rolls against a specified armor rating***\
MAXATTACKPOOL:5  \*enter the maximum dice pool size you want to test*\
ARMOR:2         \*enter # of dice in defense pool*\

function: evaluate attack ROLL:s SOAK:s{
 SUCCESSES: ROLL >= 5
 DEFENSE: SOAK >= 5
 if SUCCESSES > DEFENSE { result: SUCCESSES - DEFENSE }
 result: 0
}

loop DICE over {1..MAXATTACKPOOL} {
 output [evaluate attack DICE d6 ARMOR d6] named "[DICE]d"
}
