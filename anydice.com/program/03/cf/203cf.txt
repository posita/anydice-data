\ Attack roll calculator \
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + WEAPONMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}
\looped against enemy AC from 8-25\
ENEMYAC: d{8..25}

\***modifications to attack roll and damage***\

\**Option 1: Haste and EB, no hex, no Agonizing blast *\
\* Longsword attacks w/o advantage or hex*\
ATTACKMOD: 5  \* Attribute modifier *\
WEAPONMOD: 3  \* Weapon modifier (e.g. +1 sword)*\
DMGBONUS: 0  \* Other modifiers (e.g. Agonizing blast, sharpshooter, etc.) *\
PROFICIENCY: 5  \* Total Character level:bonus >> 1:2 , 5:3 , 9:4 , 13:5 , 17:6 *\
ACCBONUS: 0 \* Other accuracy modifiers (e.g. -5 for sharpshooter/GWM *\
CRITROLL: 20 \* Lowest number to deliver a crit *\
ATTACKTIMES: 3 
DAMAGE: 1d8 + ATTACKMOD + WEAPONMOD + DMGBONUS 
CRITDAMAGE: 8 + 1d8 + ATTACKMOD + WEAPONMOD + DMGBONUS + 40 + 5d8 \* max Smite*\
output ATTACKTIMESd[attackroll [highest 1 of 1d20] vs ENEMYAC] named "Longsword w/o Adv, w/o hex"
