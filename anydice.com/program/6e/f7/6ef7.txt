\ The die type to be rolling \
DIE: 6

\ Run our charts for skills up to this level \
MAXSKILL: 6

\ Normal Power Level is rolling 2 dice \
POWER: 2

\ Every X full points is 1 Hit \
INTERVAL: 3

output 0 named "***** d[DIE] + SKILL | Every [INTERVAL] points is 1 Hit *****"
loop N over {0..MAXSKILL}{
 output (dDIE+N)/INTERVAL named "SKILL [N]"
}

output 0 named "***** [POWER]d[DIE] + SKILL | Every [INTERVAL] points is 1 Hit *****"
loop N over {0..MAXSKILL}{
 output (POWERdDIE+N)/INTERVAL named "SKILL [N]"
}

output 0 named "***** [POWER]d[DIE] + (SKILL Per DIE) | Every [INTERVAL] points is 1 Hit *****"
loop N over {0..MAXSKILL}{
 output (POWERdDIE+N*POWER)/INTERVAL named "SKILL [N]"
}
