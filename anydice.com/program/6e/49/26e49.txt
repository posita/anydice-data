\**** Damage Calculator for level 17 fighter for question 115624 ****\
LUCKYCOUNTER: 3

\LUCKYD is pass in through every function call because I couldn't call it dynamically\
\It's only used if we have lucky rolls left\
function: attackroll ROLL:n vs AC:n LUCKYCOUNTER:n LUCKYD:n{
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
\recursively call attackroll using the lucky die\
 if LUCKYCOUNTER > 0 & LUCKYD > 0 {
  LUCKYCOUNTER: LUCKYCOUNTER - 1
  result: [attackroll LUCKYD vs AC 0 0]
 }
 result: 0
}

\AC variable, can be a die\
ENEMYAC: 19
\looped against enemy AC from 15-20\
\ENEMYAC: d{15..20}\

ATTACKMOD: 5 \Strength Bonus\
DMGBONUS: 10 \Great Weapon Master\
PROFICIENCY: 6
ACCBONUS: -5 \Great Weapon Master\
CRITROLL: 18 \Crit Range\
ATTACKTIMES:7 \Assuming a crit happens\
GWFDIE: d{{1,2},{3,4,5,6}:4} \Great Weapon Fighting Style adjustment\
DAMAGE: 4dGWFDIE + ATTACKMOD + DMGBONUS
CRITDAMAGE: 8dGWFDIE + ATTACKMOD + DMGBONUS
