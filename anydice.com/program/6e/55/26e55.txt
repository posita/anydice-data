\**** Damage Calculator for level 17 fighter for question 115624 ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\AC variable, can be a die\
ENEMYAC: 19
\looped against enemy AC from 15-20\
\ENEMYAC: d{15..20}\

ATTACKMOD: 5 \Strength Bonus\
DMGBONUS: 10 \Great Weapon Master\
PROFICIENCY: 6
ACCBONUS: -5 \Great Weapon Master\
CRITROLL: 18 \Crit Range\
ATTACKTIMES:7 \Assuming a crit happens\
UNLUCKYTIMES: ATTACKTIMES-3
GWFDIE: d{{1,2},{3,4,5,6}:4} \Great Weapon Fighting Style adjustment\
DAMAGE: 2dGWFDIE + 2dGWFDIE + ATTACKMOD + DMGBONUS
CRITDAMAGE: 4dGWFDIE + 4dGWFDIE + ATTACKMOD + DMGBONUS
\treating lucky as 3d20 for simplicity\
output UNLUCKYTIMESd[attackroll [highest 1 of 2d20] vs ENEMYAC] + 3d[attackroll [highest 1 of 3d20] vs ENEMYAC] named "GWM Champion Bugbear surprise"
