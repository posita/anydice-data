\Probability of multiple 12s in Traveller characteristic arrays under various rolling methods\

function: number of twelves in ROLLA:s and ROLLB:s if total is at least NUM:n {
 if (ROLLA+ROLLB) > (NUM-1) { result: [count 12 in ROLLA]+[count 12 in ROLLB] }
}

NROLL: 2d6
HROLL: [highest 2 of 3d6]
NCHAR: d(NROLL)
HCHAR: d(HROLL)
NSTATS: 3dNCHAR
HSTATS: 3dHCHAR

output [count 12 in 6dNCHAR] named "Chance of rolling the specified number of 12s on 6 x 2d6"
output [count 12 in NSTATS]+[count 12 in HSTATS] named "Chance of rolling the specified number of 12s on 3 x 2d6 and 3 x 3d6 drop lowest"
output [number of twelves in NSTATS and HSTATS if total is at least 42] named "Chance of rolling the specified number of 12s on 3 x 2d6 and 3 x 3d6 drop lowest, with re-roll if total below 42"
output [count 12 in 6dHCHAR] named "Chance of rolling the specified number of 12s on 6 x 3d6 drop lowest"
