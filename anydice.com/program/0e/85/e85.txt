\
 Valhalla Project Roll

 Roll Xd10, X being the ability, take the highest die.
 If it is 1-9, it is your result + your skill Y, otherwise, if it is 10,
 your result is 10 + Y per die that shows 10.
\

function: valhallarunes ROLL:s{
 if 1@ROLL = 10 {
  result: 10 + [count 10 in ROLL]*SKILL
 }
 result: 1@ROLL
}

loop ABILITY over {1,3,5,7,10}{
loop SKILL over {1,3,5,7,10} {
 output [valhallarunes ABILITYd10] named "[ABILITY]d10 with [SKILL] skill ranks"
 }
}
