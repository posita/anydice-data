

function: yonderdice  WURF:s { 
 ERFOLGE: WURF >=5 \Alles ab 5 ist ein Erfolg\
 BONUSWURF: WURF = 6 \Jede 6 ergibt einen Bonuswurf\
 if BONUSWURF > 0 { result: ERFOLGE+BONUSWURF*[yonderdice d6] } \Wurf 6 wird nochmal gewürfelt, alle Erfolge zusammengezählt \
  result: WURF >=5 \Ergebnis ist die Anzahl der Würfe, die über 5 liegen\
}

set "maximum function depth" to 2 \Jede 6 darf noch einmal geworfen werden; es ist also für jede 6 noch ein weiterer Erfolg möglich\

YONDERDICE: [yonderdice d6] \So heißt dieser spezielle Würfel\

loop N over {1..20} {
 output [lowest of 20 and NdYONDERDICE] named "[N]d"
} \das "lowest of 20"
