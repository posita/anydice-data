\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 13-15\
ENEMYAC: d{13..15}

ATTACKMOD: 3
DMGBONUS: 2
PROFICIENCY: 3
ACCBONUS: 0
CRITROLL: 20
DAMAGE: {1}@2d10 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 10 + 1d10 + ATTACKMOD + DMGBONUS
output [attackroll d20 vs ENEMYAC] + [attackroll d20 vs ENEMYAC] named "Piercer"
