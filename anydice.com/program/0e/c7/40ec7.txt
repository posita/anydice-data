\**** Damage calculator ****\

function: attack ROLL:n vs AC:n with DICE:d {
 if ROLL = 21 { result: 1@[reverse DICE] + DICE + ATTACKMOD + DMGBONUS + CRITBONUS} 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS > AC { result: DICE + ATTACKMOD + DMGBONUS }
 result: 0
}

\looped against enemy AC from 13-15\
ENEMYAC: d{13..15}

ATTACKMOD: 3
PROFICIENCY: 3
ACCBONUS: 0
DMGBONUS: 2
CRITBONUS: 0
WEAPON: {1}@2d10
output [attack 1d20 vs ENEMYAC with WEAPON] + [attack {1}@2d20 vs ENEMYAC with WEAPON] named "Piercer"
