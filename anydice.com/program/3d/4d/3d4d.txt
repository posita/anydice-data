drop high/low are for star crossed, divine inspiration, etc. 

fate blessed cut off points are when the roll is less than the expected average roll.
when rolling both good omens & fate blessed, an order has to be chosen. a cut off point
for when to trigger the other rule is also needed. the rule is that if my original roll
is less than the expected value of my replacement roll, i trigger the other rule.

for example, if i roll a good omens 2d6 < 7, i trigger fate blessed since the expected 
value on 2d6 is 7, so i am improving my odds. if i roll 2d6 < 8, i trigger fate blessed
since the expected value on a good omens 2d6 is 8, again improving my odds.

there are, obviously, lots of cut off points for where to reroll. each affects both the
average & the standard deviation. \

function: DIE:d drop high H:n low L:n { result: {(1+H)..(#DIE-L)}@DIE }
function: DIE:d drop high H:n { result: [DIE drop high H low 0] }
function: DIE:d drop low L:n { result: [DIE drop high 0 low L] }

function: reroll DIE:d if less than T:n { result: [reroll DIE helper] }
function: reroll N:n helper { if N < T { result: DIE } else { result: N } }

function: reroll DIE:d with NEW:d if less than T:n { result: [reroll DIE with NEW helper] }
function: reroll N:n with NEW:n helper { if N < T { result: NEW } else { result: N } }
