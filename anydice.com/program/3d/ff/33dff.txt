function: animaprime roll COUNT:n dados dificuldade DIFICULDADE:n {
  AGILIZAR: {1:(DIFICULDADE-1), 6:(6-DIFICULDADE), 6}
  result: [animaprime roll COUNT d AGILIZAR dificuldade DIFICULDADE]
}

function: animaprime roll ROLAGEM:s dificuldade DIFICULDADE:n {

  TIPODEDADO: 6
  SUCESSOS: {DIFICULDADE..TIPODEDADO}

  SUCESSOSTOTAIS: [count SUCESSOS in ROLAGEM]

  result: SUCESSOSTOTAIS
}

\ *EXTRAS*                                                                                                     \
\ Caso o seu 'ataque' tenha o valor de rolagem de +X (exemplo: 2d+1). saiba que o +1 é um SUCESSO AUTOMÁTICO.  \
\ Ou seja, ele auxilia na chance de gerar um crítico, porém não auxilia gera Threats!                          \
\ Além disso, existe um limites de Threats causadas por um ataque:                                             \
\ O limite é o nº de dados padrões da rolagem +1. Por exemplo: numa rolagem de 2d+1 o limite de Threats é 3!   \
\ Caso o 'inimigo' tenha Resistência contra o tipo de dano de seu ataque (exemplo: physical (2)), o nível da   \
\ sua resistência será reduzido em sucessos do seu ataque.                                                     \
\ Por fim, caso sua rolagem de ataque (Ability Dice + Action Dice + Threat Dice) ultrapasse a defesa do alvo, \
\ você causa um CRÍTICO. Todas as threats geradas são removidas do alvo e ele recebe 1 Wound (ou seja -1 HP)   \

\ *Resultados* \
\ Você pode trocar o número de dados e a dificuldade abaixo para mudar os resultados. \
output [animaprime roll 8 dados dificuldade 4]
output [animaprime roll 6 dados dificuldade 3]
