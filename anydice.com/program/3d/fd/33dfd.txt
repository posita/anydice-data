function: attacks X:n {
    TOTAL: 0
    loop N over {1..X}{
    TOTAL: TOTAL + [hitstep d6]
    }
    if FNP {result: TOTALd(d6 < FNP)}
    else {result: TOTAL}
}

function: hitstep X:n {
    if TORRNT {result: [woundstep d6]}
    if X = 1 {
     if RRHITO|RRHITS {result:[hitsteprr d6]}
     else {result:0}
}
    if (X >= CRTHIT){
     if SUSHIT&LETHIT {result:[sustainhit]+[savestep d6]}
     if SUSHIT {result:[sustainhit]+[woundstep d6]}
     if LETHIT {result:[savestep d6]}
     else {result:[woundstep d6]}
}
    if X+HITMOD >= BS {result:[woundstep d6]}
    else {
     if RRHITS {result:[hitsteprr d6]}
     else {result:0}
}
}

function: hitsteprr X:n {
    if X = 1 {result:0}
    if (X >= CRTHIT){
     if SUSHIT&LETHIT {result:[sustainhit]+[savestep d6]}
     if SUSHIT {result:[sustainhit]+[woundstep d6]}
     if LETHIT {result:[savestep d6]}
     else {result:[woundstep d6]}
}
    if X+HITMOD >= BS {result:[woundstep d6]}
    else {result:0}
}

function: sustainhit {
    TOTAL: 0
    loop N over {1..SUSHIT}{
    TOTAL: TOTAL + [woundstep d6]
}
    result: TOTAL
}   

function: woundtar {
    if S > T * 2 {result: 2}
    if S > T {result: 3}
    if S = T {result: 4}
    if S < T / 2 {result: 6}
    if S < T {result: 5}
}

function: woundstep X:n {
    if X = 0 {
     if RRWNDO|RRWNDS {result: [woundsteprr d6]}
     else {result:0}
}
    if X >= CRTWND {
     if DEVWND {result: D}
     else {result: [savestep d6]}
}
    if X+WNDMOD >= [woundtar] {result: [savestep d6]}
    else {
     if RRWNDS {result: [woundsteprr d6]}
     else {result:0}
}
}
function: woundsteprr X:n {
    if X = 0 {result:0}
    if X >= CRTWND {
     if DEVWND {result: D}
     else {result: [savestep d6]}
}
    if X+WNDMOD >= [woundtar] {result: [savestep d6]}
    else {result:0}
}


function: savestep X:n {
    if ((SV-AP)<ISV)|ISV=0 {
     if X >= (SV-AP) {result: 0}
     else {result: D}
}
    if (SV-AP)>=ISV {
     if X >= ISV {result: 0}
     else {result: D}
}
}

\This algorithm simulates one of a unit's weapons attacking as a group against a target and provides probabilities for all resulting wounds.\
\Edit the variables below to match your weapon's and the target's profile.\
\The algorithim will probably work even if you go outside expected ranges, or put dice values where they shouldn't go.\

\NUM: Number of models equipped with the weapon in unit.\
\ATK: Attacks characteristic of weapon. Can be a number, dice, or both.\
\BS: Ballistic or Weapon Skill characteristic of weapon. Should be between 2-5\
\S: Strength characteristic of weapon.\
\AP: Armor Penetration characteristic of weapon. Should be negative.\
\D: Damage characteristic of weapon. Can be a number, dice, or both.\

\TORRNT: Toggles Torrent for the weapon. 1 on, 0 off.\
\HITMOD: Modifies the Hit/BS/WS roll. Total all external rules and abilities that modify it here. Should between 1 and -1.\
\CRTHIT: Defines what number on the D6 or above counts as a Critical Hit. Should be 6, unless you have an external rule that changes that.\
\RRHITO: Toggles rerolling hit rolls of one. 1 on, 0 off.\
\RRHITS: Toggles rerolling hit rolls. 1 on, 0 off.\
\WNDMOD: Modifies the wound roll. Total all external rules and abilities that modify it here. Should between 1 and -1.\
\CRTWND: Defines what number on the D6 or above counts as a Critical Wound. Should be 6, unless you have an external rule that changes that.\
\RRWNDO: Toggles rerolling wound rolls of one. 1 on, 0 off.\
\RRWNDS: Toggles rerolling wound rolls. 1 on, 0 off.\
\LETHIT: Toggles Lethal Hits for the weapon. 1 on, 0 off.\
\SUSHIT: Enables Sustained Hits for the weapon. 0 is off, 1 gets Sustained Hits 1, 2 gets Sustained Hits 2, and so on.\
\DEVWND: Toggles Devastating Wounds for the weapon. 1 on, 0 off.\

\T: The target's Toughness characteristic.\
\SV: The target's Save characteristic.\
\ISV: The target's Invulnerable Save ability. 0 if they don't have one.\
\FNP: The target's Feel No Pain ability. 0 if they don't have one.\
