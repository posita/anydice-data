function: oppose A and D using S{
  result: (D+1)@(A+D+1)dS
}
\Variables Below:\
S: 6 \Number of sides\
LA: 1 \Lowest attacker value to test\
HA: 8 \Highest attacker value to test\
LD: 1 \Lowest defender value to test\
HD: 8 \Highest defender value to test\

loop N over {LA..HA}{
  loop M over{LD..HD}{
  A: N+M+1
  output [oppose N and M using S]>(S/2) named "[N] opposes [M] using [A]d[S]"
 }
}
