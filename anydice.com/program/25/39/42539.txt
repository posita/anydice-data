\this function rolls 4d6 and drops the lowest, repeating all 1s\
\well, kind off, 1d6 repeating all 1s is the same as 1d5 +1\
function: roll {
result: 3 +[highest 3 of 4d5]
}

\converts ability score to ability score bonus\
function: convert A {
result: ([highest of A  and 8] /2 - 5)
}

\makes and converts one roll, then outputs the result\
output [convert[roll]]

\rolls and converts six times, outputs the sum\
R: 0
loop N over {1..6} {
R: R + [convert[roll]]
}
output R
