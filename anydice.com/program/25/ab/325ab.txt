set "position order" to "lowest first"

function: roll A:s and B:s pick low two {
 if #(B) = 1 {
  if 1@B > 2@A {
   result: {1,2}@A
  }
  else {
   result: 1@A + 1@B
  }
 }
 else {
  if 1@B > 2@A {
   result: {1,2}@A 
  }
  else {
   if 1@A < 2@B {
    result: 1@A + 1@B
   }
   else {
    result: {1,2}@B
   }
  }
 }
}

output [roll 2d10 and 2d10 pick low two] / # podaj dwa zestawy kości, wynik określa sumę dwóch najgorszych rzutów /
