\ Returns the number of drinks you must take. You are A, opponent is B. Minimum Drink of 2 \
function: battle A:n and B:n {
 if A - B = 0 { result: 2}
 if A - B = -1 {result: 2}
 if A - B > 0 { result: 0}
 result: [absolute A - B]
}

\ Subtracts ten from the total, if total is greater than 10 \
function: slaps A:n {
 if A > 10 { result: A - 10}
 result: A
}

\ Returns the number of drinks you must take, assuming you trade slaps for ten drinks \
output [slaps [battle 1d20 and 1d20]]
