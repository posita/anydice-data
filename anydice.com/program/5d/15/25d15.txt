function: attack ROLL:n MOD:n AC:n DAM:d CRITRNG:n CRITDAM:d {
  if ROLL >= CRITRNG { result: DAM + CRITDAM }
  if ROLL + MOD >= AC { result: DAM }
  result: 0
}

BASEDAM: 1d8+3
CRITDAM: 1d8
HITMOD: 5
AC: 10
output [attack 1d20 HITMOD AC BASEDAM 20 CRITDAM] named "Normal attack"
output [attack 1@2d20 HITMOD AC BASEDAM 20 CRITDAM] named "Advantage attack (vow of enmity)"
output [attack 1d20 HITMOD AC BASEDAM+2 19 CRITDAM] named "Hexblade's curse attack"
