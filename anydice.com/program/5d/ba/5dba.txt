\ to get a reroll effect, create a custom function that only includes results for the values you want to keep \

function: ignore below N:n for ROLL:n {
 if N < ROLL { result: 1d6 + [ignore other 1 for 1d6]}
 else {result: ROLL + [ignore other 1 for 1d6]}
}

function: ignore other N:n for ROLL:n {
 if N < ROLL { result: 1d6 }
 else { result: ROLL }
}

output [ignore below 1 for 1d6] \adds another d6\
