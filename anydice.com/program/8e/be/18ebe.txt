MODIFIER: 3 \Modifier on roll. Ranges from -5 to +5. Based on the amount of resolve \
BONUS: 4 \Skill + Attribute, ranges from +1 to +6.\
DC: 9 \DC to beat\

function: caltrops bonus of B:n modifier of M:n {
  if M > 0 { result: [highest 2 of (2+M)d4]+B }
  if M < 0 { result: [lowest 2 of (2-M)d4]+B }
  result: 2d4+B
}

function: skillcheck dc of D:n vs R:n {
  if R > D+3 { result: 5 }
  if R > D { result: 4 }
  if R = D { result: 3 }
  if R > D-3 { result: 2 }
  result: 1
}

output [caltrops bonus of BONUS modifier of MODIFIER] named "Roll Results"
output [skillcheck dc of DC vs [caltrops bonus of BONUS modifier of MODIFIER]] named "Pass%"
