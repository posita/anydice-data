function: animaprime A:s {
  X: 0
  loop DIFICULDADE over A {
\ *ALTERAR DIFICULDADE*                                                                                                            \
\ O número abaixo representa a dificuldade da Rolagem. Troque esse número para a dificuldade que quiser testar sua rolagem contra. \
\ Normalmente as rolagens de ataque do sistema são feitas na dificuldade de 3.                                                     \
\ Rolagens de "Traits and Abilities" são feitas nas seguinte dificuldades: Regular: 3, Hard: 4, Very Hard: 5 e Near Impossible: 6. \
    if DIFICULDADE >= 3 {
      X: X + 1
    }
  }
  result: X
}

\ *RESULTADO*                                                                                                  \
\ Você pode alterar a quantidade de dados abaixo para representar sua pool de dados durante a rolagem.         \
\ Muitos dados podem fazer com que a função falhe em carregar. O limite testado foram 17d6.                    \
\ O resultado dirá quantos SUCESSOS você terá na rolagem que você digitou.                                     \

\ *EXTRAS*                                                                                                     \
\ Caso o seu 'ataque' tenha o valor de rolagem de +X (exemplo: 2d+1). saiba que o +1 é um SUCESSO AUTOMÁTICO.  \
\ Ou seja, ele auxilia na chance de gerar um crítico, porém não auxilia gera Threats!                          \

\ Além disso, existe um limites de Threats causadas por um ataque!                                             \
\ O limite é o nº de dados padrões da rolagem +1. Por exemplo: numa rolagem de 2d+1 o limite de Threats é 3!   \

\ Caso o 'inimigo' tenha Resistência contra o tipo de dano de seu ataque (exemplo: physical (2)), o nível da   \
\ sua resistência será reduzido em sucessos do seu ataque.                                                     \

\ Por fim, caso sua rolagem de ataque (Abilitie Dice + Action Dice + Threat Dice) ultrapasse a defesa do alvo. \
\ Você causa um CRÍTICO. Todas as threats geradas são removidas do alvo e ele recebe 1 Wound (ou seja -1 HP)   \

output [animaprime 5d6]
