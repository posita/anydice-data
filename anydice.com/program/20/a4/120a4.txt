\ Hyper Breeding Dice: See https://www.f-list.net/c/hyper%20breeding%20dice for explanation! \
VIRILITY: 6 \ minimum is 2 \
FERTILITY: 5 \ minimum is 2 \
RESISTANCE: 2 \ minimum is 0, max is (VIRILITY * FERTILITY - 1) \
ROUNDS: 5 \ minimum is 1, 3 is recommended \

function: round {
 result: [highest of 0 and (1dVIRILITY * 1dFERTILITY) - RESISTANCE]
}

output ROUNDSd[round]
