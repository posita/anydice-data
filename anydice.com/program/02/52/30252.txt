\
H&G  1d12 vs. 10-stat, roll EQUAL TO OR HIGHER.
Critical: if natural 12.
Stats range = 0-7 (avg=4)

 -1: Miss.
 +1: Clean success.
 +2: Critical success!
\

function: roll A:s vs S:s {
 R: -1
 if A > S {
   R: 1
 }
 if A = S {
   R: 2
 }
 result: R
}

loop P over {0..7} {
 output [roll 1d12 vs 10-P] named "[P]"
}
