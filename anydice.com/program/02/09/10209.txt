
X: 3d10
L:[lowest 1 of X]
M:[middle 1 of X]
H:[highest 1 of X]
SECONDLOWEST: 3@X - 2@X

function: wtf X:d L:n M:n H:n{
if L <= 5 & M <= 5 { result: H+2d10 }
if L <= 5 & M > 5 { result: H+M+1d10 }
if L > 5 & M > 5 { result: H+M+L }
if L > 5 & M <= 5 { result: L+H+1d10 }
}

output L named "lowest"
output SECONDLOWEST named "second lowest"
output M named "medium"
