\ Code for comparing pool rolls. \

\
 Wrapper function for counting successes.
 "4v3" means 4d6 with target number 3.
\
function: N v T {
 result: Nd(d6 >= T)
}

\
 A function to check which side has more successes.
 Ties are resolved by a coin flip.
 1 means A wins, 2 means B wins.
\
function: A:n vs B:n {
 if A > B { result: 1 }
 if B > A { result: 2 }
 result: d2
}

\ Examples of single pool rolls. \
output [6v4] named "6v4"
output [4v2] named "4v2"

\ Examples of pool vs pool rolls. \
output [[6v4] vs [3v3]] named "6v4 vs 3v3"
output [[5v5] vs [4v2]] named "5v5 vs 4v2"
output [[4v3] vs [4v4]] named "4v3 vs 4v4"
