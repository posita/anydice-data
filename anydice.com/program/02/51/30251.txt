\
H&G  2d12 vs. 10-stat, roll EQUAL OR HIGHER.
Critical: if 2 successes equal to each other.
Stats range = 0-7 (avg=4)

 -1: Miss.
 0: Success, but with undesired consequences.
 +1: Clean success.
 +2: Critical success!
\

function: roll A:s and B:s vs S:s {
 R: -1
 if A >= S {
   R: R+1
 }
 if B >= S {
   R: R+1
 }
 if A=B {
     R: 2
 }
 result: R
}

loop P over {0..7} {
 output [roll 1d12 and 1d12 vs 10-P] named "[P]"
}
