\Function code\
function: reroll N:n below M:n
{if N < M {result: [highest of d6 and N]} result: N }
function: reroll N:n above M:n
{if N > M {result: [lowest of d6 and N]} result: N }


\Outputs\
output 1d6 named "No rerolls"
output 1d [reroll d6 below 2] named "Rerolling 1s"
output 1d [reroll d6 above 5] named "Rerolling 6s"
