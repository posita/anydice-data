function: bug VALUES:s in SEQUENCE:s with ROLL:d mod MOD:n {
    COUNT: 0
    SUM: 0
    loop P over {1..#VALUES}
    {
        SUM: SUM + P@VALUES*(P@VALUES = SEQUENCE)
    }
    result: ROLL < (SUM-MOD)
}

BUGGED: {4}
BUGDICE: d16
EXPVAL: 1
