\
 This is an oscillating "implode", because you keep subtracting negative numbers.
 So an explode chain could look like 1 - (1 - (1 - (1 - 6))) = 6.
\

function: implode N:n {
 if N = 1 { result: 1 - [implode d6] }
 result: N
}

loop D over {0..3} {
 set "maximum function depth" to D + 1
 output [implode d6] named "imploded at most [D] times"
}
