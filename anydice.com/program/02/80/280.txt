\
 A full straight means all rolled dice form a single straight sequence.
 Mathematically, rolling a full straight has the following odds:
 DICE! * 1/6 ^ DICE, to get a specific one,
 (7 - DICE) * DICE! * 1/6 ^ DICE to get any.

 The AnyDice function below verifies this.
 It checks whether a roll sequence is consecutive.
 A result of 1 means it is, a result of 0 means it isn't.
 
 The transposed graph is a nice way to view the results.
\

function: ROLL:s is a full straight {
 N: 1@ROLL
 loop P over {2..#ROLL} {
  if P@ROLL = N - 1 { N: N - 1 }
  else { result: 0 }
 }
 result: 1
}

loop DICE over {1..6} {
 output [DICEd6 is a full straight] named "[DICE]d6 full straight"
}
