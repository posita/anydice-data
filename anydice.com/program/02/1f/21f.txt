\
 This is an aggravating "implode", because you'll always subtract.
 So an explode chain could look like 1 - 1 - 1 - 1 - 6 = -8.
\

function: implode N:n first roll FIRSTROLL:n {
 if N = 1 {
  if FIRSTROLL { result: 1 - [implode d6 first roll 0] }
  result: 1 + [implode d6 first roll 0]
 }
 result: N
}

loop D over {0..3} {
 set "maximum function depth" to D + 1
 output [implode d6 first roll 1] named "imploded at most [D] times"
}
