\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
\ Function: simulate             \
\ Simulates damage based on the  \
\ attack roll, the roll of on    \
\ the die required to hit, and   \
\ damage parameters.             \
\ X = attack roll die            \
\ Y = required roll to hit       \
\ A = baseline weapon damage     \
\ B = bonus damage on every hit  \
\ C = bonus damage on a crit     \
\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
function: X:n Y A B C simulate {
if X > 18 {result: A + A + B + C}
if X < Y {result: 0}
result: A + B
}
output [d20 8 1d10 9 30 simulate] named "Hunter Boon"

function: X:n Y A B C simulate {

if X < Y {result: 0}
result: A + B
}
output [d20 8 1d10 9 0 simulate] named "No Hunter Boon"
