function: attackroll ROLL:n vs AC:n {
if ROLL >= CRITROLL { result: CRITDAMAGE }
if ROLL = 1 { result: 0 }
if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
result: 0
}

ADV: 1@2d20
DIS: 2@2d20

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

ATTACKMOD: 6
DMGBONUS: 12
PROFICIENCY: 5
ACCBONUS: -1
CRITROLL: 20
ATTACKTIMES: 2
DAMAGE: 1d8 + 1d6 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d8 + 2d6 + ATTACKMOD + DMGBONUS


output ATTACKTIMESd[attackroll 1@2d20 vs ENEMYAC] named "Longbow +2 SS"
DMGBONUS: 2
ACCBONUS: 4
output ATTACKTIMESd[attackroll 1@2d20 vs ENEMYAC] named "Longbow +2"
