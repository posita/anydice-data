NOTHIT: 6
NOTPEN: 7
NOTKILL: 8
\Exit values for all the failures\

function: rend N:n \Rending weapon quality, self-explanatory...\{
 if N = 6 { result: 6 + d3 }
 result:N
}

function: hit DICE:n BS:n TL:n {
 BSRR: 0
 if BS > 5 {
  BSRR: BS-5
  BS: 5
 }
 \If the Ballistic Skill is very high, you are granted a fixed re-roll. You need to roll 7-BS to hit (so with BS4 you need 3+), and if your BS is 7, for example, you hit on 2+ and then on 5+ if failed.\
 \TL for twin-linked, it allows to re-roll the hit dice\
 if TL = 1 {
  BSRR: BS
 }
 if DICE >= 7-BS {
  result: 1
 }
 else {
  if BSRR > 0 {
   result: [hit d6 BSRR TL]
  }
 result: 0
 }
}

function: wrappen PEN:n { \Function to wrap up Penetration. If rolled under 10, you auto-failed to damage any vehicle, if over 15 — it's enough to penetrate any vehicle.\
 if PEN < 10 {
  result: NOTPEN
 }
 if PEN > 15 {
  result: 15
 }
 result: PEN
}

function: penetrate HIT:n S:n N:n REND:n { \S for Strength, N for the number of dice. Penetration roll is S+d6, if you roll equal to or higher than Armor Value of the vehicle (AV), you get a damage result. Certain weapons may roll S+2d6 or even S+4d6, as they are very powerful. Rending weapons get an additional D3 if they rolled a 6 on penetration roll, so S+6+D3. I haven't met any S+2d6 Rending weapons, though...\
 if HIT = 1 {
  if REND = 1 {
   result: [wrappen S+[rend d6]]
  }
  result: [wrappen S+Nd6]
 }
 else { result: NOTHIT }
}

function: destroy DIE:n PEN:n AV:n AP:n OT:n FO:n { \Various modifiers (0 or 1) included. To destroy a vehicle, you need to roll 5+ with d6 on the vehicle damage chart. There are 4 modifiers here, 3 as parameters and the other is for the glancing hit, when your penetration roll is exactly equal to the Armor Value in question.\
 if PEN = NOTHIT {
  result: NOTHIT
 }
 if PEN < AV {
  result: NOTPEN
 }
 if PEN = AV {
  GLANCE: -2
 }
 if PEN > AV {
  GLANCE: 0
 }
 if DIE+GLANCE+AP+OT+FO >= 5 {
  result: AV
 }
 else { result: NOTKILL }
}
