SPELLBONUS: 8
SAMPLEAC: 15

function: attack D:n plus BONUS:n vs AC:n  {
    if D = 20 { result: 2 } 
    else if D = 1 { result: 0}
    else if D + BONUS >= AC+10 { result: 2 }
    else if D + BONUS >= AC { result: 1 } 
    else { result:0}
}

output 2d4 * [attack 1d20 plus SPELLBONUS vs SAMPLEAC] named "Electric Arc"
output 3d4+3 named "Force Barrage"
