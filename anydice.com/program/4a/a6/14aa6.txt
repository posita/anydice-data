\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}


function: pamattackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: PAMCRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: PAMDAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

ATTACKMOD: 4
DMGBONUS: 2
PROFICIENCY: 3
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 2
DAMAGE: 1d6 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d6 + ATTACKMOD + DMGBONUS
PAMDAMAGE: 1d4 + ATTACKMOD + DMGBONUS
PAMCRITDAMAGE: 2d4 + ATTACKMOD + DMGBONUS

output ATTACKTIMESd[attackroll d20 vs ENEMYAC] + ATTACKTIMESd[pamattackroll d20 vs ENEMYAC] named "Rokara 2 Swings of Spear + PAM Current Strength Duelist"
