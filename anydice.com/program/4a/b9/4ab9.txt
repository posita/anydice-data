function: odd DIE:d {
 MAX: [maximum of DIE]/#(DIE) \[maximum of ..] calculates the max of all die, so we divide by the number we rolled to get the max. of a single one\
 result: [odd helper DIE MAX] \we need to pass the DIE to a helper function, otherwise each instance of DIE will be independent and the function won't work\
}

function: odd helper SEQ:s N:n { \convert the DIE we passed to a sequence. Now we'll use the same roll throughout the function\
 X: [count N in SEQ] \count the number of times we rolled the max. value\
 result: {1..X+1}@[sort SEQ] \sum all the max. dice, plus one for the second highest value\
}
