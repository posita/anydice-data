\ Wild Imaginary West Dice Pool \

BLACK:  d{0,0,15,1,1,2}
GOLD:  d{0,15,1,1,1,2}

function: attack REMOVE:n SPUR:n AMOUNT:n DICE:d ROLL:n{
REMOVAL:15
if SPUR=1 & REMOVE=1 {REMOVAL:14}
loop X over {1..AMOUNT}{
if ROLL >= 15 {ROLL: ROLL - REMOVAL}
}
result: ROLL
}

\ attack REMOVE SPUR AMOUNT DICE ROLL \
output [attack 1 1 1 1dBLACK 1dBLACK] named "1B"
output [attack 1 1 1 1dGOLD 1dGOLD] named "1G"
output [attack 1 1 2 1dBLACK 2dBLACK] named "2B"
output [attack 1 1 2 1dGOLD 2dGOLD] named "2G"
output [attack 1 1 3 1dBLACK 3dBLACK] named "3B"
output [attack 1 1 3 1dGOLD 3dGOLD] named "3G"
output [attack 1 1 4 1dBLACK 4dBLACK] named "4B"
output [attack 1 1 4 1dGOLD 4dGOLD] named "4G"

\ REMOVE: This should always be set to 1. \
\ SPUR: If the model or unit has surge, set this to 1, otherwise set it to 0. \
\ AMOUNT: Amount of dice rolled (number of models or dice on weapon). \
\ DICE: Type of dice being rolled. \
\ ROLL: The total amount of dice rolled. \
