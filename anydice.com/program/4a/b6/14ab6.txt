\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 14-18. Common enemy AC, find the average against this range of enemies.\
ENEMYAC: d{16}

ATTACKMOD: 4
DMGBONUS: 3
PROFICIENCY: 3
ACCBONUS: 0
CRITROLL: 19
ATTACKTIMES: 4
DAMAGE: 1d10 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d10 + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll [highest 1 of 3d20] vs ENEMYAC] named "Hexblade's Curse Eldritch Blast + Quickened Blast"
