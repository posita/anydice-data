\ spend blood tokens to interpolate between 2@3d6 (normal) and 1@3d6 (advantage)
  roll <= blood on control die to pick best of the other three dice \

function: switch S:n {
 if S < 1 { result: 1@3d6 }
 result: 2@3d6
}

output 2@3d6 named "0 blood (2@3d6)"
output [switch d6 - 1] named "1 blood"
output [switch d6 - 2] named "2 blood"
output [switch d6 - 3] named "3 blood"
output [switch d6 - 4] named "4 blood"
output [switch d6 - 5] named "5 blood"
output 1@3d6 named "6 blood (1@3d6)"
