\ Result 0: Critical miss; Result 1: Miss; Result 2: Hit; Result 3: Critical hit \

\ To convert to Emberbrand, average ability scores according to attribute, rounded up, then use the modifier from the result as MOD \
\ Body: STR, DEX, CON \
\ Mind: INT, WIS, CHA \
\ Spirit: STR, CON, WIS \

\ Inputs \
DC: 10
MOD: 1

\ Functions \
function: X:n conditional {

if X - MOD = 1 {result: 0}

if X - MOD = 20 {result: 3}

if X >= DC {result: 2}

else { result: 1 }
}


output [d20 + MOD conditional]
