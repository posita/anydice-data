\Planquest dice calculator. 2d50 version. By Derpmind.\ 
\Best used in Transposed view. Rolling too many dice will cause an error, so adjust DICEMIN and DICEMAX to keep your dice reasonable!\

DICEMIM: 1 \Starting number of dice rolled\
DICEMAX: 2 \Ending number of dice rolled\
DIEBONUS: 0 \Bonus per die\
CURRENTPROGRESS: 0 \Your current project progress\
PROGRESSTARGET: 40 \The total progress the project needs\
OMAKEBONUS: 15 \Omake bonus points\
NOBONUSDICE: 0 \Number of extra dice rolled without a bonus\

function: evaluate ROLL:s {
 SUCCESSES: ROLL >= 0
 if SUCCESSES > 0 { result: 1 }
 result: 0
}

loop DICE over {DICEMIM..DICEMAX} {
 output [evaluate (2*DICE)d50+(2*NOBONUSDICE)d50+DIEBONUS*DICE+CURRENTPROGRESS-PROGRESSTARGET+OMAKEBONUS] named "[DICE]"
}
