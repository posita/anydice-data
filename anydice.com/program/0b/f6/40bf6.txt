A: 1d20
B: 11+2
C: 15
D: 4d8-1d8+10
E: -1d8
F: -2d8

function: roll A:n add B:n check for C:n return D:n on success E:n on failure and F:n on crit failure {
if A+B <= C-10 {result: F}
if A = 1 {result: F}
if A+B < C & A+B > C-10 {result: E}
if A+B >= C {result: D}
}
output [roll A add B check for C return D on success E on failure and F on crit failure]
