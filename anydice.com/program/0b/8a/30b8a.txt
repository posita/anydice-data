\ interpolation between 2@3d6 and 1@3d6 
  by swapping out white dice for red dice @ 1/blood token spent
  result is the highest of the middle value and the highest red die \

function: blood B:n from X:n and Y:n and Z:n {
if B < 1 { result: 2@[sort {X, Y, Z}] }
else {
  if B = 1 { result: [highest of 2@[sort {X, Y, Z}] and X] }
  else {
    if B = 2 { result: [highest of 2@[sort {X, Y, Z}] and 1@[sort {X, Y}]] }
    else {
      if B >= 3 { result: 1@[sort {X, Y, Z}] }
      }
    }
  }
}

output [blood 0 from d6 and d6 and d6] named "no blood"
output [blood 1 from d6 and d6 and d6] named "1 blood"
output [blood 2 from d6 and d6 and d6] named "2 blood"
output [blood 3 from d6 and d6 and d6] named "3 blood"
