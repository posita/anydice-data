function: range on DIE:n between LOW:n and HIGH:n {

if DIE >= LOW & DIE <= HIGH { result: 1 }

result: 0

}

DIE : 2d6
RAA : 7
RAB : 9
RBA : 10
RBB : 11
RCA : 12
RCB : 25

loop S over {-3..5} {
output [range on DIE + S between RAA and RAB] named "[DIE] + [S], target between [RAA] and [RAB]"
output [range on DIE+ S between RBA and RBB] named "[DIE] + [S], target between [RBA] and [RBB]"
output [range on DIE+ S between RAA and RAB] named "[DIE] + [S], target between [RBA] and [RBB]"
}
