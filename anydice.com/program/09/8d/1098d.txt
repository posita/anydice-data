FORT:{-1,0,0,0,0,1}

\*** The "1" result is the probability of a having a fortune (misfortune is the same probability)   The "0" result is misleading because it is only the probability of not having fortune rather than the probability of neither extreme ***\
output 1d(1d6 = 1)>0 named "(Mis-)fortune 1d6"
output 2d(1d6 = 1)>0 named "(Mis-)fortune 2d6"
output 3d(1d6 = 1)>0 named "(Mis-)fortune 3d6"
output 4d(1d6 = 1)>0 named "(Mis-)fortune 4d6"

\*** The "1" result is the probability of a having a fortune (misfortune is the same probability) if any number of fortunes and misfortunes cancel each other out.   The "0" result is misleading because it is only the probability of not having fortune that is not cancelled ***\
output (1dFORT=-1)*(1dFORT<1) named "(Mis-)fortune 2d6 cancelling but not stacking"
output (1dFORT=-1)*(1dFORT<1)*(1dFORT<1) named "(Mis-)fortune 3d6 cancelling but not stacking"
output (1dFORT=-1)*(1dFORT<1)*(1dFORT<1)*(1dFORT<1) named "(Mis-)fortune 4d6 cancelling but not stacking"

\*** For cancelling/stacking fortune, show all of the results.  Negative numbers are misfortune(s), positive are fortune(s).  ***\
output 2dFORT named "(Mis-)fortune 2d6 stacking and cancelling" 
output 3dFORT named "(Mis-)fortune 3d6 stacking and cancelling"
output 4dFORT named "(Mis-)fortune 4d6 stacking and cancelling"

\ *** Probability of success *** \
output 1d(1d6 >4)>0 named "Success 1d6"
output 2d(1d6 >4)>0 named "Success 2d6"
output 3d(1d6 >4)>0 named "Success 3d6"
output 4d(1d6 >4)>0 named "Success 4d6"
