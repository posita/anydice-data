\DnD Character Creation Roller - ShadowFox Release\
\Function below can be used to reroll any chosen number or collection of numbers out of a pool of dice rolled for character creation. Die size, pool size, and reroll number may be modified in order to achieve desired roll statistics probability density function.\

function: charroll reroll N:n on DIE:n {
 if DIE = N { result: [charroll reroll N on DIE] }
 result: DIE
}

\Reroll Number - always make this a die so that multiple numbers may be rerolled if desired.\
MAGICNUMBER: (d1*2)
\Die size\
SIZE: 4
\Pool size\
POOL: 7
\Selection number must not allow for a condition that would exceed 20, therefore it will be calculated here\
SELECT: POOL
loop N over {1..POOL} {
 if (SELECT * SIZE) > 20 { SELECT: SELECT - 1 }
 SELECT: SELECT
}

output [lowest SELECT of POOLd[charroll reroll MAGICNUMBER on dSIZE]] named "Lowest [SELECT] out of [POOL]d[SIZE], re-rolling certain values (in code)"
