\Maze Rats average damage calculator\

function: roll A:n B C {
  if A <= 6 - B {result : 0}
  result: A - 6 + B + C
}

\
A: The roll
B: Attack Bonus - armor
C: Weapon Damage
\
X: 2d6

output [roll X (-2) 0] named "-2"
output [roll X (-1) 0] named "-1"
output [roll X 0 0] named "0"
output [roll X 1 0] named "1"
output [roll X 2 0] named "2"
output [roll X 3 0] named "3"
output [roll X 4 0] named "4"
