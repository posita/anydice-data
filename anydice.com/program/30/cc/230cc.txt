\Maze Rats average damage calculator\

function: roll A:n B C {
  if A <= 6 - B {result : 0}
  result: A - 6 + B + C
}

\
A: The roll
B: Attack Bonus - armor
C: Weapon Damage

output [roll 2d6 (-2) 0] named "-2"
output [roll 2d6 (-1) 0] named "-1"
output [roll 2d6 0 0] named "0"
output [roll 2d6 1 0] named "1"
output [roll 2d6 2 0] named "2"
output [roll 2d6 3 0] named "3"
output [roll 2d6 4 0] named "4"
