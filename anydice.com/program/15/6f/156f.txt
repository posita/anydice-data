FATE: 4d{-1..1}
function: ROLL:s fudge with N:n virtual pluses {
  COUNT: [count -1 in ROLL]
  BONUS: [lowest of COUNT and N]
  result: ROLL+BONUS
}
function: ROLL:s fudge with N:n virtual minuses {
  COUNT: [count 1 in ROLL]
  BONUS: [lowest of COUNT and N]
  result: ROLL-BONUS
}
PLUS:2
MINUS:2
output FATE named "4DF"
output FATE + 2d{0,0,1} named "4DF plus 2 stunt dice"
output [FATE fudge with PLUS virtual pluses] named "4DF with [PLUS] virtual pluses"
output [FATE fudge with MINUS virtual minuses] named "4DF with [MINUS] virtual minuses"
output [FATE fudge with PLUS virtual pluses]+2d{0,0,1} named "4DF with [PLUS] virtual pluses and 2 stunt dice"
