MISS: 0
HIT: 1
CRITICALHIT: 2

function: attack ROLL:n ATTACK:n vs DEFENSE:n {
 if ROLL = 1 { result: MISS }
 if ROLL = 20 {
  if ROLL + ATTACK >= DEFENSE { result: CRITICALHIT }
  result: HIT
 }
 if ROLL + ATTACK >= DEFENSE { result: HIT }
 result: MISS
}

AT: 2
PRO: 2
BONUS: AT + PRO
AC: 15
output [attack d20 BONUS vs AC]
output [attack [highest 1 of 2d20] BONUS vs AC]
output [attack [lowest 1 of 2d20] BONUS vs AC]
