function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\


ATTACKMOD: 0
DMGBONUS: 0
loop PROFICIENCY over {1..12} {
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 2
DAMAGE: 1d8 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d8 + ATTACKMOD + DMGBONUS
X: ATTACKTIMESd[attackroll d20 vs 20]
Y: ATTACKTIMESd[attackroll d20 vs 21]
T: PROFICIENCY + ATTACKMOD
output X-Y named "Average Extra Damage (20 vs 21 AC), Enemy Total Skill Bonus [T]"
}
