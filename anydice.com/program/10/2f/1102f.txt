function: ROLL:s matching dice {
     Z: 0
     loop N over {1..#ROLL-1} {
          X: N@ROLL
          Y: X-(N+1)@ROLL = 0
          Z: Z+Y
     }
     result: Z > 0
     \ "result: Z" should count the probability of each possible number of matching dice, rather than just returning success/fail\
}
output [2d4 matching dice] named "2d4; 0=success 1=fail"
