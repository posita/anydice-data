\HMG damage simulator\

ARMOR: 1 \assumed average armor\
TN: 9 \E-Def or Evasion\
ACC: 0 \Accuracy applied\
DIFF: 1 \Difficulty applied - either accuracy or difficulty should be 0, at least 1 Diff from Inaccurate\

ROLL: d20 + [highest 1 of ACCd6] - [highest 1 of DIFFd6]
function: hmg R:n TN:n {
  if R >= TN {
    if R >= 20 {
      result: [highest 2 of 4d6]+4-ARMOR
    } else {
      result: 2d6+4-ARMOR
    }
  } else {
    result: 0
  }
}

output [hmg ROLL TN]
