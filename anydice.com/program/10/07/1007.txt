\
 3 is removed from play
 2 is flee
 1 is initiative loss
 0 is nothing
\

\ conversion of result range 0-6 to results listed above \
CONVERSION: {0, 0, 1, 1, 2, 3}

function: ATTACK:s vs DEFENSE:s {
 A: 1@ATTACK
 AP: 1
 loop P over {1..#DEFENSE} {
  if P@DEFENSE >= A {
   \ eliminate die \
   AP: AP + 1
   A: AP@ATTACK
  }
  else { P: 100 } \ break out of loop \
 }
 \ convert remaining die, remove @CONVERSION to show original die value \
 result: A@CONVERSION
}

output [5d6 vs 3d6] named "5 vs 3"
