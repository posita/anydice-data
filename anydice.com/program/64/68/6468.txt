function: roll D:s T M {
  \ fumble \
  if D = 1 {
    result: 0
  }

  \ success \
  if D > 1 & D <= T {
    result: 2
  }

  \ failure \
  if D > T & D < 20 {
    result: 1
  }

  \ crit \
  if D = 20 {
    result: 3
  }
}

function: compare HERO with RESISTANCE {
  result: HERO > RESISTANCE 
}

output [compare [roll d20 17 0] with [roll d20 14 0]]
