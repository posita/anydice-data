\ Fail on a 1 or below, on a dice with 6 sides \
FAIL: 1 
DICE: 9

\ this function loops X amount of times, calculating the chance we lose point(s)\
function: i have X:n wounds {
LOSS: 0

\Here we make a sequence equal to the amount of wounds we have, the value of all the numbers in the same as FAIL. N then loops through all the (identical) values.\
loop N over {FAIL:X} {
\Calculate the odds this dice loses us a point, and add the odds of the previous dices\
LOSS: LOSS + (NdDICE <= FAIL)
}

result: LOSS
}
\The average is WOUNDS/(DICE^FAIL), if fail is 1 you can get the avg with WOUNDS / DICE\
output[i have 1 wounds] named "1 wound"
output[i have 6 wounds] named "6 wounds"
output[i have 9 wounds] named "9 wounds"
output[i have 12 wounds] named "12 wounds"
