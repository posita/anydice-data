MISS: 0
HIT: 1
CRITICALHIT: 2
PROFBONUS: 2
ABMOD: 3
ATTACK: ABMOD + PROFBONUS
DICENUM: 1
DICESIZE: 4
HITDAMAGE: DICENUMdDICESIZE + ABMOD
MISSDAMAGE: 0
CRITDAMAGE: (2*DICENUM)dDICESIZE + ABMOD
function: attack ROLL:n vs DEFENSE:n { 
if ROLL = 1{result: MISS}
if ROLL = 20{
if ROLL + ATTACK >= DEFENSE{result: CRITICALHIT}
result: HIT
}
if ROLL + ATTACK >= DEFENSE{result: HIT}
result: MISS
}
function: damage for ATTACKRESULT:n{
if ATTACKRESULT = HIT{
result: HITDAMAGE
}
if ATTACKRESULT = CRITICALHIT{
result: CRITDAMAGE
}
result: MISSDAMAGE
}
output [damage for [attack d20 vs 8]]
