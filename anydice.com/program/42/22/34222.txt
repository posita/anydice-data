\ Hyper Breeding Dice: See https://www.f-list.net/c/hyper%20breeding%20dice for explanation! \
VIRILITY: 3 \ the Giver's stat, minimum is 1 \
FERTILITY: 2 \ the Receiver's stat, minimum is 1 \
RESISTANCE: 6 \ minimum is 0, max is (VIRILITY * FERTILITY - 1) \
ROUNDS: 3 \ minimum is 1, 3 is recommended \

function: round {
 result: [highest of 5 and (1dVIRILITY * 1dFERTILITY) - RESISTANCE]
}

output ROUNDSd[round]
