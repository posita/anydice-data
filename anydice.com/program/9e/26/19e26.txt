COUNT: 5 \This value represents how many dice are being rolled. Change the value to calculate with more dice\
DICE: 10 \This value represents the dice type, i.e. the number of sides. 6 for d6's, 8 for d8's, etc. Change for different dice types.\

MOD: COUNT - ((COUNT/2)*2) \this is just a quick fix for the fact that this program always rounds down\

output COUNTdDICE named "Normal Damage"
output COUNTd(DICE*2) named "Critical Damage by Doubling The Roll"
output COUNTdDICE*2 named "Critical Damage by Rolling Double Dice"
output COUNTdDICE+(((COUNT + MOD)/2)*DICE)named "New Critical Damage"
