\ 'Castroll' returns casting value if cast is a success with the casting roll vs. cast difficulty (D) \
\ 'Dispell' is just 2d6+dispell modifier \
\ 'Resolve' returns 1 if i. castroll is higher than the casting difficulty & ii. dispell roll is not higher than the casting value \
\ Does not take miscasts into account \



function: castroll A:n
{
 if A >= D { result: A }
 result: 0
}
D:7 

function: dispell W:n{
result: 2d6+4
}


function: resolve B:n C:n {
if C <= B {
result: 1
}
result: 0}

output [castroll 2d6+4] named "cast value"


output [resolve [castroll 2d6+4] [dispell 4]] named "cast success"
