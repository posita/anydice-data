\Custom Die Function\
\Die roll with custom numbers\
\Example: output [newdie]\
function: newdie { 
result: d{0, 1, 30, 55, 40, 0, 10000}
}


\Reroll Function\
\Force a reroll when a roll results in value n\
\Example: output [reroll 1 for d6]\
function: reroll N:n for ROLL:n {
if N != ROLL {result:ROLL}
}


\Ignore Below Function\
\Ignores results below value n for a roll\
\Example: output [ignore below 10 for 4d6]\
function: ignore below N:n for ROLL:n {
if N <= ROLL {result: ROLL}
}
