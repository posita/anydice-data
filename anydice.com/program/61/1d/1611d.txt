AC: 15
function: attack ROLL:n HITBONUS:n DAMAGE:d CRITTHRESHOLD:n CRITDAMAGE:d {
  if ROLL >= CRITTHRESHOLD {
    result: DAMAGE + CRITDAMAGE
  } else if ROLL = 0 {
    result: 0
  } else if ROLL + HITBONUS >= AC {
    result: DAMAGE
  }
}

SMITE: 3d8
