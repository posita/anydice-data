function: stunt DICE:s STUNT:n DC:n {
  if 1@DICE + 2@DICE + STUNT < DC {
    result: -1
  }
  ECID: [reverse DICE]
  loop X over {1..#ECID-1}{
    if (X@DICE = (X+1)@DICE | X@DICE = STUNT | (X+1)@DICE = STUNT) & X@DICE + (X+1)@DICE + STUNT >= DC { result: STUNT }
  }
  result: 0
}

DC: 11
output [stunt 2d6 1d6 DC]
output [stunt 3d6 1d6 DC]
output [stunt 4d6 1d6 DC]
output [stunt 5d6 1d6 DC]
output [stunt 6d6 1d6 DC]
