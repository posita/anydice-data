
F:{-1,0,1}
BONUS:{0,0,1}
X:3


\ 1. roll (4+X)dF, keep highest 4, count N 1's rolled, add NdBONUS \
function: roll DICE:s {
 N:[count 1 in DICE]
 result: [highest 4 of DICE] + NdBONUS
}
output [roll (4+X)dF] named "[4+X]dF no limits"

\ 2. roll (4+X)dF, keep highest 4, count N 1's rolled, add (min N,X)dBONUS \
function: roll DICE:s and limit bonus rolls {
 N:[count 1 in DICE]
 result: [highest 4 of DICE] + [lowest of N and X]dBONUS
}
output [roll (4+X)dF and limit bonus rolls] named "[4+X]dF with max [X] bonus rolls"

\ 3. roll (4)dF, and roll XdF, keep highest 4. Count N 1's rolled in XdF, add NdBONUS \
function: roll DICE:s and limit bonus rolls to xpool {
 N:[count 1 in DICE] - [count 1 in {1@DICE,2@DICE,3@DICE,4@DICE}]
 result: [highest 4 of DICE] + [lowest of N and X]dBONUS
}
output [roll (4+X)dF and limit bonus rolls to xpool] named "[4+X]dF and limit bonus rolls to xpool"

\ 4. roll (4)dF, and roll XdF, keep highest 4. Count N 1's rolled in XdF, add (N*X)dBONUS \
function: roll DICE:s and limit bonus rolls to xpool exploding {
 N:[count 1 in DICE] - [count 1 in {1@DICE,2@DICE,3@DICE,4@DICE}]
 result: [highest 4 of DICE] + [lowest of N*X and X]dBONUS
}
output [roll (4+X)dF and limit bonus rolls to xpool exploding] named "[4+X]dF and limit bonus rolls to xpool. Exploding with max [X] bonus rolls"
