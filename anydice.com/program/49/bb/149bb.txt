function: middle of ROLL:s if lowest in RANGE:s {
  if !(3@ROLL = RANGE) { result: -1 }
  result: 2@ROLL
}

CHARACTERISTIC: 11
FOO: [middle of 3d20 if lowest in {1 .. CHARACTERISTIC-1}]
BAR: [middle of 3d20 if lowest in {CHARACTERISTIC .. 20}]

function: restrict ROLL:n to RANGE:s else REROLL:d {
  if ROLL = RANGE { result: ROLL }
  else { result: REROLL }
}
function: restrict ROLL:d to RANGE:s {
  TEMP: ROLL
  loop I over {1..20} {
    TEMP: [restrict TEMP to RANGE else TEMP]
  }
  result: TEMP
}

output [restrict FOO to {CHARACTERISTIC..20}] named "middle if lowest < [CHARACTERISTIC] and middle >= [CHARACTERISTIC]"
output [restrict BAR to {CHARACTERISTIC..20}] named "middle if lowest >= [CHARACTERISTIC]"
