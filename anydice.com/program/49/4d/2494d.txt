function: VALUE:s min MIN {
if VALUE < MIN {result: MIN}
else {result: VALUE}
}

function: dw roll ROLL:s vs SKILL:n for DDICE:n dten plus ADDITION:n max MAX:n each TYPE:n {
SUCCESSES: [((SKILL-ROLL)/10)/TYPE min 0]
if SUCCESSES>MAX {ATTACKS:MAX} else {ATTACKS:SUCCESSES}

result: ( (ATTACKS*DDICE)d10 + ATTACKS*ADDITION )

}

output 1d([dw roll 1d100 vs 53 for 1 dten plus 19 max 10 each 2]) named "now: single relic blade (pen 9)"
