function: test D:n plus MOD:n vs DC:n {
  RES: 0
  if D+MOD >= DC {
    RES: RES+1
  }
  if D = 20 {
    RES: RES+1
  }
  if D = 1 {
    RES: RES-1
  }
  result: RES
}

function: test D:d N:n times plus MOD:n vs DC:n {
  RESS: 0
  loop I over {1..N} {
    RESS: RESS + [test D plus MOD vs DC]
  }
  result: RESS
}


MOD: 7
DC: 18
N: 10
output [test 1d20 N times plus MOD vs DC] named "1d20"
