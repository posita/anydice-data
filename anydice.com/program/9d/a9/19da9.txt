ST:3
HST:ST*2

function: mightattack A:n with S:n bonus B:n type X:n armor Y:n{
H:A + S
D:0-Y
if A = 20 {
 D:D+1}

if H >14 {
 D:D+1+B}
if H >19 {
 D:D+1}
if H >24 {
 D:D+1}
if D < 0 {
 D:0}
 result: DdX
}
