function: explode N:n at C:n {
 if N >= C { result: 10 + [explode d10 at C] }
 result: 0
}

function: N:n x P:n at C:n {
 result: ([highest 1 of Nd[explode d10 at C]]) + 1dC + P
}

set "maximum function depth" to 6

\ 命中骰 在 output 後面修改骰子 [Ndx+M@C] => [N x M at C] ，沒有 M 就打 0\
output [24 x 6 at 7] named "命中期望值"

\ 傷害骰 d10 + ([上面output的內容]/10)d10 + 傷害加值 \
output d10 + ([24 x 6 at 7]/10)d10 + 60 named "傷害期望值"
