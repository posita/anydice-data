\ Only change the number 2 in this function (at the end) to the number of misses that fail the roll \

function: calc VALUES:s in SEQUENCE:s {
 COUNT: 0
 TOTAL: 0
 loop P over {1..#VALUES} {
  COUNT: COUNT + (P@VALUES = SEQUENCE)
  TOTAL: TOTAL + SEQUENCE
 }
 
  result: (COUNT < 2) * TOTAL
}

\ Here you can change the 3 to the number of dice and the {0, 0, 1, 1, 1, 2} to the wanted die (0 = miss) \
output[calc 0 in 3d{0, 0, 1, 1, 1, 2}]
