\by default, gives hit rolls with psidice in terms of their excess\
\on the minimum hit requirement, for easy hit chance calculation\
\i.e. anything 0 or above is a hit, and anything negative is a miss\
\can be turned into a normal hit roll by removing "-(AC-PROF-DEX)" from the output line\
AC:16 
PROF:3
DEX:5
function: if NUM:n add prof add dex less than AC:n then add BONUS:d {
   if NUM+PROF+DEX<AC {
      result:NUM+BONUS
   }
   else {result:NUM}
}
function: ROLL:d add prof add dex less than AC then add BONUS:d{
   result: [if ROLL add prof add dex less than AC then add BONUS]
}
loop PSIDIE over {8,10,12} {
    output [d20 add prof add dex less than AC then add dPSIDIE]-(AC-PROF-DEX) named "d[PSIDIE] psi dice hits on 0 or more"
}
