\Rolling too many dice will cause an error, so adjust DICEMIN and DICEMAX to keep your dice reasonable!\

DICEMIM: 1 \Starting number of dice rolled\
DICEMAX: 8 \Ending number of dice rolled\
DIEBONUS: 10 \Bonus per die\
CURRENTPROGRESS: 0 \Your current project progress\
PROGRESSTARGET: 400 \The total progress the project needs\

function: evaluate ROLL:s {
 SUCCESSES: ROLL >= 0
 if SUCCESSES > 0 { result: 1 }
 result: 0
}

loop DICE over {DICEMIM..DICEMAX} {
 output [evaluate DICEd100+DIEBONUS*DICE+CURRENTPROGRESS-PROGRESSTARGET] named "[DICE]"
}
