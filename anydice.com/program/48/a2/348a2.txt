\ spellstrike with slashing gust with both attacks of the spell being targetted at one person (due to lack of statement that the spell needs to hit a different person with the second hit, it is technically legal to use it this way) \

\variables\
MISS: 0
HIT: 1
CRITICALHIT: 2
HITDAMAGE: 2d6 + 10d4 + 9
CRITICALDAMAGE: (2d6 + 10d4 + 9) * 2 + 16d4 \ 16d4 in here is the assumed 4 turns of persistent damage with it's 25% of actually being stopped, crude, but effective \
MISSDAMAGE: 0
ATTACK: 4 + 12 + 1 + 2
AC: 26
DTWT: d20

function: attack ROLL:n vs DEFENSE:n {
  if ROLL = 20 {
    if ROLL >= DEFENSE {result: CRITICALHIT }
    if ROLL < DEFENSE {
      if ROLL >= DEFENSE - 10 {result: HIT}
    }
  }
  if ROLL + ATTACK >= DEFENSE + 10 { result: CRITICALHIT }
  if ROLL + ATTACK >= DEFENSE { result: HIT }
  if ROLL + ATTACK < DEFENSE { result: MISS }
}

function: damage for ATTACKRESULT:n {
 if ATTACKRESULT = CRITICALHIT {
  result: CRITICALDAMAGE
 }
 if ATTACKRESULT = HIT {
  result: HITDAMAGE
 }
 result: MISSDAMAGE
}

function: surestrike ROLLO:n vs ROLLT:n {
  if ROLLO > ROLLT {result: ROLLO}
  if ROLLO < ROLLT {result: ROLLT}
  if ROLLO = ROLLT {result: ROLLO}
}

\output all functions at once :) \
output [damage for [attack [surestrike d20 vs d20] vs AC]]
