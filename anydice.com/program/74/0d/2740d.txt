
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE }
 if ROLL = 1 {result: 0}
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE}
 result: 0
}

TARGETAC: d{10..15}

PROFICIENCY: 2
ATTACKMOD: 3
DMGBONUS: 0
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 1
DAMAGE: 1d8 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d8 + ATTACKMOD + DMGBONUS
output ATTACKTIMES d[attackroll 1@2d20 vs TARGETAC] named "adv attack"  

\ NormalRoll = 1d20 ||| AdvantageRoll = 1@2d20 ||| DisadvantageRoll 2@2d20 \

\ "TARGETAC: d{X..Y}" = Calculates average of all AC values between X and Y ||| "TARGETAC: X" can be used to check avg damage against a unique AC value. \
