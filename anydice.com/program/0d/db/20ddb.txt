\if a dice roll contains more than one zero, it sets the result to zero\
\otherwise it just returns the original roll\
function: fail on two or more zeros in DICE:s{
if [count 0 in DICE] > 1{
 result: 0
}
else{
 result: DICE
}
}

\explodes the "n-th face" of a die, regardless of its value compared to other faces\
\so it would explode a 4 on a d{1,6,6,4} \
function: top explode BANGDICE:s with explode on BANGVAL:n{
BANGS: BANGDICE/1000
result: BANGDICE - (BANGS * 1000) + (BANGS * BANGVAL)
}

\creates a series of output for different numbers of dice\
loop N over {2..6}{
output [top explode [fail on two or more zeros in N d [explode d{0,0,2,3,3,1000}]] with explode on 4] named "[N] exploding custom dice"
}
