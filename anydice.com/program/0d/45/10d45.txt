\ FU-WORLD \
\ Always roll at least 2d \
\ If you are at advantage, add as many dice as advantages and read only the two best \
\ If you are at disavantage, same as above, but read only the two worst \
\ On each read dice, a 4-6 brings a success \
\ With no success, brace yourself \
\ With 1 success, you did it, but you face a cost, a hard choice or an ugly outcome ; you may prefer to fail \
\ With 2 successes, you dit it fully, getting the best that could be hoped for \

loop N over {2..6}{
 output [lowest 2 of Nd{0,1}] named "[N] d6 kL 2"
}

loop N over {2..10}{
 output [highest 2 of Nd{0,1}] named "[N] d6 kH 2"
}
