\**** Fighting modifiable Calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}


\looped against enemy AC from 15-20\
ENEMYAC: d{13..20}


\Barb20 gsword vanilla\
ATTACKMOD: 7
DMGBONUS: 4
PROFICIENCY: 6
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 3
DAMAGE: 2d6 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 7d6 + ATTACKMOD + DMGBONUS
output ATTACKTIMESd[attackroll [highest 1 of 2d20] vs ENEMYAC] named "B20 gsword: RA"
