\For Great Weapon Fighter with a Greatsword you would use the following:                                             \

\NUMDIE: 2   (Greatsword damage: [2]d6)                                                                              \

\VARIABLES - Edit these to change output!                           \
\-------------------------------------------------------------------\
MOD:3     \This is the damage modifier. I.E. Strength modifier, etc.\
DIE:4     \Die size. I.E. DIE:6 for d6                              \
NUMDIE:3  \Number of damage dice. I.E. NUMDIE:2 for Greatsword/Maul \
MIN:2     \Lower limit for Great Weapon Fighter. Default of 2       \
CRIT: 20  \Minimum roll for a crit. Default 20                      \                                
\-------------------------------------------------------------------\

\It's been documented for others to understand functionality \
\------------------------------------------------------------\

\Description -----------------------------------------------------------------------------------\
\This function returns the re-rolled value of the die if it is lower or equal to the lower limit\
\Variables -------------------------------------------------------------------------------------\
\A = Initial input value. I.E. 1 to 6 for d6                                                    \
\B = Lower reroll limit. I.E. 2 for Great Weapon Fighter                                        \
\C = Dice value. I.E. 6 for d6                                                                  \
\-----------------------------------------------------------------------------------------------\

function: reroll A:n B:n C:n{ 
 if A <= B { 
  A: 1dC 
 }  
 result: A 
}

\Description ---------------------------------------------\
\This function simulates the Great Weapon Master feat     \
\Variables -----------------------------------------------\
\A = Dice value I.E. A=6 for 1d6                          \
\B = Number of damage dice I.E. B = 2 for Greatsword (2d6)\
\C = Lower limit for reroll. Default 2                    \
\---------------------------------------------------------\

function: greatweaponmaster A:n B:n C:n{
 X:0
 loop N over {1..B} {
  X: X + [reroll 1dA C A]
 }
 result: X
}

\Description ---------------------------------------------\
\This function returns 2 if the user crits                \
\Variables -----------------------------------------------\
\A = Dice roll. Default 1 to 20 (1d20)                    \
\B = Minimum critical roll. Default 20                    \
\---------------------------------------------------------\

function: getcrit A:n B:n { 
 if A >= 20 { 
  result:2 
 }
 else { 
  result:1 
 } 
}

MUL:[getcrit 1d20 CRIT]

output (MUL * [greatweaponmaster DIE NUMDIE MIN]) + MOD
