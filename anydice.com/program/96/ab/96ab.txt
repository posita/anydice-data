X: 1d6
RX : 1d6
Y: 1d6
RY : 1d6

function: reroll XA:n if less than A:n with YA {
  if XA<=A
  {
    X: YA
  }
  else
  {
    X: XA
  }
result: X
}

output X+Y named "Greatsword"
output [reroll X if less than 2 with RX] + [reroll Y if less than 2 with RY] named "Greatsword with GWF"
output 1d8 + 2 named "Longsword with Duelist"
output 1d12 named "Greataxe"
output [reroll 1d12 if less than 2 with 1d12] named "Greataxe with GWF"
