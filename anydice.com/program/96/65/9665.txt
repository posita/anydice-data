function: reroll R:n under N:n A:n B:n C:n {
   if R > N {result: R} else if A > 1 {result: A} else if B > 1 {result: B} else if C > 1 {result: C} 
}

function: stats A:n B:n C:n D:n {
   LOWEST: A
   if LOWEST > B {LOWEST: B}
   if LOWEST > C {LOWEST: C}
   if LOWEST > D {LOWEST: D}
   result: A+B+C+D-LOWEST
}

output [stats [reroll 1d6 under 1 1d6 1d6 1d6] [reroll 1d6 under 2 1d6 1d6 1d6] [reroll 1d6 under 2 1d6 1d6 1d6] [reroll 1d6 under 2 1d6 1d6 1d6] ]
