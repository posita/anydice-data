\- Computes the higher of two rolls -\
function: highest of A:n and B:n {
 if A > B { result: A }
 result: B
}

\- Computes the highest result of N rolls -\
function: highest A:d of N:n {
 S : A
 loop I over {2..N}{
  S : [highest of S and A]
 }
 result : S
}
