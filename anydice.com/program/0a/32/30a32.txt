\
H&G  2d12 vs. 10-stat, roll HIGHER.
Critical: if 2 successes equal to each other.
Stats range = 0-7 (avg=4)

 -1: Miss.
 0: Success, but with undesired consequences.
 +1: Clean success.
 +2: Critical success!
\

function: fate A:s and B:s and M:n {
 R: A+B+M
 O: 2
 if R < 18 {
   O: 1
 }
 if R < 11 {
   O: -1
 }
 if R < 5 {
   O: -2
 }
 result: O
}

loop M over {-5..5} {
 output [fate 1d10 and 1d10 and M] named "[M]"
}
