\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

ATTACKMOD: 5
PROFICIENCY: 5
HEX: 1d6
DMGBONUS: ATTACKMOD
ACCBONUS: 5
CRITROLL: 20
ATTACKTIMES: 3
DAMAGE: 1d10 + DMGBONUS + HEX
CRITDAMAGE: 2d10 + DMGBONUS + HEX + HEX
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "Agonizing Blast with Hex"
