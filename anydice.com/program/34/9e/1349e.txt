\Conan Skill Success probabilities\

FOC:5   \ <---- set your focus\
TN:19  \ <---- set you target number (skill+ability)\

function: D:s target T:n focus F:n {
   if (D<=F) {result:2}
   else if (D<=T){result: 1}
   else {result:0}
}

loop S over {2..5} {

  output Sd[d20 target TN focus FOC] named "[S]d20 Target [TN] Focus [FOC]"
}
