function: DIE:d against difficulty of TARGETNUMBER:n {
  Y: 1@DIE                                         \ Högsta utfallet av tärningen.       \
  X: #DIE@DIE                                      \ Lägsta utfallet av tärningsmängden. \
  result: [checkmax Y X TARGETNUMBER]          \ Ett resultat på 0 innebär misslyckat slag. \
}

function: checkmax D:n DD:n TN:n {
  if D >= TN { result: DD }                         \ Om högsta tärningen uppnått svårighetsgraden. \

  result: 0                                         \ ...vilket innebär att detta är misslyckat slag. \
}

output [2d10 against difficulty of 1@2d10]         \ Mot ett annat värde (högsta tärningen av 2d10) \
output [2d10 against difficulty of 6]              \ Mot 6 i svårighetsgrad. \
