\
 SOTURNOS Probabilidade dos resultados
 0: Azar ou falha. O pior acontece.
 1: Sucesso, mas com consequências indesejáveis.
 2: Sucesso total.
 3: Sucesso crítico!
\

function: soturnos ROLL:s {
 if 1@ROLL+1 >= 6 {
  result: 2 + (2@ROLL >= 6)
 }
 result: 1@ROLL+1 >= 4
}

output [soturnos 2@2d6] named "0d"
loop P over {1..6} {
 output [soturnos Pd6] named "[P]d"
}
