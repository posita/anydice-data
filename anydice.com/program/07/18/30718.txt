function: reroll lowest of ROLL:s as REROLL:d if less than MIN:n {
  SUM: ROLL + 0  \ force the sequence to be summed into a single number! \
  if SUM >= MIN {
    result: ROLL + 0
  } else {
    REST: {1..#ROLL-1}@ROLL  \ all but the lowest die \
    result: REST + REROLL
  }
}

output 2d6 named "2d6"
output 2d6 + 1 named "2d6 + 1"
output 2d6 + 2 named "2d6 + 2"
output [reroll lowest of 2d6 as d6 if less than 7] named "2d6 replace lowest if < 7"
output [highest 1 of 2d6] + 1d6 named "2d6 always replace lowest"
