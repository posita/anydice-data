\ Visar utfallet 1 så är det ett lyckat resultat. \

loop AG over {2..6} {
  output [count {AG..6} in 1@2d6] named "Slå 1T6 mot [AG]+ med re-roll"
}

\ AG           = svårighetsgraden.                                                             \
\ {AG..6}      = mellan svårighetsgraden och 6.                                                \
\ 1@2d6        = högsta utfallet av två T6:or.                                                 \
\ count X in Y = fördefinierad funktion. Visar 1 om tärningsslaget Y resulterar i sekvensen X. \
