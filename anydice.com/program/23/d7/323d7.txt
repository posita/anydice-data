function: rounds R:n aattack AA:s adefense AD:s aarmor AS:n ahealth AH:n battack BA:s bdefense BD:s barmor BS:n bhealth BH:n{
if BD+BS<AA{BH:BH+(BD-AA+BS)}
if AD+AS<BA{AH:AH+(AD-BA+AS)}
if 1>AH{result: R}
if 1>BH{result: R}
result: [rounds R+1 aattack AA adefense AD aarmor AS ahealth AH battack BA bdefense BD barmor BS bhealth BH]
}
output [rounds 1 aattack 1@2d6 adefense 1@2d6 aarmor 0 ahealth 6 battack 1@2d6 bdefense 1@2d6 barmor 0 bhealth 6]
