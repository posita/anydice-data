\ a fudge die: note that we'll treat +1 as a success and -1 as a (possible) failure below for simplicity \
F: {+1:2, 0:2, -1:2}

\ this function returns the net total of successes (+1) minus failures (-1) in ROLL, with failures capped at DIFF \
function: ROLL:s difficulty DIFF:n {
  SUCCESSES: [count +1 in ROLL]
  FAILURES: [lowest of DIFF and [count -1 in ROLL]]
  result: SUCCESSES - FAILURES
}

\ loop over each pool size and difficulty, output probability of at least one success \
loop N over {1..12} {
  loop D over {0..6} {
    output [NdF difficulty D] > 0 named "[N]dF difficulty [D]"
  }
}
