
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 18\
ENEMYAC: 16

ATTACKMOD: 4
DMGBONUS: 6d6 + 11
PROFICIENCY: 4
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 1
DAMAGE: 1d12 + ATTACKMOD + DMGBONUS
CRITDAMAGE: DAMAGE + DAMAGE
output ATTACKTIMESd[attackroll [highest 1 of 2d20] vs ENEMYAC] named "Longsword +2 proficiency +3 Attack modifier"
