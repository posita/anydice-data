\ AGE System \
function: age a DA:s b DB:s stunt SD:s bonus B:n sg SG:n {
  RES: 0
  X: DA + DB + SD + B
  if X >= SG {
    RES: 10
    if DA = DB {result: RES + SD}
    if DA = SD {result: RES + SD}
    if DB = SD {result: RES + SD}
  }
  result: RES
} 
output [age a 1d6 b 1d6 stunt 1d6 bonus 0 sg 18]
