\**** Flynn Attack****\

\*** Attack Action***\
function: attackrolla ROLL:n vs AC:n{
 if ROLL = 20 { result: CRITDMGA }
 if ROLL + ATKBONUS >= AC { result: DAMAGEA }
 if ROLL + ATKBONUS < AC { result: 0 }
 if ROLL = 1 { result: 0 }
 result: 0
}

\*** Pole Arm Bonus Action ***\
function: attackrollb ROLL:n vs AC:n{
 if ROLL = 20 { result: CRITDMGB }
 if ROLL + ATKBONUS >= AC { result: DAMAGEB }
 if ROLL + ATKBONUS < AC { result: 0 }
 if ROLL = 1 { result: 0 }
 result: 0
}

\** Variables **\
ATKBONUS: 6
AC: 16
DAMAGEA: 1d10 + 4
CRITDMGA: 2d10 + 4
ATTACKS: 1
DAMAGEB: 1d4 + 4
CRITDMGB: 2d4 + 4

\* Output *\
output ATTACKSd[attackrolla d20 vs AC] named "Attack Action"
output 1d[attackrollb d20 vs AC] named "Bonus Action"
