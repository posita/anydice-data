\**** Damage calculator ****\

\*** Attack Roll ***\
function: attackroll ROLL:n vs AC:n{
 if ROLL = 20 { result: CRITDMG }
 if ROLL + ATKBONUS >= AC { result: DAMAGEA }
 if ROLL + ATKBONUS < AC { result: 0 }
 if ROLL = 1 { result: 0 }
 result: 0
}

\*** SAVING THROW FOR HALF ***\
function: savingthrow ROLL:n vs DC:n {
 if ROLL + ABILITYMOD <= DC { result: DAMAGES }
 if ROLL + ABILITYMOD > DC { result: HALFDAMAGE }
 result: 0
}

\** VARIABLES **\
ATKBONUS: 6
AC: 16
DAMAGEA: 1d10 + 4
CRITDMG: 2d10 + 4
ATTACKS: 1

ABILITYMOD: +2
DC: 14
DAMAGES: 1d12
HALFDAMAGE: 0
TARGETS: 1
