\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 else if ROLL = 1 { result: 0 }
 else if ROLL + 4 + 7 + ACCBONUS >= AC { result: DAMAGE }
 else {result: 0}
}


\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}
ENEMYBC: d{15..20}
ATTACKMOD: 7
DMGBONUS: 7
PROFICIENCY: 4
ACCBONUS: 0
CRITROLL: 19-20
ATTACKTIMES: 2

DAMAGE: 2d6 + ATTACKMOD + DMGBONUS + 6d10
CRITDAMAGE: DAMAGE * 2
output ATTACKTIMESd[attackroll d20 vs 15] named "Rabi Feather Sword"

output 1d[attackroll d20 vs ENEMYBC] + ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "Rabi Total"
