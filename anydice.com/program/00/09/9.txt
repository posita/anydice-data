KEEP: 3
DICE: 5

\
LotFR, with possible Emphasis mechanic.
Assumes dice are rolled, then all 1s are rerolled once.
Then highest dice are selected, then 10s are exploded.
Then the results are summed.
\

function: keep K:n and explode ROLL:s {
 EXPLOSIONS: ROLL = 10
 if EXPLOSIONS > K { EXPLOSIONS: K }
 result: {1..K}@ROLL + EXPLOSIONS d KABOOM
}

KABOOM: [explode d10] \ saved for speed \

output [keep KEEP and explode DICE d10] named "[DICE]k[KEEP] regular"
output [keep KEEP and explode DICE d{1,2..10:11}] named "[DICE]k[KEEP] emphasis"
