function: A vs B {
if A < B { M: A } else if A > B { M: B } else {M : 0}
AA: A+M
BB: B+M
R: 1d{0..(AA)} - 1d{0..(BB)}
result: R }

output [1 vs 2]>0 named "1v2 Win"
output [1 vs 2]<0 named "1v2 Lose"
output [1 vs 2]=0 named "1v2 Tie"
