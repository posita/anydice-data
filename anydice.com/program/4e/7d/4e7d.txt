\ Poker dice - figure probabilities of each roll

Results of pokerdice:
* Five of a kind = 6
* Four of a kind = 5
* Full house = 4
* Three of a kind = 3
* Two pair = 2
* One pair = 1
* High die = 0 \

function: pokerdice S:s {
  TWO: 0
  THREE: 0
  FOUR: 0
  FIVE: 0
  loop N over {1..6} {
    J: (S = N)
    if J = 5 { FIVE: FIVE + 1 }
    else if J = 4 { FOUR: FOUR + 1 }
    else if J = 3 { THREE: THREE + 1 }
    else if J = 2 { TWO: TWO + 1 }
  }
  if FIVE { result: 6 }
  else if FOUR { result: 5 }
  else if THREE { result: 3 + TWO }
  else { result: TWO }
}

output [pokerdice 5d6]
