\ first part defines the range function and the second part defines a dice type called special. The last part asks for an output of the probability of getting in that range.\

function: range on DIE:n between LOW:n and HIGH:n {
  if DIE >= LOW & DIE <= HIGH { result: 1 }
  result: 0
}

SPECIAL: d{0,1,1,5,6,9}

output [range on SPECIAL between 4 and 7]
