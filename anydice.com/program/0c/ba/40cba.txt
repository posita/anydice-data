\ CONFIGURATION \
POOL: 10        \ Change this to your number of dice \
DIFFICULTY: 6  \ Change this to your target number (3-9) \

\ MECHANIC 1: WITH RULE OF ONES \
\ 1s count as -1 success \
\ 2 through (Difficulty-1) count as 0 \
\ Difficulty through 10 count as +1 success \

\ We define a custom 10-sided die using the sequence syntax: \
\ -1 appears once (the 1 face) \
\ 0 appears (DIFFICULTY - 2) times \
\ 1 appears (11 - DIFFICULTY) times \

RULE_OF_ONES_DIE: d{-1, 0:(DIFFICULTY - 2), 1:(11 - DIFFICULTY)}

output POOL d RULE_OF_ONES_DIE named "With Rule of Ones"


\ MECHANIC 2: WITHOUT RULE OF ONES \
\ 1s count as 0 successes (same as 2s, 3s, etc.) \
\ Difficulty through 10 count as +1 success \

\ We define a custom 10-sided die: \
\ 0 appears (DIFFICULTY - 1) times \
\ 1 appears (11 - DIFFICULTY) times \

NO_ONES_DIE: d{0:(DIFFICULTY - 1), 1:(11 - DIFFICULTY)}

output POOL d NO_ONES_DIE named "Without Rule of Ones"
