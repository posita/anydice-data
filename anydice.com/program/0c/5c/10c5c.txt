\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
if ROLL = 1 { result: 0 }
if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
result: 0
}

\looped against enemy AC from 16-20\
ENEMYAC: d{16..20}

ATTACKMOD: 3
DMGBONUS: 3
PROFICIENCY: 3
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 2
DAMAGE: 1d8 + DMGBONUS + 1d6
CRITDAMAGE: 2d8 + DMGBONUS + 2d6
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "LS & Hex"
