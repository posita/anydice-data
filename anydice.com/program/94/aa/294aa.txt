\** Attack Roll Function **\
function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ABILITYMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\** Attack looped against enemy AC from 15-20 **\
ENEMYAC: d{15..20}

\** Edit Here **\
ABILITYMOD: 5
DMGBONUS: 13
PROFICIENCY: 6
ACCBONUS: 3
CRITROLL: 20
ATTACKTIMES: 4
DAMAGE: (2d6+3d10+2d6+3d8)+ ABILITYMOD + DMGBONUS
CRITDAMAGE: (4d6+6d10+4d6+3d8)+ ABILITYMOD + DMGBONUS

\** Output with Normal, Advantage and Disadvantage **\
output ATTACKTIMESd[attackroll d20 vs ENEMYAC] named "normal attack"
output ATTACKTIMESd[attackroll [highest 1 of 2d20] vs ENEMYAC] named "attack with advantage"
output ATTACKTIMESd[attackroll [lowest 1 of 2d20] vs ENEMYAC] named "attack with disadvantage"
