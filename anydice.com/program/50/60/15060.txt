\Big Fuck-off Function of Orc Variant and Half-Orc Critical damage\
\Basic premise is that Variant Orcs had Exploding dice on crits (limit 2)\
\And the use of a new weapon: Katana with the Versatile (2d4) Property\
\The most useful views are Normal Graph, Transposed Table, and Summary Table\


DIE:4    \Die size. I.E. DIE:6 for d6                              \
NUMDIE:4  \Number of damage dice. I.E. NUMDIE:2 for Greatsword/Maul \
MIN:2     \Lower limit for Great Weapon Fighter. Default of 2       \  
set "explode depth" to 2                          

function: reroll A:n B:n C:n{ 
 if A <= B {  A: 1dC }  
 result: A 
}

function: greatweaponmaster A:n B:n C:n{
 X:0
 loop N over {1..B} { X: X + [reroll 1dA C A]}
 result: X
}

function: greatweaponmasterb A:n B:n C:n{
 X:0
 loop N over {1..B} { X: X + [reroll [explode 1dA] C A]}
 result: X
}

output ( [greatweaponmasterb DIE NUMDIE MIN]) named "Orc Katana Crit GWF"

output 4d [explode d4] named "Orc Katana Crit"
