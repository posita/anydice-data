SMITE:2d8

function: savage DICE:d{
 TMP: 0
 loop X over {1..#(DICE)} {
  TMP: TMP + [highest 1 of 2d[maximum of X@DICE]]
 }
 result: TMP
}

output 2d6 + SMITE + 1d6 + 1d4       named "normal"
output 2d6 + SMITE + 1d6 + 1d4 + 10  named "HWM"
output [savage 2d6] + [savage SMITE] + [savage 1d6] + [savage 1d4]      named "Savage attacker"
output [savage 2d6] + [savage SMITE] + [savage 1d6] + [savage 1d4] + 10 named "Savage attacker + HWM"
