function: hits A B {
 X: 0
 loop N over {1..A}{
  X: X + (1dB / 4)
 }
 result: X
}

output [hits 1 10] >= 1 named "Hits with 1d10"
output [hits 2 10] >= 2 named "Hits with 2d10"
output [hits 3 10] >= 3 named "Hits with 3d10"
output [hits 4 10] >= 4 named "Hits with 4d10"
output [hits 5 10] >= 5 named "Hits with 5d10"
