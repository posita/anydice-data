function: hits A B {
 X: 0
 loop N over {1..A}{
  X: X + (1dB / 4)
 }
 result: X
}

output [hits 1 12] >= 1 named "Success with 1d12"
output [hits 2 12] >= 2 named "Success with 2d12"
output [hits 3 12] >= 3 named "Success with 3d12"
output [hits 4 12] >= 4 named "Success with 4d12"
output [hits 5 12] >= 5 named "Success with 5d12"
