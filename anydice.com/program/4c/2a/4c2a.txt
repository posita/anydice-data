\ Rolls N custom dice \
\ 1: Hobbits; 2: Precious; 3: Blank; 4: Eye \
function: roll N:n dice {
  result: Nd{1,1,2,2,3,4}
}

\ ------------------------------ \

\ Counts the number of X results in Y rolls \
function: X:n in Y:n dice {
  result: [count X in [roll Y dice]]
}

\ ------------------------------ \

\ Calculate number of dice to re-roll \
function: dice available with N:n of M:n dice locked{
  result: M - N
}

\ ------------------------------ \


\ Player start with 3 dice \
BASE: 3


\ Probabilities of scoring 2s on first roll \
output [2 in BASE dice]


\ Get probabilities for number of dice for a  second roll \
LOCKED: [4 in BASE dice]
ROLL_TWO: [dice available with LOCKED of 3 dice locked]


\ Probabilities of scoring 2s on second roll \
output [2 in ROLL_TWO dice]
