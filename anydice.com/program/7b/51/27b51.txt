\
 Blades in the Dark-style with d10s
 -1: Bad bad bad bad.
 0: A Miss/Failure; Things go bad.
 1: Weak Hit/Mixed Success. i.e. a Success with but with undesirable consequences
 2: Good Hit/Clean success.
 3: Strong Hit/Critical success! Success but better things happen
\

function: blades ROLL:s {
 if 1@ROLL = 10 {
  result: 2 + (2@ROLL = 10)
 }
 if 1@ROLL <3 {
  result: -1
 }
  result: 1@ROLL >= 7
}

output [blades 2@2d10] named "0d"
loop P over {1..7} {
 output [blades Pd10] named "[P]d"
}
