function: attack A:n B:n roll D:n BONUS:n vs AC:n {
  if D = 20 {
    result: A+A+B
  } else if D = 1 {
    result: 0
  } else if D + BONUS >= AC {
    result: A+B
  } else {
    result: 0
  }
}

AC:17
ADV: [highest 1 of 2d20]
DIS: [lowest 1 of 2d20]
NORM: d20
D: NORM

A: [attack d12+1d6 7 roll D 10+d4 vs AC]
AB: [attack d12+1d6 7 roll ADV 10+d4 vs AC]
B: [attack d8 4 roll D 0 vs AC]
output A+A+B
output AB+AB
