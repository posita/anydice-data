\ Random number from A to rolled number B\
function: rand A:n up to B:s {
 NEW: {}
 loop N over B{
  NEW: {NEW, A..N}
 }
 result: NEW
}

\ Vendor shortage stock distribution with max of B\
function: shortstock to B:n {
 result: [rand 0 up to {(B/2)..(B)}]
}

output [shortstock to 10] named "Shortage Stock 10"
