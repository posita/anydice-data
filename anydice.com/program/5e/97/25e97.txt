function: roll ROLL:n at plus ATTACKBONUS:n vs AC:n ac for CRITABLEDAM:d plus HITDAM:d {
    if ROLL >= CRITRANGE {
    result: CRITABLEDAM + CRITABLEDAM + HITDAM
    }
        if ROLL = 1 {
        result: 0
        }
            if ROLL >= AC - ATTACKBONUS {
            result: CRITABLEDAM + HITDAM
            }
                else {
                result: 0
                }
}

    \"Enter your your attack information below. Advantage is written 1@2d20 and disadvantage is 2@2d20. CRITDAMAGE is the damage that should double as a crit; your flat damage bonus and other non-critable damage sources is HITDAM. To calculate per round damage with multiple attacks, set your attack info to a capitalized variable and add the variables together. Multiplication will not reroll the attack dice, so you must add."\
