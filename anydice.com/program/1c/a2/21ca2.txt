function: explode N:n THRESHOLD:n {
 if N >= THRESHOLD { result: [helper d6 1 THRESHOLD] }
 result: 0
}

function: helper N:n SUCCESSES:n THRESHOLD:n {
 if N >= THRESHOLD { result: [helper d6 SUCCESSES+1 THRESHOLD] }
 result: SUCCESSES
}

set "maximum function depth" to 10

SIMPLE:d{1,2,0,0,0,0}
SKILL:8

\"lowest" is to limit the infinite but very thin tail of the exploding die to a reasonable maximum\
output [lowest of SKILLd[explode d6 5] and (SKILL*3)] named "Explode"
output SKILLdSIMPLE named "Approximation"
