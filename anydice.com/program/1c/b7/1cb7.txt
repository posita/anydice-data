ATTACKBONUS: 0
DEFENSEBONUS: 0

function: outcome of AONE:n ATWO:n ATHREE:n AB versus D:n {
 A: AONE+ATWO+ATHREE+AB
 if A > D
 {
   if ((AONE=ATWO)|(ATWO=ATHREE)|(AONE=ATHREE))
   {
    SP: (A-D)/2
    result: SP
   }
   else
   {
     result: 0
   }
 }
 else
 {
   result: -1
 }
}

ATTACK: 3d6+ATTACKBONUS
DEFENSE: 3d6+DEFENSEBONUS
output [outcome of 1d6 1d6 1d6 ATTACKBONUS versus DEFENSE] named "Outcomes for DEF [DEFENSEBONUS] vs ATT [ATTACKBONUS]"
