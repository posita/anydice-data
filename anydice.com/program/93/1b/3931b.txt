M:6     \ die size i.e. sides of the die \
N:2     \ number of dice \

\ this function makes a reroll die e.g. brutal \
function: reroll REROLL:s on ROLL:n to REPLACEMENT:d {
    if [REROLL contains ROLL] {
        result: REPLACEMENT
    }
    result: ROLL
}

RDIE: [reroll 1 on d6 to d6] \ this die has one 1, and 7 of each of the other numbers \

output NdRDIE named "[N]d8 brutal"
