GLANCE: 3
HIT: 9

function: ROLL:n wound S:n vs T:n dmg DMG:d {
 if ROLL <= GLANCE - S + T {
result: 1
} else if ROLL <= HIT {
result: DMG
} else {
result: DMG * 2
}
}


BITE: [d10 wound 4 vs 3 dmg 2d8+2]
CLAW: [d10 wound 4 vs 3 dmg d8+2]

output BITE * 1 + CLAW * 2 named "jagras"
output  BITE * 2 + CLAW * 2 named "anger jagras"
output [d10 wound 3 vs 4 dmg d8+1] * 8 named "4 yabbos"
