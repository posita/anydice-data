function: oppose A and D using S{
  result: (D+1)@(A+D+1)dS
}
\Variables Below:\
S: 6 \Number of sides\
LA: 0 \Lowest attacker value to test\
HA: 6 \Highest attacker value to test\
LD: 0 \Lowest defender value to test\
HD: 6
 \Highest defender value to test\

loop N over {LA..HA}{
  loop M over{LD..HD}{
  output [oppose N and M using S] named "[N] opposes [M] using d[S]"
 }
}
