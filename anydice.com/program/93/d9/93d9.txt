\This looks at the concept where you roll a group of dice and come up with a total success number based on a set difficulty.\
\But we also presume an armor value, whereby successes are deducted from that.\

function: evaluate ROLL:s {

 MAJOR: ROLL >= 11
 MINOR: ROLL >= 9
 MINFAIL: ROLL <= 3
 MODFAIL: ROLL = 1

 GOOD: ((MINOR-MAJOR) + (MAJOR*2))
 BAD: ((MINFAIL-MODFAIL) + (MODFAIL*2))

 result: (GOOD - BAD)-2
 
}

loop DICE over {1..7} {
 output [evaluate DICE d12] named "[DICE]d"
}
