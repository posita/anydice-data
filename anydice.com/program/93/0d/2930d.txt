function: echo P:s {
if (1@P = N@P) { result: P + (N-1)d(S-2)}
if ((1@P = 3@P) | (2@P = N@P)) { result: P + (N-2)d(S-2)}
if ((1@P = 2@P) | (2@P = 3@P) | (3@A = N@P)) { result: P + (N-3)d(S-2)}
result: A
}

N: 4 \ N for Number of dice in the initial dice pool \
S: 10 \ S for Size of the dice in the initial dice pool \ 
P: NdS \ P for initial dice Pool \

output [echo A] named "echoing [A]"
output 4d[explode dS] named "exploding [A]"
output A named "[A]"
