\ ● W represents a Success die under the Weary condition. \
W: {0,0,0,4..6}
\ ● D represents the difficulty level, the Target Number. Change it
	as needed; it may be a single number, a sequence, or a range. \
loop D over {14,18} {
\ ● S represents the skill level, the number of pips, the number of d6. \
 loop S over {0..6} {
  function: onering FEAT:s SUCCESS:s DIFF:n{
   if FEAT = 12 {TOTAL: 1000}
   else {TOTAL: FEAT + SUCCESS}
   TENG: [count 6 in SUCCESS]
   if TENG >1 {LEVEL: 3}
   if TENG = 1 {LEVEL: 2}
   if TENG = 0 {LEVEL: 1}
   if TOTAL > DIFF {result: LEVEL}
   result: 0
}
\ ● Comment out one or the other line below to focus on one condition. \
   output [onering d12 Sd6 D] named "skill-[S], TN=[D], Normal"
   output [onering d12 SdW D] named "skill-[S], TN=[D], Weary"
 } \ end loop D \
} \ end loop S \
