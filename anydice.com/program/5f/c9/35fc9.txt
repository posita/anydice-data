function: countsuccesses A:s C:n MIN:n {
  S: 0
  loop I over A {
    if I >= C {
      S: S + 2
    } else if I >= MIN {
      S: S + 1
    }
  }
  result: S
}

\ number of dice to roll \
N: 2

\ count successes for d10, where 10 is a double and 5+ is success \
TEN: [countsuccesses Nd10 10 5] 

\ count successes for d6, where 6 & 5 are a double and nothing else is a success \
IMPSIX: [countsuccesses Nd6 5 5]

\ Output both success counts to compare in the graph at least view \
output TEN named "Normal D10 Success Rate"
output IMPSIX named "Risky D6 Success Rate"

\ outputs (normal table) how likely is the D10 to result a better / worse performance to the Risky D6 \
output (TEN > IMPSIX) - (TEN < IMPSIX) named "Chances of D10 outpreforming the Risky D6"

\ outputs (normal table) how much the D10 result might be better performing than the Risky D6 \
 output (TEN - IMPSIX) named "Chances of D10 outpreforming the Risky D6"
