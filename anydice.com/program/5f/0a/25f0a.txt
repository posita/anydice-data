\ Success Chance Calculator \

\ Reroll function, do not touch \
function: ROLL:n reroll BAD:s as REROLL:d {
  if ROLL = BAD { result: REROLL }
  result: ROLL
}


\ Change X to change the number of dice being rolled \
X: 5



\ The rolls are below. They count the odds of getting a Hit, aka, above x number, on a d6. X determines the number of d6's rolled, while Advantage and Disadvantage change the target number, from only a 6 on Disadvantage, to 3-6 on Major Advantage. \

\ Basic Variables, used for Rerolls \
A: [d6 reroll {1, 2} as d6]
B: [d6 reroll {1, 2, 3} as d6]
C: [d6 reroll {1, 2, 3, 4} as d6]
D: [d6 reroll {1, 2, 3, 4, 5} as d6]

\ The basic rolls \
output [count {3, 4, 5, 6} in {X}d6] named "Weak Major Advantage"
output [count {3, 4, 5, 6} in {X+5}d6] named "Major Advantage"
output [count {3, 4, 5, 6} in XdA] named "Major Advantage, Reroll Misses Once"
output [count {4, 5, 6} in {X}d6] named "Weak Advantage"
output [count {4, 5, 6} in {X+5}d6] named "Advantage"
output [count {4, 5, 6} in XdB] named "Advantage, Reroll Misses Once"
output [count {5, 6} in {X}d6] named "Weak Normal"
output [count {5, 6} in {X+5}d6] named "Normal"
output [count {5, 6} in XdC] named "Normal, Reroll Misses Once"
output [count {6} in {X}d6] named "Weak Disadvantage"
output [count {6} in {X+5}d6] named "Disadvantage"
output [count {6} in XdD] named "Disadvantage, Reroll Misses Once"
