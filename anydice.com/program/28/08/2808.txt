\Click calculate then click the view and data options to more usefully analyse the results. In my opinion, an exploding d10 is sufficiently close to an Ars Magica stress die as makes no odds, and i a hell of a lot less confusing! (n.b. a roll of 0 on the stress die is set to give a result in the data of 1 so it's more easily comparable to an exploding d10. In either case a botch roll would be required as normal)\

set "maximum function depth" to 3

function: stress ROLL:n {
  if ROLL = 10 {result: 1 }
  if ROLL = 1 {result: 2* [stress d10] }
  result: ROLL
}

output [explode d10] named "explode d10"
output [stress d10] named "stress"
