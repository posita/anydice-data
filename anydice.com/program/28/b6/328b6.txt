function: reroll lowest of ROLL:s as REROLL:d if less than MIN:n {
  LOWEST: (#ROLL)@ROLL       \ the lowest die is sorted last \
  if LOWEST >= MIN {
    result: ROLL + 0
  } else {
    REST: {1..#ROLL-1}@ROLL  \ all but the lowest die \
    result: REST + REROLL
  }
}
output 2d6 named "base 2d6"
output [reroll lowest of 2d6 as d6 if less than 12] named "2d replace lowest if < 12"
output [highest 2 of 3d6] named "3d6 drop lowest"
output [highest of 2d6 and 2d6] named "2d6 with advantage"
