\Click calculate then click graph or transpose to more usefully analyse the results. In my opinion, an exploding d10 is sufficiently close to an Ars Magica stress die as makes no odds, and i a hell of a lot less confusing!\

set "maximum function depth" to 3

function: stress ROLL:n {
  if ROLL = 10 {result: 1 }
  if ROLL = 1 {result: 2* [stress d10] }
  result: ROLL
}

output [explode d10] named "explode d10"
output [stress d10] named "stress"
