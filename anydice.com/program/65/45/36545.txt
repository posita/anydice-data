function: successes in N:s up to M threes {  \ ':s' expects a sequence: the function is invoked for all possible sequences that can be made by rolling dice \
 result: [count {6,5} in N]               \ count 6s, 5s, and 4s \
       + [lowest of M and [count {4} in N]]   \ plus count at most M 3s \
}

loop N over {2..8} {
  output [successes in Nd6 up to 1 threes] + [count {5..8} in 1d8] named "[N]d6+1's w d8"
}
