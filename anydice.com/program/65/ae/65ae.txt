DR: 10

\Damage vs DR calculation\
\A= DAM, B= DR, C= Pen, D= Wounding\
function: damage A:n B:n C:n D:n{
 if C=0 { B: B*2 C: 1}
 if C=1000 {B: 0}
 if D=0 { X: (A - B / C) / 2}
 else {X: (A - B / C) * D}
 if X < 1 {result: 0}
 result: X
}

DAM: 2d6+2
PEN: 1
WOUND: 1
output [damage DAM DR PEN WOUND] named "9mm"
