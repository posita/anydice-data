\**** Damage calculator ****\

function: attackroll ROLL:n and BLESS:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + BLESS + PROF + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

CRITDAMAGE: 2d10 + ATTACKMOD + DMGBONUS + PROF
