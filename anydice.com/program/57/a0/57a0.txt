\ Use normal view for straights exactly X long, at-least for X or longer straights. \

function: straight ROLL:s {
 S: 1
 SMAX: 1
 PREV: 1@ROLL
 loop N over {2..#ROLL} {
  if N@ROLL = (PREV - 1) {
   S: S + 1
   SMAX: [highest of SMAX and S]
  }
  else { S: 1 }
  PREV: N@ROLL
 }
 result: SMAX
}

output [straight 3d6] named "getting a straight of length X"
