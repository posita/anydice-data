function: attackroll ROLL:n vs AC:n misfire MF:n{
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL <= MF { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\looped against enemy AC from 15-20\
ENEMYAC: d{15..20}

ATTACKMOD: 4
DMGBONUS: 0
PROFICIENCY: 3
ACCBONUS: 2
CRITROLL: 20
DAMAGE: 2d8 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 4d8 + ATTACKMOD + DMGBONUS
output [attackroll 1d20 vs ENEMYAC misfire 2] named "Blunderbuss"
