\how many rolls does it take to get X successes?\
SUCCESSDIE: d10>5
X: 4

function: roll DIE:n rollcount N:n successcount M:n{
 \roll counter ups 1\
 N: N+1
 \success counter may up 1. TODO: make variable by N\
 if DIE { M: M+1 }
 \if we don't have enough successes, recurse\
 if M < X { result: [roll SUCCESSDIE rollcount N successcount M] }
 else { result: N }
}

output [roll SUCCESSDIE rollcount 0 successcount X]
