\How does allowing for a finite number of rerolls change the distribution?\
\SUCCESSDIE: type of pool-type die being rolled. N: # rerolls\

SUCCESSDIE: d6 > 4
N: 5

\looping the output over different N's\
loop MAXREROLLS over {0..N}{

\R: success counter. We loop through all dice. If it's a success, move on. If it's not, roll a new one, subtract a reroll, and move on.\
function: ROLL:s with rerolls {
 REROLLS: MAXREROLLS
 R: 0
 loop P over {1..#ROLL} {
  if P@ROLL { R: R + 1 }
  else if REROLLS > 0 {
   R: R + SUCCESSDIE
   REROLLS: REROLLS - 1
  }
 }
 result: R
}



output [N d SUCCESSDIE with rerolls] named "[MAXREROLLS]" 
}
