\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL <= CRITFAIL { result: 0 }
 if ROLL + PROFICIENCY >= AC { result: DAMAGE }
 result: FAILDAMAGE
}

\looped against enemy AC\

ENEMYAC: 48
DMGBONUS: 32
NONDOUBLEDDAMAGE: 12
PROFICIENCY: 27
CRITFAIL: [highest of (ENEMYAC-10-PROFICIENCY) and 1]
CRITROLL: [lowest of (ENEMYAC+10-PROFICIENCY) and 20]
FAILDAMAGE: 44
DAMAGE: 4d8 + DMGBONUS + NONDOUBLEDDAMAGE
CRITDAMAGE: (8d8 + (DMGBONUS*2)) + NONDOUBLEDDAMAGE

output [attackroll d20 vs ENEMYAC]
