\ Function that counts the net total of pluses \
\ Assumption is that 3 in a d3 is + \
\ Assumption is that 2 in a d3 is 0 \
\ Assumption is that 1 in a d3 is - \
\ It doesn't really matter as long as there's a bijection mapping. \
function: countfudges A {
    FUDGEROLL: Ad3
    PLUSES: [count #{3} in FUDGEROLL]
    MINUSES: [count #{1} in FUDGEROLL]

    result: PLUSES - MINUSES
}

\ Function that takes in the number of dice and the base bonus \
\ and returns the total \
function: fudge A dice plus B {
    result: [countfudges A] + B
}

\ Usage: fudge N dice plus BONUS \
\ Where in IRC it would be: Ndf+BONUS \
X: [fudge 4 dice plus 5]
output X named "Fudge 4df+5"
YYYY: [fudge 3 dice plus 6]
output YYYY named "Fudge 3df+6"
