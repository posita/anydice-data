
function: ROLL:s count crits{
 DICE_COUNT: #{ROLL}
 if {DICE_COUNT-1}@ROLL = 1 & DICE_COUNT@ROLL = 1{
  result: 1
 }
 if 1@ROLL = 6 & 2@ROLL = 6{
  result: 6
 }
 result: 0
}

ROLL: 4d6 
output ROLL named "roll result"
output {1,2}@ROLL named "Xd6 Keep highest 2"
output [ROLL count crits] named "crits"
