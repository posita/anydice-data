function: roll A:s B:s against T:s {
   R: 0
   P: [sort {A, B}]
   loop Q over T {if A+B >= Q {R: R+1}}
   if A+B < 1@T {R:R + 10}
   if 1@P = 2@P {R: R+1}
   if 2@P = 3@P {R: R+1}
   R: R + 10*[count 1 in P]
   result: R
}

loop N over {4, 6, 8, 10, 12} { output [roll 2d6 1dN against {11, 15}] named "with d[N]"}
