\*** Average total damage over 5 rounds "Fighter 2, Assassin 5" vs "Assassin 7," assuming one surprise round and all sneak attacks ***\
DEXMOD: 4
ARCHERY: 2
PURETOHIT: DEXMOD
MCTOHIT: DEXMOD + ARCHERY

RIFLE: 1d12
SNEAKMC: 3d6
SNEAKPURE: 4d6

ACTIONSURGE: 2

SNEAKTOTALMC: RIFLE + SNEAKMC + DEXMOD
SNEAKTOTALPURE: RIFLE + SNEAKPURE + DEXMOD

SURPRISETOTALMC: RIFLE + RIFLE + SNEAKMC + SNEAKMC + DEXMOD
SURPRISETOTALPURE: RIFLE + RIFLE + SNEAKPURE + SNEAKPURE + DEXMOD

CRITROLL: 20
PROFICIENCY: 3
ATTACKMOD: DEXMOD

function: sneakattackrollmc ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: SURPRISETOTALMC } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ARCHERY >= AC { result: SNEAKTOTALMC }
 result: 0
}

}

}

 if ROLL = 1 { result: 0 }

}
