set "explode depth" to 3
function: evaluate ROLL:s {
  if [count 6 in ROLL] >= 1 {
    result: ROLL + [explode d6] -1
  } else {
    result: ROLL
  }
}

\ In the example below the "11" is the caster penetration and the 18 is the magic resistance of the unit \
output ([evaluate 2d6]+11) - ([evaluate 2d6]+18)
