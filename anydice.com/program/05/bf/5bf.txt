\ DRAGON AGE EXPECTED DAMAGE \
\ This program calculates the probability distribution of how much damage you'll deal in an attack. It does not calculate stunt points. Created by Matt Morrison, Feb 2011. I place this in the public domain. Version 3. \

\ Set the ATTACKBONUS, DAMAGE, DEFENSE, and ARMOR variables to the values you want and then click the Calculate button. You might also want to click the Graph button. \
ATTACKBONUS: 6            \ Your bonus on the attack roll. \
DAMAGE: 3d6+3             \ Your damage roll (dice plus bonus). \
DEFENSE: 12               \ The enemy's defense rating. \
ARMOR: 7                  \ The enemy's armor rating. \








\============= output ===============\

\ Show the attack roll distribution. \
SUM: 3d6+ATTACKBONUS
output SUM named "Attack roll"

\ Show how many of them hit. \
HITS: SUM>=DEFENSE
output HITS named "Hits"

\ Show the expected damage. \
DAMAGEOUTPUT: HITS*DAMAGE
output DAMAGEOUTPUT named "Damage output"

\ Show how much damage is actually taken. \
DAMAGEDEALT: DAMAGEOUTPUT-ARMOR
DAMAGEDEALT: [highest of DAMAGEDEALT and 0]  \ No negative damage. \
output DAMAGEDEALT named "Damage dealt"
