function: handcrossbow ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: HANDCROSSBOWCRIT } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: HANDCROSSBOW }
 result: 0
}

function: handcrossbowsharpshooter ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: HANDCROSSBOWCRIT +10 } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS -5 >= AC { result: HANDCROSSBOW +10 }
 result: 0
}

HIGHENEMYAC: d{23..25}
LOWENEMYAC: d{5..22}
ATTACKMOD: 5+2
DMGBONUS: 0
PROFICIENCY: 4
ACCBONUS: 2
CRITROLL: 20
ATTACKTIMES: 3

HANDCROSSBOW: 1d6+d4 + ATTACKMOD + DMGBONUS
HANDCROSSBOWCRIT: 2d6+2d4 + ATTACKMOD + DMGBONUS

output ATTACKTIMESd[handcrossbow d20 vs HIGHENEMYAC]+d[handcrossbow d20 vs HIGHENEMYAC] named "hand crossbow"
output ATTACKTIMESd[handcrossbowsharpshooter d20 vs LOWENEMYAC]+d[handcrossbowsharpshooter d20 vs LOWENEMYAC] named "hand crossbow, sharpshooter"
