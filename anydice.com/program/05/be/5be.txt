\ DRAGON AGE ATTACK ROLLS \
\ This program calculates how often an attack will succeed for a given attack bonus against a range of defense values. Created by Matt Morrison, Feb 2011. I place this in the public domain. Version 2. \

\ To use this program, set the BONUS and TN variables to the values you want and then click the Calculate button. You might also want to click the Graph button and/or Transpose buttons. \
BONUS: 3                 \ The bonus on the attack roll. \
DEFENSERANGE: {10..20}   \ The range of defense values to examine. \








\ Sum the dice. \
SUM: 3d6+BONUS

\ Loop over the defense range. \
RESULT: {}
loop I over {1..#DEFENSERANGE}
{
  \ Get the defense value. \
  DEFENCE: I@DEFENSERANGE

  \ Find out how many of the rolls passed the defence. \
  SUCCESSROLLS: SUM>=DEFENCE
  
  \ Show the result. \
  output SUCCESSROLLS named "Success against defense [DEFENCE]"
}
