\***Probability for a range of attack rolls against a specified armor rating using bonus die***\
ATTACK:2         \* number of dice to be rolled for attack *\
ATTBONUSDIE:1    \* Enter number of bonus dice to add to attacker's roll *\
ARMOR:2          \*enter # of dice in defense pool*\
DEFBONUSDIE:1    \* Enter number of bonus dice to add to defender's roll *\

function: evaluate attack ROLL:s ATTBONUS:s SOAK:s DEFBONUS:s{
 SUCCESSES: ROLL >= 5
 ATTBONUSRESULT: ATTBONUS = 6
 DEFENSE: SOAK >= 5
 DEFBONUSRESULT: DEFBONUS = 6
 TOTALSUCCESSES: SUCCESSES + ATTBONUSRESULT
 TOTALDEFENSE: DEFENSE + DEFBONUSRESULT
 if TOTALSUCCESSES > TOTALDEFENSE { result: TOTALSUCCESSES - TOTALDEFENSE }
 result: 0
}

output [evaluate attack ATTACK d6 ATTBONUSDIE d6 ARMOR d6 DEFBONUSDIE d6] named "[ATTACK]d"
