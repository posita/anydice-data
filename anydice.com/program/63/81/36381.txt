function: ceiling VALUE:n in SEQUENCE:n {
 if (SEQUENCE >= VALUE){
   result: VALUE
 }
 result: SEQUENCE
}
function: limit to zero in SEQUENCE:n {
 if (SEQUENCE < 0){result: 0}
 result: SEQUENCE
}
function: limit to max in SEQUENCE:n {
 if (SEQUENCE > 100){result: 100}
 result: SEQUENCE
}


\ Remove 3s from the Hits roll \
function: size of ROLL:s without threes {
 COUNT: 0
 loop N over {1..#ROLL} {
   if(N@ROLL != 3){ result: #ROLL -N+1 }
 }
 result: 0
}

\ Passed Hits : Hits Roll into Save ROll : \
function: HITSROLL:s attacks with modifier MF:n and save SAVE:n{
 R: [size of HITSROLL without threes]
 result: [limit to zero in (HITSROLL+MF - (RdQ + SAVE))]
}

\ Unsaved Damages : Passed Hits into Damages : \
function: PASSEDHITS:s into D:n damages{
 result: [limit to max in (PASSEDHITS * D)]
}
