\**** Damage calculator normal attack ****\

function: attackroll ROLL:n vs DEF:n {
 if ROLL >= CRITROLL { result: (ROLL + ATTACKBONUS - DEF)*EXCMULTI + (WPNDAMAGEA*2)} 
 if ROLL <= 2 { result: 0 }
 if ROLL + ATTACKBONUS >= DEF { result: (ROLL + ATTACKBONUS - DEF)*EXCMULTI + WPNDAMAGEA}
 result: 0
}

\looped against enemy DEF\
ENEMYDEF: d{10}

ATTACKBONUS: 5
EXCMULTI: 2
CRITROLL: 12
WPNDAMAGEA: 14
output [attackroll [highest 1 of 2d(2d6)] vs ENEMYDEF] named "normal attack"
output 2d6
