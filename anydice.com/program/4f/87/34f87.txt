\ Damage from a longbow with in Crown and Skull, depending on your interpretation of the rules \
\ "Dice pool explodes" means you only reroll if every die lands on its maximum value. \
\ "Individual dice explode" means you reroll any dice that land on their maximum value. \

\ Set N to the number of dice you're interested in, i.e.: \
\ N = 1 for a normal longbow. \
\ +1 for every skill/ability/enhancement that grants extra dice. \

N: 2

\ Change this if you want to simulate different die values (d8, d10, etc.) \

D: 8

\ Change this to a large number if you want to simulate "Battlemaster" \
\ higher numbers will make the calculation more accurate, but with diminishing returns \
\ since the probability of re-rolling many times is low \

set "explode depth" to 1

output 1d[explode Nd8] named "Dice pool explodes"
