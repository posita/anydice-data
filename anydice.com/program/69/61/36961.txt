\ Dramatic Task for Joker trying to get REACH tokens in ROUNDS rounds with RANK Skill \
function: jokerdramatictask RANK REACH ROUNDS {
  TOKENS: 0
  loop N over {1..ROUNDS} {
    TOKENS: TOKENS + ([highest of [explode RANK] and [explode d6]] / 4)
  }
  result: (TOKENS >= REACH)
}
