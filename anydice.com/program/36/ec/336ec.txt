\ KDM HERB GATHERING CHANCE CALCULATOR \
\ put in the number of dice rolled by each survivor for A, B, C, D and E \
\ if you have fewer survivors set them to 0 \
\ if after Overwhelming Darkness set BONUS to 10, otherwise 0 \
A: 3
B: 3
C: 3
D: 3
E: 0
BONUS: 10




function: zero if dupe on X:s {
 if #X = 1 {
  result: 1@X
 }
 if #X = 0 {
  result: 0
 }
 loop I over {2..#X}{
  if (I-1)@X = I@X {
   result: 0
  }
 }
 result: X
}
R: [zero if dupe on Ad10] + [zero if dupe on Bd10] + [zero if dupe on Cd10] + [zero if dupe on Dd10] + [zero if dupe on Ed10] + BONUS

output R > 11 named "at least 12"
output R > 44 named "at least 45"
