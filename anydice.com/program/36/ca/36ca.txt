D : 3 \ number of dices \
H : 3 \ hit chances \
G : {3,3,6,10,11} \ hit that generate bonus dices \

P : 1 \ probability of bonus dices \
loop N over G {
  P : P + (1d12 > N)
}

S : 1d12 > H \ probability of success on one dice \
S : S * P
R : S \ result \
loop N over {2..D}{
  R : R + S
}

output R
