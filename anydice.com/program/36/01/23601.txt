function: D:n BONHIT DAMAGE CRIT vs AC:n  {
    if D = 20 { result: CRIT } 
    else if D = 1 { result: 0}
    else if D + BONHIT >= AC { result: DAMAGE } 
    else { result:0}
}

BONUSDAM: STRENGTH + ATTACK
BONHIT: BONUSDAM + PROF

DAMAGE: (1d10 + 2d6 + 2d12 + BONUSDAM)
CRIT: 2*(1d10 + 2d6 + 2d12)  + BONUSDAM
