\**** Damage calculator ****\

function: attackroll ROLL:n vs AC:n {
if ROLL >= CRITROLL { result: CRITDAMAGE }
if ROLL = 1 { result: 0 }
if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: DAMAGE }
result: 0
}

\GFW die\
DGG: {{1..2}, {3..10}:6}
DGP: {{1..2}, {3..4}:3}
DGS: {{1..2}, {3..6}:4}

\looped against enemy AC from 14-19\
ENEMYAC: d{14..19}


ATTACKMOD: 6
DMGBONUS: 2
PROFICIENCY: 4
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 10
DAMAGE: 1d6 + 1d8 + ATTACKMOD + DMGBONUS
CRITDAMAGE: 2d6 + 2d8 + 6d8 + ATTACKMOD + DMGBONUS
