BASE_DC:       10
PERCY_INT_MOD:  3
ZAHRA_ARCANA:   2
VAX_PERSUASION: 2

function: adj ROLL:n MOD:n DC:n {
  if ROLL + MOD >= DC { result: -3 }
  result: +1
}

\ Percy's was a straight up intelligence check;
  Zhara's was an arcana check,
     when she was lvl11 she had Int14, don't know about skill;
  Vax had a persuasion check with advantage.
  Gives an overal 84.55% success rate.
\
A: [adj   1d20 PERCY_INT_MOD  BASE_DC]
B: [adj   1d20 ZAHRA_ARCANA   BASE_DC]
C: [adj 1@2d20 VAX_PERSUASION BASE_DC]

\ To see the DC without any modifiers un-comment this.
  Gives an overall 73% success rate.
A: [adj 1d20 0 10]
B: [adj 1d20 0 10]
C: [adj 1d20 0 10]
\

DC: BASE_DC + A + B + C

output A  named "Percy"
output B  named "Zhara"
output C  named "Vax"
output DC named "DC"
output 1d20 >= DC named "Revival"
