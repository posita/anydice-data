MELEE: 0
EVASION: 10
DAMAGEDICE: 1
DAMAGESIDES: 8
WEAPONWEIGHT: 1
PROTECTION: 5
CRITMOD: 2

MELEEROLL: [highest of (MELEE + 1d20) - (EVASION) and 0]

function: MELEEROLL: n {
if MELEEROLL = 0 {result: 1000}
if MELEEROLL > 0 {result: PROTECTION} 
}

CRITS: MELEEROLL / (CRITMOD + WEAPONWEIGHT)

output [highest of ((CRITS+DAMAGEDICE)dDAMAGESIDES - [MELEEROLL]) and 0]
