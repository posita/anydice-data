\Anima Die Rolls, Instructions at Bottom\

\The standard Anima roll, continues to a maximum depth of four consecutive Open rolls\
function: roll N:n open D:n{
 if N <= 3 {
  if D = 90 {result: -1d100}
 } 
 if N >= D {
  if D < 93 {result: N + [roll 1d100 open (D+1)]}
  result: N
 }
 result: N
}

\A combat function giving odds of inflicting different amounts of damage, feel free to change the armour and damage parameters. OBS! Anima frequently rounds to the nearest 5 in damage calculations, which this function doesn't, ergo the results differ slightly from reality.\
function: combat R:n{
 ARMOUR: 0
 DAMAGE: 50
 if R >= 20 + ARMOUR {result: (R - 20 - ARMOUR)*DAMAGE/10}
 \ if R < 0 {result: R/2} \ \This gives indication of counterattacks\
 result: 0
}

\These are the actual rolls the program performs\
output [roll 1d100 open 90] named "Standard Roll"
\ output [combat [roll 1d100 open 90] - [roll 1d100 open 90]] named "Combat Statistics" \

\Anything surrounded by backlashes are treated as comments which are not read by the program, if you want to switch between looking at standard rolls and combat stuff simply change which output is commented out.\
