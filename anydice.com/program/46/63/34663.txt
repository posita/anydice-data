\Result 1 means action succeeds:\
GREEN: d12
YELLOW: d8
RED: d6

CRIT: [maximum of GREEN]
function: ACTION:n beat OPPOSED:n {

if OPPOSED = 10 {result: 0}
if ACTION = CRIT & OPPOSED != 10 {result: 1}
if ACTION > OPPOSED {result: 1}
else {result: 0}
}

output [RED+3 beat d10] named "Green Action Success"
