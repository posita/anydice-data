function: attack ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: CRITDAMAGE } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS + ACCMOD >= AC { result: ATKDAMAGE }
 result: 0
}

ENEMYAC: d{10..20}

PROFICIENCY: 3
ATTACKMOD: 4

CRITROLL: 20
ATTACKTIMES: 1
DMGBONUS: 0
ACCBONUS: 0
EXTRADMGDICE: 0

ACCMOD: 0

ATKDAMAGE: 1d6 + ATTACKMOD + DMGBONUS + EXTRADMGDICE + 2
CRITDAMAGE: 2d6 + ATTACKMOD + DMGBONUS + 2dEXTRADMGDICE + 2

output ATTACKTIMESd[attack [highest 1 of 2d20] vs ENEMYAC] + ATTACKTIMESd[attack d20 vs ENEMYAC] named "Stealth Javelin Dueling"

ACCMOD: 2

ATKDAMAGE: 1d8 + ATTACKMOD + DMGBONUS + EXTRADMGDICE
CRITDAMAGE: 2d8 + ATTACKMOD + DMGBONUS + 2dEXTRADMGDICE

output ATTACKTIMESd[attack [highest 1 of 2d20] vs ENEMYAC] + ATTACKTIMESd[attack d20 vs ENEMYAC] named "Stealth Longbow Archery"
