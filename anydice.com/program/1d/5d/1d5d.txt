

function: attaque en puissance avec attaque A:n puissance PUI:n furror FUR:n arme DGT:n vs def DEF:n res RES:n{
 if A+FUR>= 14+DEF {
   result: A+FUR-14-DEF+PUI+DGT-RES
 }else {
   result: 0
 }
}

 }else {
   result: 0
 }
}
