\ pour avoir un resultat en nombre de succes dé blanc ; dé noir \
\ les dés blanc sont mulitplié par 100 pour transformé le vecture nb succes (blanc;noir)\
\ le parmaetre écalat est le nombre d'écalt car avoir éclat 1,2,3 a les même probablité de succes que 2,4,5 par exemple\


function: lance DE:s eclat EC:n {
R:0
loop N over {1..EC} {
 if (DE = N) >= 1 { R:R+1} 
}
result:R

}

function: lance DE:s compt ntime N:n sur de DT:n {
R:0
loop X over {1..DT} {
 if (DE=X )=N { R:R+1} 
}
result:R
}



DEBLANC:3 \ nombre de dé blanc\
DENOIR:3 \ nombre de dé noir\
ECLAT:2 \ nombre d'éclat\
DETYPE:6 \le type de dé\


output[lance DENOIRdDETYPE eclat ECLAT]+100*[lance DEBLANCdDETYPE eclat ECLAT] named "noir et blanc sur les éclats"
output[lance DENOIRdDETYPE eclat DETYPE-ECLAT]+100*[lance DEBLANCdDETYPE eclat ECLAT] named "blanc sur les éclats et noir sur les non éclat"

output[lance 3d6 compt ntime 3 sur de 6] named "pourcentage de triple sur 3d6"
