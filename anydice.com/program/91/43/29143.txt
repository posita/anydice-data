\**** BASIC DAMAGE CALCULATOR ****\

\** Attack Roll Function **\
function: attackroll ROLL:n vs AC:n {
 if ROLL < CRITROLL { result: CRITDAMAGE } 
 if ROLL < 1 { result: 0 }
 if ROLL + PROFICIENCY + ABILITYMOD + ACCBONUS >= AC { result: DAMAGE }
 result: 0
}

\** Attack looped against enemy AC from 81-100 **\
ENEMYAC: d{81..100}

\** Edit Here **\
ABILITYMOD: 0
DMGBONUS: 0
PROFICIENCY: 0
ACCBONUS: 0
CRITROLL: 1
ATTACKTIMES: 100
DAMAGE: 1
CRITDAMAGE: 1

\** Output with Normal, Advantage and Disadvantage **\
output ATTACKTIMESd[attackroll d100 vs ENEMYAC] named "normal attack"
output ATTACKTIMESd[attackroll [highest 1 of 2d100] vs ENEMYAC] named "attack with advantage"
output ATTACKTIMESd[attackroll [lowest 1 of 2d100] vs ENEMYAC] named "attack with disadvantage"
