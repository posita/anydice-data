function: hit A:n for AD:s with AB:n and DB:n against AC:n{
  if A = 20 {result: 2*(AD) + DB}
  if A + AB >= AC & A != 1 {result: 1*(AD) + DB}
  result: 0
}

function: championhit A:n for AD:s with AB:n and DB:n against AC:n{
  if A >= 19 {result: 2*(AD) + DB}
  if A + AB >= AC & A != 1 {result: 1*(AD) + DB}
  result: 0
}
