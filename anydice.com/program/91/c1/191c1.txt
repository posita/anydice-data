\**** Damage calculator full basic damage dealing list ****\

function: attackroll ROLL:n vs AC:n {
 if ROLL >= CRITROLL { result: 1 } 
 if ROLL = 1 { result: 0 }
 if ROLL + PROFICIENCY + ATTACKMOD + ACCBONUS >= AC { result: 1}
 result: 0
}

\looped against enemy AC from 16-19\
ENEMYAC: d{18}
DMGDIE: 1d12

ATTACKMOD: 4
PROFICIENCY: 5
ACCBONUS: 0
CRITROLL: 20
ATTACKTIMES: 1

DMGRESULT: [attackroll d20 vs ENEMYAC]
output ATTACKTIMESdDMGRESULT named "normal attack"

DMGRESULT: [attackroll [highest 1 of 2d20] vs ENEMYAC]
output ATTACKTIMESdDMGRESULT named "attack with advantage"
