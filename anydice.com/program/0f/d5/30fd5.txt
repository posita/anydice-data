\ first part defines the range function and the second part defines two dice types called special and specialist. The last part asks for an output of the probability of getting in that range when you add the result of those dice.\

function: range on DIE:n between LOW:n and HIGH:n {
  if DIE >= LOW & DIE <= HIGH { result: 1 }
  result: 0
}

SIX: d{1,2,3,4,5,6}
THREE: d{1,1,2,2,3,3}

output SIX + THREE named "Sum"

output [range on SIX + THREE + 2 between 2 and 3] named "Chance of Success"
