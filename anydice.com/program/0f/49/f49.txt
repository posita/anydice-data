\This counts, for X 10-sided dice, how many of those dice come up as matches. In the thousands place, it will also count how many of those dice come up as singles (i.e. its value does not match with any other die).\

function: checkem X:n {
  A: 0
  loop N over {1..10} {
    B: [count N in Xd10]
    A: A + [mydubs B]
  }
  result: A
}

function: mydubs C:n {
  if C = 1 {
    result: 1000
  } else {
    result: 0
  }
}

output [checkem 8]
