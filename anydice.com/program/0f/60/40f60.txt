\ Aumenta o limite de 'explosões' para evitar erros de cálculo \
set "maximum function depth" to 5

\ Função que define o valor de cada face \
function: verificar R:n {
 if R = 6 { result: 1 + [verificar d6] } \ Se cair 6: conta 1 sucesso E rola de novo \
 if R >= 4 { result: 1 } \ Se cair 4 ou 5: conta 1 sucesso \
 result: 0 \ Se cair 1, 2 ou 3: conta 0 \
}

\ Executa a rolagem de 3 dados usando a regra acima \
output 7d[verificar d6] named "Quantidade de Sucessos (4+ com explosão)"
